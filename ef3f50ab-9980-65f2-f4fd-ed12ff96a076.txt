　　第85章
<br/>　　程大雷听说杏花岭的人来了，立刻从屋里出来。刚到外面，他就陷入众香的包围中。
<br/>　　“这就是程当家么，长得真俊！”
<br/>　　“哈，我隐藏这么深，也看出来啦。”
<br/>　　“程当家怎么不去我们山寨做客，我们山寨可有很多人想见程当家呢。”
<br/>　　“这个……是真的么，我和大家的心情是一样的。”
<br/>　　莺歌燕语一片，程大雷刚出来就被一群女人拉扯住了，这状况却是从未有过的体验。
<br/>　　小白狼把程大雷叫到一旁，道：“程当家，我能不能先把粮食存在你这里，等以后要用的时候再来取？”
<br/>　　“为什么要这么做，你们自己拉走不好嘛？”
<br/>　　“怕的是拉走了，我们山寨守不住。”小白狼脸上露出苦笑。
<br/>　　程大雷目光转了一圈，见高飞豹一帮人眼睛红得血都有滴出来。
<br/>　　凛冬将至，大家都在想办法存粮过冬，有了这批粮食后，杏花岭过冬是不成问题。然而，周围群狼虎视眈眈，未必不会向杏花岭下手。
<br/>　　至于这粮食存在蛤蟆寨，却是不必担心有这个问题的，蛤蟆寨的名声是打出来的，如今又有了人，更不必担心谁会来抢蛤蟆寨。
<br/>　　程大雷答应下来小白狼，先让她取走一批，其他打上条子，什么时候需要，凭条来取便是。
<br/>　　渐渐，又有其他山寨前来兑粮，还真有好几个打的是和小白狼一样主意。不过更多人还是信不过程大雷选择把粮食拉回山寨。
<br/>　　趁乱，高飞豹悄悄拉着马车溜走了，再不走，他没把握不死在众人的吐沫星子下。
<br/>　　飞虎寨，高飞豹看着他牵着马车回来，车上垛着麻袋，心中立刻明白怎么会事。
<br/>　　“我说什么来着，吃瘪了吧，嗨，放心回头我找到程大雷，好好说说这件事。”
<br/>　　高飞豹一脸看白痴的神情：“大哥……你过来看看？”
<br/>　　“看什么呀，原样拉回来有什么好看的。”
<br/>　　高飞豹抽刀插入麻袋中，金黄的麦子从里面滚出来。
<br/>　　高飞虎的眼睛立刻睁大，难以置信道：“这……他真给换了？”
<br/>　　“这还有假的么，足斤足量，人家可一点便宜都没占。你是没看见杏花岭换了多少粮食，少说也有几千斤。幸亏我当初拿回来种一些，否则这次的事情咱们一点便宜也占不着。”
<br/>　　高飞虎脸皮发烧，这样的事情他还真没想到过。
<br/>　　“姓高的……你他娘的给我滚出来！”
<br/>　　“我把你十八代祖宗……”
<br/>　　正在此时，这声音突然从山寨外传来。高飞豹瞬间勃然大怒，道：“是谁敢在我飞虎寨放肆，不想活了嘛！”
<br/>　　“等等……”高飞豹拦住大哥：“别出去，骂街的来了。”
<br/>　　大家都见到杏花岭换了粮食，这件事自然不会再有什么错。知道这件事后，多少人捶胸顿足，如果当初多少种了些，这个冬天还用担心么。
<br/>　　追根溯源，这件事还是该怨高飞豹，他当初不让大家种，自己却偷偷回山寨种。
<br/>　　有这样的人么！
<br/>　　大家也打不过飞虎寨，没有办法，就把山寨的女人派出来，在飞虎寨门口破口大骂。
<br/>　　污言秽语传到高飞虎耳中，越骂越难听，连他脸上都挂不住了。
<br/>　　“他们自己不种，碍我们什么事，凭什么来我们这里骂？”
<br/>　　“这个……是我觉得这件事不靠谱，不让他们种的。”高飞豹道：“可我也没让他们都听我的啊，他们那个时候还说，跟着高二爷，保管走不错路。”
<br/>　　……
<br/>　　在将辣椒准备后好，程大雷便安排时间和来自青州的客商交易。对方把粮食拉到蛤蟆岭下，就在蛤蟆岭下进行交接。
<br/>　　在知道生意对象是山贼后，对方也没有太大反应。反正商人逐利，只要有钱赚就可以。而如此，在交易的时候，自然就不会出现缺斤短两这样的事。
<br/>　　程大雷也算厚道，一斗辣椒一斗粮食，没有刻意仗着山贼的身份欺压对方。
<br/>　　做生意就是这样，若不留半点利益给对方，一次合作后，对方就很难给你合作。而若是把自己利益让给对方，对方却又会觉得你是个傻瓜，开始想方设法侵占你的利益。
<br/>　　关键，还是分寸的把握。
<br/>　　最后大概是十比一的兑换，一斤辣椒可以换十斤粮食，程大雷用五千斤辣椒换来五万斤粮食。
<br/>　　那一天，不知道多少山寨派人打探这边的消息，大家只看到蛤蟆寨将一车又一车的粮食往山上拉。更有传言，说蛤蟆寨如今的存粮有五十万斤。
<br/>　　传言自然是信不得的，不过对整个青牛山来说，如今的蛤蟆寨是暴富了。
<br/>　　看着堆满粮仓的粮食，程大雷心里也踏实起来。如今，就算是谁围山一年半载，蛤蟆寨也绝不担心会饿肚子。
<br/>　　这大概就是家中有粮，心中不慌的道理。
<br/>　　程大雷尚且如此，何况山寨其他人。在这个时代，对于大多数而言，填饱肚子才是主要的。有这些粮食，大家在几年之内不虐有肚饿的担忧。
<br/>　　程大雷考虑是不是要搞个篝火晚会庆祝一下。刚将青州客商送走，便有一大伙人来拜山，都是其他山寨来的，那场面，把程大雷都吓住了。
<br/>　　“程当家，我们是特意过来帮忙种辣椒的。”
<br/>　　“当初我就想种，都是高飞豹这王八蛋……”
<br/>　　“说来，我可是一直看好程当家的，当初有人说程当家坏话，我是不信的。”
<br/>　　程大雷恍然大悟，原来这帮人都是看到杏花岭的收入眼馋，如今来到蛤蟆寨，死缠着程大雷要种一些辣椒。
<br/>　　“诸位，诸位……冬天要来了，农时你们肯定比我要懂，要种也得等明年开春。”
<br/>　　“那明年大当家可不要忘了我们！”
<br/>　　“我排第一个，一开春就来找大当家。”
<br/>　　如此，这帮人才离开蛤蟆寨，走的时候还对高飞豹骂个不停。他们走后不久，高飞豹就来了，说的也是同样的事。
<br/>　　不过冬天来了，什么都种不了。程大雷如此打发了他走。
<br/>　　是的，冬天要来了。
<br/>　　一阵凉风扑面，树梢最后一片叶子落在程大雷肩头，没有一丝丝防备，一个漫长且寒冷的冬天就这样猝不及防的来了。
<br/>
