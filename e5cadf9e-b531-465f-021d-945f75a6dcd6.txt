　　第二天，众多青云弟子齐聚通天峰主殿前擂台。
<br/>　　今天将在这里决出四强，按照惯例先去抽签，结果一公布，苏重就心里一沉。他的对手是齐昊。
<br/>　　苏重可不会以为当初自己挡住齐昊一剑，就能轻松胜过他。
<br/>　　神情凝重走上擂台，苏重和对面一脸温和笑意的齐昊见礼。
<br/>　　青云门是玄门正宗，门风正派，虽然少不了几个歪瓜裂枣弟子，但作为一峰的大师兄，品行和实力都还是靠得住的。
<br/>　　起码在礼节上，齐昊对苏重表示出了足够的尊重。
<br/>　　“师弟，没想到会是我们两对擂。”齐昊笑着道。
<br/>　　“见过师兄。”
<br/>　　“师弟这几天过五关斩六将，一手剑气功夫响彻青云，师兄极为佩服。”齐昊一脸赞叹。
<br/>　　“不要寒暄了，开始吧。”旁边的长老催促两人开打。旁边擂台已经打起来了，你们俩怎么还聊上了。
<br/>　　两人再次行礼，各自走到擂台两边。长老一声开始喊出，苏重就毫不犹豫打出五道剑气。
<br/>　　面对齐昊他，他丝毫不敢大意。庚金剑气厉害，但齐昊修炼时间长啊！远超苏重十多倍的修炼时间，同时还有十多倍的战斗经验。一上来苏重就全力出手。
<br/>　　砰！
<br/>　　一道白色冰墙骤然出现。五道白色剑气迅捷撞上去，轰的一下就把冰墙炸的粉碎。可冰墙之后却不见齐昊身影。
<br/>　　“师弟，你的剑气很厉害，但我却发现，你似乎还不回御剑飞行。”齐昊声音从天空传来。
<br/>　　苏重抬头看去，正好看到飘在半空的齐昊。
<br/>　　“只要躲过师弟第一波迅捷攻击，飞入天空拉开距离，就能最大程度规避师弟剑气。师弟小心了。”齐昊当初被苏重挡住飞剑，怎么可能不注意苏重。
<br/>　　比试了这么长时间，他不止一次研究过苏重手段。
<br/>　　“苍松，你这弟子也太欺负人了吧！”田不易脸色涨红。
<br/>　　这是明摆着欺负自己徒弟不会御剑飞行啊！
<br/>　　“欺负人？不至于吧，大家各凭本事，有什么欺负人不欺负人的。你徒弟不是厉害吗，让他也飞天上打啊。”苍松淡淡道。
<br/>　　田不易怒气冲冲：“我徒弟才修炼多久，怎么可能会御剑飞行！”他心里懊悔，本打算等七脉会武之后再给自家小徒弟寻找飞遁法门，没行到却在这里栽了个跟头。
<br/>　　“不会御剑飞行？你徒弟不是天才吗，怎么可能不会飞？”苍松戏谑道。
<br/>　　田不易气的说不出话。
<br/>　　擂台上两人可不知道师傅们斗嘴。
<br/>　　齐昊飞在天上，一剑猛然劈下。半空中猛然出现数十道冰晶长剑，骤然射向苏重。
<br/>　　苏重脸色大变，竭尽全力移动身体。实在躲不过，就用剑气抵挡。
<br/>　　“该死，绝对不能被冰晶剑伤到！”苏重更有些后悔：“太失策了，只想着修炼剑气，竟然把防御给忘了！”
<br/>　　剑气攻击犀利，但防御能力不强。真要被天上冰晶剑刺中，苏重基本就跪了。
<br/>　　看着天空好整一下的齐昊，苏重猛然一咬牙，他也不用剑气抵挡，全力施展身法躲避冰晶剑。然后朝着天空狠狠扔剑气。他苏重从来不是个被动挨打之辈。
<br/>　　剑气飞快，几乎脱手瞬间即来到齐昊面前。齐昊虽惊不乱，身形一转就躲过剑气。
<br/>　　“师弟，你的剑气虽然很快，但似乎只能走直线。”齐昊从容不迫。飞上天空，他就有更大的躲闪空间！
<br/>　　苏重盯着齐昊心里有些烦躁，这家伙真难缠。
<br/>　　他没了继续缠斗的心思，身形一闪退倒擂台边缘。刚才站立之处，瞬间就被冰晶剑淹没，顷刻就砸出一个大坑。
<br/>　　土石飞溅，在他脸上刮擦出数到血痕。苏重丝毫不管，五指张开虚拢在胸前，眼睛死死盯着齐昊。
<br/>　　嗡！
<br/>　　一连串剑气水晶从掌心飞出，在胸前猛然碰撞。只这一下操作，苏重额头就沁出一层细密汗珠
<br/>　　“斩仙！”一声爆喝，胸腔双掌猛然退出。
<br/>　　一道耀眼白光冲天而起，直径一米光柱直冲齐昊。几乎眨眼间就来到齐昊身前。
<br/>　　“怎么那么快！”齐昊平明鼓动体内法力，双手握剑猛然劈下。体内所有法力疯了一般输出。
<br/>　　“一定要挡住，一定要挡住！”常年战斗经验告诉他，他一旦阻挡失败，下场将极为凄惨！
<br/>　　轰！
<br/>　　一道白色屏障骤然出现在齐昊身前。白色光柱猛然冲击而上。
<br/>　　半圆屏障立刻就被冲击的扭曲变形，齐昊咬牙切齿，双眼布满血丝。砰的一下就被撞飞出去，白色光柱擦着齐昊飞入天空，没入云层消失不见。
<br/>　　全场一片寂静！
<br/>　　“田不易，你教的好徒弟！要是齐昊有个三长两短，我和你没完！”苍松双眼通红，骤然化作遁光飞向齐昊。
<br/>　　田不易也不敢怠慢，立刻飞到苏重身边。齐昊真出了事，他怕苍松发疯打伤自家徒弟。
<br/>　　苏重喘着粗气，刚才他一口气聚合了三百六十枚剑气符，直接把体内一半剑气放出。威力之强，就连苏重也吃惊不已。
<br/>　　“这才是一剑斩万法的威风！”苏重满心火热。看到身旁不住打量远处的田不易，苏重知道对方在担心什么。
<br/>　　“师傅不用担心，刚才我把剑气偏转，只是擦了个边。他一点儿事都不会有。”
<br/>　　“真哒？”田不易惊喜。
<br/>　　“那还有假。那家伙只不过是被剑气顶出擂台，一点伤都不会有。”苏重淡然道。
<br/>　　果然，这回旧件苍松已经把齐昊扶了起来。远远看去，除了脸色苍白心有余悸，好像还真没什么伤害。
<br/>　　“二蛋，行啊！真给你师傅我长脸！”田不易哈哈大笑，一脸轻松走到齐昊身前。
<br/>　　“师侄，没事吧。那臭小子出手没个轻重，这瓶里是两粒黄芽丹，快拿去修养身体。”田不易递过去一个拇指肚大小瓷瓶。
<br/>　　黄芽丹是个好东西，齐昊没受伤只是消耗大。田不易再送上黄芽丹，苍松就是想找苏重麻烦也没借口。
<br/>　　果然，看到田不易递过来的丹药，苍松脸色好看不少。
<br/>　　“还不谢过你师叔！”苍松呵斥了齐昊一句，然后就让弟子带走齐昊去修养，自己化作一道遁光回到通天峰主殿。
<br/>　　田不易呵呵一笑：“尔旦，你先回去整一桌好菜，今晚咱们庆祝庆祝！”自家弟子夺了四强，他高兴啊。
<br/>　　苏重笑着答应，在众多师兄恭喜中回到驻地。后面还有张小凡一场比赛，苏重知道结果，没什么兴趣观看。
<br/>　　留在那里虽然能看到御雷真诀，但他不能练，看了也是干眼馋，还不如不看。
