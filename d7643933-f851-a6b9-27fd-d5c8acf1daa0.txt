　　第132章
<br/>　　万籁俱寂，空余风啸雪动之音。不，倒也不能说绝对安静。地上的单飞抱着腿，发出哭爹喊娘般的惨叫。
<br/>　　程大雷微微皱眉。
<br/>　　秦蛮甩刀，捅进单飞口中，穿骨而出，血从后脑渗出来，浸红石板。
<br/>　　此刻，才算得上万籁俱寂。
<br/>　　所有人的目光都落在秦蛮身上，然后又同时转向程大雷。刚来就杀人，而且还是以如此残忍的手段，在场众人虽大多都是穷凶极恶之徒，看到单飞的死状却仍觉得一身寒意。
<br/>　　“青史几行名姓，北邙无数荒丘。前人田地后人收，说甚龙争虎斗。”
<br/>　　鸦雀无声间，老道的声音却是突然响起，看着程大雷，面带微笑。
<br/>　　“如此年轻，想来应该是蛤蟆寨程寨主？”
<br/>　　程大雷目光挪到老道身上，上下打量其一遍，忽然道：“贵姓？”
<br/>　　“姓郭。”
<br/>　　程大雷眼前一亮：“德纲？”
<br/>　　老道一脸大写的懵逼：“草字烦人，号北斗先生。”
<br/>　　到此时，周围乱纷纷的声音才响起，众人目光看着程大雷交头接耳。
<br/>　　“他便是青牛山那魔头！”
<br/>　　“原来今天他也来了，听说杜老大就是被他干掉的。”
<br/>　　“还在黑石城内烧杀抢掠，宰了人刚过门的媳妇。”
<br/>　　“但他的样子看上去也平平无奇……”
<br/>　　高飞虎此时分开众人，来到程大雷身边，看着地上单飞的尸体，也是一阵无语，怎么能刚来就杀人呢。自然，如单飞这样的恶贼，杀了也就是杀了。即便在杀人越货的绿林道，对采花贼也是深恶痛绝。
<br/>　　大家看到高飞虎立在程大雷身后，一时有些困惑，随即突然意识到一件匪夷所思的事情：难道青牛山已经不是高飞虎说了算么，现在当家的是这年纪轻轻的程大雷。
<br/>　　他刚来便坐第一把交椅，这显然是以幽州绿林道魁首自居，也明显是不把整个幽州的江湖豪杰自居，还包括哪位尚未露面的伏虎山熊老大。
<br/>　　哼，他狂由他狂，且看他能狂到几时。都说初生牛犊不怕虎，可最后还是要死在虎嘴里。
<br/>　　程大雷委实没有这个心思，他那里了解，这时代关于一张椅子有这么多弯弯绕绕。他纯粹是不想在外面吹风而已。
<br/>　　来之前，高飞虎已向他介绍过幽州绿林道的情况，对于这单飞程大雷也知道些，杀了也就杀了，并无多少心理负担。
<br/>　　心里正琢磨着今天的事情，忽觉得周围安静下来。程大雷下意识抬起头，看到西北方向远远过来一队人。
<br/>　　这队人中间一辆车，车上一身材肥胖，面目丑陋的男人，身下铺着虎皮，身后两人一左一右为他扛着斧头。
<br/>　　最出奇的是，拉车的不是马，而是一头黑熊。黑熊套着缰绳，身上伤痕累累，驱车的人用鞭子一下一下抽着。
<br/>　　程大雷下意识竟然想站起来，喉咙鼓动流口水。这是谁呀，不说其他，这头黑熊就太帅气了，究竟要怎样才能驯服一头黑熊当坐骑呢。这出场方式实在太气派了，自己和人家一比，根本就是天上地下。还有那张虎皮，记得山寨也猎到一头老虎，怎么不想着把虎皮改成坐垫呢。
<br/>　　不用说什么，此人的名字已出来了。
<br/>　　伏虎山大当家黑旋风孙蚪，江湖人称熊老大。
<br/>　　熊老大还未下车，一众人便迎了过去，那个马家父子，什么狗兄鼠弟此刻都乖乖站成一排。
<br/>　　程大雷没有动地方。
<br/>　　“熊老大！”
<br/>　　“老大，您最近可是瘦了，小的我挖到一根百年山参，送给你补补。”
<br/>　　“熊老大，我是黑娃，您还记得我么？”
<br/>　　乱哄哄一片，能上前套句近乎的都是道上有头有脸的人物，有些自觉身份低微，也不敢上前说话，只是站在后面，垂手低头。
<br/>　　熊老大没有回应任何人的招呼，直直的走进风雨亭，目光转了一圈，最后还是落在程大雷身上。
<br/>　　周围人都看着这一幕，心中颇有几分幸灾乐祸的意思，这熊老大什么脾气，岂能受得了这个，稍微动动手指头，也就把程大雷捏死了。哼，让你如此狂妄，惹到熊老大，也是自作自受。
<br/>　　熊老大一颗硕大的脑袋杵在程大雷面前，一双眼睛直勾勾盯着程大雷。被人这么近距离逼视，程大雷有些尴尬，关键是这熊老大长得太丑了。
<br/>　　煤球一样的肤色，树皮一样的脸，鼻毛从鼻孔里翻出来，头发披散在脑袋后面。
<br/>　　程大雷忍了半晌，终究是抬起手来，指着熊老大，忐忑不安道：“你……有眼屎。”
<br/>　　“……”众人。
<br/>　　熊老大愣了愣，随即仰起头哈哈大笑。程大雷尴尬得捂住耳朵：“笑比哭还难听呢。”
<br/>　　“死吧。”
<br/>　　一轮大斧迎面劈来，锵地一声，秦蛮用刀拨开，向前跨出一步，拦在程大雷面前，与熊老大目光相对，一样凶恶。
<br/>　　“且住。”程大雷此时忽然起身。
<br/>　　众人暗付：此人终究是怕了熊老大，但此刻求饶怕不是太晚了些。
<br/>　　“秦蛮，取我的大斧来，这个交给我对付。”程大雷跳出亭子，冲熊老大做个『来』的手势。
<br/>　　众人一愣：原来不是求饶，而是要亲自战熊老大。
<br/>　　程大雷好不容易遇到一个和自己一样用斧头的人，只不过对方用的是双手斧，自己用的是一柄。程大雷也想和对方碰碰，最好把他的虎皮和黑熊都夺过来。
<br/>　　“大当家，小心。”秦蛮把斧头交给程大雷，低声嘱咐。
<br/>　　“不必担心。”程大雷刚才已看过，这熊老大也是顶级山贼，实力不弱秦蛮，不过凭自己的三板斧，三招之内对方是奈何不了自己的，所以他嘱咐：“秦蛮，三招后你就出手。”
<br/>　　“明白。”
<br/>　　熊老大有些发懵，这神经病是谁啊，竟真敢和我动手。不管了，杀了一了百了，反正自己斧下也不缺这个断头鬼。
<br/>　　“二位英雄，且慢动手，且慢动手。”正在此时，一锦袍人带着一队人从外面走来，满脸堆笑：“小可乃是幽州王座下军师李善言，二位英雄听我一句劝，且莫动手，别伤了两家和气。”
