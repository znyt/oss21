　　成圣，证混元！
<br/>　　圣人与混元大罗金仙从来不是同一回事儿，虽然它们的境界都踏入了证道第三步，但第三步仍然分两种。
<br/>　　一为圣道证天。此路走的是圣道，以圣道经过十二重天，最终踏入天道境。因此，走此路的修士，亦称圣人。圣人乃魂寄天道，通过感悟大世界天道修炼，天道不灭，圣人不死。圣人在自己世界之中，还可以借助天道作战，故而，一般圣人的战力在本世界同境界中，要比混元大罗金仙强。
<br/>　　二为混元证天。此路走的是混元大道，混元，亦称浑源，凝聚自己的大道本源，经过十二重大道本源，最终成为天道境。故而，走此路的修士，只能靠自己，不能依靠天道，而且有时候，天道还会成为你的阻碍。混元十二重天修炼的是大道，大道与天道虽然同为道，但终究是有所不同的。
<br/>　　两者虽然不同，但最终证的却都是天道境。只不过，圣人的最高成就也就是天道境就结束了，若是还想要突破，那就只能靠提升自己世界的等级来提升自己的实力了。毕竟，圣人成道之时借助了天道，有得必有失，天道故而最后也成了他的限制。
<br/>　　而混元天道境就不同了，混元证得的天道境是依靠自己证来的，故而，混元修士所追求的终点为大道不朽境。
<br/>　　所以，还有句话说，圣人为大罗金仙，但大罗金仙终究不为圣人。
<br/>　　元始如今打算让自己的“超我”原始来成就圣人境界，以此来偿还洪荒世界的因果，而自己的本尊元始，则是选择证道混元大罗金仙境。
<br/>　　“既然决定好，那便开始吧。”元始理清了自己未来的道路，便不再犹豫，想要立刻突破。
<br/>　　“不过，突破混元大罗金仙不同于突破圣人境界，到时必然会降下种种灾难前来阻拦，这个世界的天道，不容许强大的战力存在，而破坏了洪荒的平衡。它需要的，只有圣人，只因混元大罗金仙是一种不听管教的修士”。
<br/>　　“是以，吾突破混元之时，为了减少麻烦，还是找一个能够蒙蔽天机的地方。天道不能涉及，自然便不会知道吾要突破，也自然不会降下灾劫。至于突破后的事情，既然自己都已经突破了，天道那时想管也管不了了。”
<br/>　　“而能满足这个条件的地方，元始印中的本源世界，便是一个最好的地方。”
<br/>　　“所以，吾便到元始印中去突破，至于外面，就让原始先守着吧。”
<br/>　　元始做好决定，不再耽搁，一个闪身便来到元始印空间，元始印的本源世界经过元始这些年收集法则，刻录天道法则，以及鸿钧道祖讲道时所讲的法则，现如今已经扩大到了一个非常大的地步，已经有洪荒的四分之三般大小了。
<br/>　　这个世界，已经不算小了，在这个世界之中，唯一与外界不同的就是，此处没有生命。其他的，一点也不比洪荒差。在这个本源世界中，元始收录的大道法则，清晰地显现在他眼前，只需要他不断地领悟，便可掌握，这就是本源世界最大的好处。
<br/>　　经过多年的领悟，元始也差不多快掌握三千大道法则了，差的只是更深层次的本源领悟。不过，那些不急，那已经是混元大罗金仙以后的事情了。
<br/>　　此时在本源空间之中，除了元始本尊之外，还有两道身影，那便是他的“道我”与“逝我”。元始没有打扰两人的参悟，他一步踏出，来到世界最中间，在这里，有一片广阔无垠的大海，此大海与一般的大海不同，他的颜色为苍茫色，大海之上，五光十色，各种色彩，闪闪发光。
<br/>　　在大海之中，可以看见各种法则链条，在隐隐闪现，相互交织。
<br/>　　此海名为“道海”。
<br/>　　在道海的中央，有一座小岛，小岛并不大，只有方圆九十九丈。此岛名为“无岛”。取自悟道之意，还有一层意思，就是，在真正的大道之海中，哪有什么岛能够让你停顿的。
<br/>　　元始此次闭关之所，便是在此小岛之上。
<br/>　　不再多想，元始盘坐在小岛之上，心神放空，与此方天地交融，合二为一，吾即是天，天即是吾。吾悟道之时，天地皆来助之。
<br/>　　“道茫茫而无之乎，心傥傥而无羁乎，物迭迭而无非乎。”
<br/>　　元始坐立虚空，心有所感，道无形，无穷尽，怎能把握，怎能言清？
<br/>　　大道无形，此刻有了本源世界以及道海的帮助，却宛若有了形迹，似乎是什么将它拘谨过来一般。
<br/>　　刹那间，元始陷入了大道之海之中。
<br/>　　无穷的道，无穷的法，自元始眼前流淌。
<br/>　　岁月悠悠，好似倒流而回，元始仿佛看到了一个条条大道河流逆流向上，一条河流，代表着一条大道。
<br/>　　万千条河流，便是万千条大道。
<br/>　　道海无涯，无尽大道河流灌入到海之中，海纳百川，有容乃大。
<br/>　　逆流而上，则是循着一条条大道河流回溯过往，循着那河流的源头而去。
<br/>　　他漫步向前，像是置身无数年前的星空下。
<br/>　　好似有一副副破碎的画卷展现在元始眼前，一道道虚幻的道则若隐若现，明灭不定。
<br/>　　“望断仙路，前不见古人，后不见来者，独怆然而立！”
<br/>　　“大如沧海，微若尘埃，万物……”
<br/>　　“万道争流，决胜最强！”元始眼中神光越发明亮，比日月星辰还要明亮百倍、千倍、万倍，他的气息起伏不定，纵然是道海的波澜，亦在这股气息的压迫下，不能动弹分毫！
<br/>　　他沉浸在各种大道之中，不可自拔。无穷大道，直指大道起点，玄之又玄……
<br/>　　世间诸般大道，时间为王，空间为尊，命运不出，因果称王，造化无双，轮回无限……
<br/>　　万般大道，各有优势。
<br/>　　空间大道，可以演化空间储物，空间瞬息，空间切割，空间破碎，不管是大战，还是逃跑，都最为强大。
<br/>　　吞噬大道，吞天噬地，吞尽一切造化，晋级最快。
<br/>　　时间大道，攻击最强，可使时间静止，静止某一段时间，让敌人暂时被定住，随意击杀。亦可施展时间加速，一瞬间数十万年，让敌人苍老，剥夺他人的寿元。甚至，施展时间倒流，让敌人回到弱小之时，随意击杀。
<br/>　　元始坐于悟道古茶树下，周身神光涌出，一道道神轮嗡嗡浮现，竖立在他脑后，无数道则翻飞，有如霞光，有如飘带，美轮美奂，道音阵阵，悦耳动听。
<br/>　　他开始尝试悟道，悟出自己的元始大道，做到万道合流。
<br/>　　面前有万道流淌，他要以自己的元始大道，将无数大道整合归一，又要不破坏这些道则的构架，做到合流的大道必须能够做到随时随地化作种种大道，演化出各种各样的神通。
<br/>　　万道汇聚成海，海亦可随时化作万道河流。
<br/>　　……
