　　“圣王！”
<br/>　　乾元也是猛然暴起，一连吞噬了无数丹药，将其中的力量全部吞噬，也爆发出一股股灵力。
<br/>　　全然加持在了荒古大道上，令这荒古大道，又是庞大了几分。

<br/>　　巨大的战龙之爪，轰击而下，其中骇人的力量，居然打在这荒古大道上，都没有足以撼动这荒古大道。
<br/>　　其中蕴含的恐怖力量，都是随着荒古大道，传入到了岁月长河之中，流入到了荒古时代。

<br/>　　“是吗？敢让我自废修为？你的古家的人吧，古尊那老家伙，都不敢说这种大话，你一个小小的蝼蚁，也敢开口？”

<br/>　　从楚风眠的口中，淡然吐出两个字。
<br/>　　“造化！”
<br/>　　这两个字，说出的时刻极轻，令其他人都没有听见。
<br/>　　只见从楚风眠的身躯之中，他身躯中的本命剑元，缓缓的动了。
<br/>　　从楚风眠的手中，这本命剑元的力量立刻凝聚，化为了传说中的造化之剑。
<br/>　　“断！”
<br/>　　楚风眠手中，造化之剑轻轻挥动，直接向着那荒古大道，斩了过去。
<br/>　　这一剑，看起来仅仅是轻描淡写一般，没有任何太过强势的力量。
<br/>　　“可笑，仅仅这种剑，也想要斩断我的荒古大道？一切力量，都要融入到荒古之中！”
<br/>　　古林不屑的大笑一声，直接催动荒古大道的力量，迎了上去。
<br/>　　要将这造化之剑的力量，全然吞噬，统统顺着荒古大道，融入到荒古之中去。

<br/>　　这造化之剑，在接近到荒古大道的一刻，其中的力量，在彻底展现了出来。
<br/>　　这力量并不强大，甚至是远远不如楚风眠的随手一击，但是其中蕴含的力量，却是玄妙到了极致的地步。
<br/>　　这种玄妙，不是力量的境地，足以弥补的，而是根本不是一个等级的力量。
<br/>　　仿佛是足以做到碾压诸天一切，主宰天地的力量。
<br/>　　造化之剑，化为了一道剑锋，直接向着荒古大道，斩了过去，斩在了荒古大道上。
<br/>　　顿时这荒古大道，都在颤抖，仿佛是感觉到了什么令他惧怕的力量。

<br/>　　这荒古大道，居然是在中途，就在这造化之剑下，断裂开来了。
<br/>　　跟荒古沟通的道路，全部断开。

<br/>　　古林的脸色，骤然一变，脸色苍白之际，身上的灵力疯狂暴起，要再一次的凝聚出荒古大道来，立于不败之地。

<br/>　　因为造化，才是世间最为强大的力量。
<br/>　　“没有荒古大道，你们荒古门的人，不过都是一群废物罢了！一剑万重！给我灭杀！”
