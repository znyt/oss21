　　慕清霄的突破速度，使得站在通道口的美杜莎柳眉狂跳，修炼天才她不是没有见到过，她本身就是万中无一的天才。
<br/>　　但看到这一幕后，心中的骄傲深深的受到了打击，实力达到斗皇层次，每提升一个境界都无比困难。
<br/>　　而盘坐在岩浆上的青年，却在半小时内，连续突破两重。
<br/>　　最重要的是，美杜莎已经完全相信，慕清霄不属于她所知晓的凤族，她可没听说过，凤族敢直接吞噬异火！
<br/>　　嘴中吐出一口浊气，慕清霄睁开双眼，精芒一闪即逝，握了握拳头，体质有全方面的提升，化身黑凤后，哪怕是斗宗他也有一战之力。
<br/>　　最重要的是，九霄琉璃焱也随着吞噬异火，威能比先前更加强大。
<br/>　　不愧是诸天万界最高贵，最强悍的血脉，层次越高就越战优势，凤凰可是神兽，人类根本无法相比。
<br/>　　比如，实力处于同一个层次，慕清霄轻轻松松就能秒杀，越级秒杀更不在话下。
<br/>　　化神四重初期，慕清霄相当的满意，但还不够，看了一眼洞口的美杜莎，后者视线依旧清冷的望着他。
<br/>　　微微摇头，慕清霄直接从储存空间中取出容器，随后一枚晶莹剔透的石子，石子中流淌着纯净狂暴的能量。
<br/>　　神药！
<br/>　　当初元婴境界的时候，慕清霄依靠神药突破了化神期，现在他实力更强，已经可以再吸收一部分神药中的能量，来增强自身的实力。
<br/>　　神药刚出现，整个洞窟中都弥漫着一股清香，站在洞口的美杜莎，美眸盯着慕清霄手中的神药，睫毛一颤一颤的。
<br/>　　她能感受到，神药中那股恐怖的能量，哪怕是她吞下去，恐怕都会爆体而亡，这小子怎会有如此多的好东西。
<br/>　　美杜莎心中极其的羡慕，但她却不敢打神药的注意，先不说现在的她无法吞服，而且，慕清霄实力估计已经能够碾压她。
<br/>　　神药中的能量还剩下许多，慕清霄也不敢贸然吞下去，依旧是用老办法，用阴阳灵气，牵引着神药中的能量，渐渐融入体内。
<br/>　　神药中的能量依旧是那么的狂暴，却在他的忍耐范围内，一边引动能量，一边炼化起来。
<br/>　　......
<br/>　　......
<br/>　　修炼无岁月，转眼就过去一周的时间。
<br/>　　岩浆表面的中央处，一股恐怖的气息在空中荡起一阵涟漪，随后向四周扩散开来，连下面的岩浆都剧烈动荡起来。
<br/>　　慕清霄逐渐睁开双眼，眼中闪烁着黑色火焰，将他神骏的脸颊，照耀的邪异非常，低喃一声，道：“化神八重初期！”
<br/>　　突破化神八重初期后，慕清霄隐隐觉得，自己与这片天地契合，这估计就是斗宗所掌控的空间奥妙。
<br/>　　悬浮在空中的神药落下，慕清霄将之窜在掌心，神药明显比先前暗淡许多，但其中依旧蕴含这一股不可小觑的能量。
<br/>　　“连续突破四重境界，接下来的时间得熟练一下突破所带来的力量，神药估计还能吸收最后一次，留在关键的时刻在服用吧。”
<br/>　　站起身来，慕清霄低头一看，却发现自己身无寸缕，以至于大鸟都暴露在空气中。
<br/>　　抬起头来，瞥了一眼通道口，却发现美杜莎依旧站在那里，淡淡的紫色眸子一眨不眨的盯着他。
<br/>　　至于慕清霄裸露在身体，美杜莎视若无睹，其实，她心中早已麻木了，在这一个星期中，慕清霄又连续突破了四个小境界。
<br/>　　虽说是借助外力突破，但她却能察觉到，后者气息极其的稳定，也不知道当初是如何筑基的，为何根基如此稳固。
<br/>　　慕清霄不知道美杜莎在想什么，但嘴角却是抽了抽。
<br/>　　这女人真是一根筋，将神药储存好后，连忙取出一套备用白袍穿上，随后又恢复平日里翩翩公子的模样。
<br/>　　来到岸上，慕清霄微微一笑，道：“让你久等了，马上就可以助你蜕变。”
<br/>　　闻言，美杜莎美眸眨了眨，仔细的打量着他，虽然已经极其确定，但依旧忍不住问道：“你......真的突破八星斗皇了？”
<br/>　　“你不都已经看到了吗？”慕清霄露出一个满意的笑容，他要的就是这种效果。
<br/>　　“我现在相信了，你的天赋是我印象中最恐怖的，你肯定不属于凤族，他们没有这样的天赋，我们换个地方吧，我不是很喜欢此处的气息。”
<br/>　　不知不觉中，美杜莎已经不再自称本王，或许连她自己都没有注意到这种细节问题。
<br/>　　美杜莎血脉阴寒，确实不可能喜欢此处，于是点头跟在她的身后，望着她走在前面，扭动着丰满玲珑的娇躯，慕清霄舔了舔干涩的嘴唇。
<br/>　　当真是绝世尤物！
<br/>　　察觉到异样的目光，美杜莎丝毫不在意，而是继续前进，此刻已经没有比脱变，更能使她在意的事情了。
<br/>　　离开地底洞穴后，美杜莎所在的空间一阵扭曲，随后就消失不见。
<br/>　　慕清霄现在是化神八重初期，也就相当于八星斗皇，但实力却完全不是八星斗皇能比的，不化身成黑凤，他就有自信战胜九星斗皇巅峰。
<br/>　　若是化身成黑凤，像美杜莎这种实力，估计也就一爪子的事情。
<br/>　　美杜莎全速离开，慕清霄则悠哉悠哉的跟在她的身后，欣赏着映入眼中的美景，谁也不知道他心中想着什么。
<br/>　　十几分钟后，空间一阵扭曲，两道身影就凭空出现在一片绿意盎然的森林上空。
<br/>　　此处，是塔戈尔大沙漠的最深处，而下方则是置身于大沙漠中的一片绿洲，绿洲覆盖范围并不大，也就方圆几十里。
<br/>　　望着下面的绿洲，慕清霄脑海中浮现冰皇给他的地图，上面标记着这片绿洲，应该是最接近美杜莎住处的地方。
<br/>　　看来这女人对自己，还是保持着一定的戒心啊，不过话又说回来，倘若自己真想把她怎样，谁又能挡住？
<br/>　　思考的同时，美杜莎已经落在的森林下面，慕清霄也缓缓落下，跟在她的身后，行走在森林中的幽静小道上。
<br/>　　转过身来，美杜莎淡紫色的美眸盯着慕清霄，声音依旧清冷，道：“你的眼睛让我很不舒服。”
