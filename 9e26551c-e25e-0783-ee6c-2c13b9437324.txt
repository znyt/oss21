        &lt;h3&gt;第三卷第一百六十六章  目标一致&lt;/h3&gt;
<br/>
<br/>        张沛尧被老婆的一个电话打回了家，最近工作太忙，都是加班加点的工作，从调回北京之后，一步步走过来实在是不容易，虽然和陆家关系不错，但是陆家和吴家又怎么可能让别的家族成长起来?
<br/>
<br/>        好的是张家比较低调，一直都是不争不夺，他的父亲已经离开了，二叔已经不管世事很多年，所以他和三弟等于是夹缝中生存。
<br/>
<br/>        这几年对于张函的表现他是看在眼里的，张函的突然转性让他很是奇怪了一段时间，但是看着孩子越来越好，他从心里还是很欣慰的。
<br/>
<br/>        弥青云打电话的口气不容置疑，他不知道怎么回事就放下手中的工作赶了回来。
<br/>
<br/>        当他回到家时，发现张函也在，这让他感觉有点奇怪，这么多年，张函很少和他们坐在一起。
<br/>
<br/>        “青云，怎么了？”张沛尧问弥青云。
<br/>
<br/>        “你有孙子了。”弥青云直截了当。
<br/>
<br/>        “就这事情你打电话叫我回来？”张沛尧觉得不可思议，不过张函和陆玉结婚这么久，才有孩子确实也值得高兴，毕竟他们老张家人丁单薄。
<br/>
<br/>        “这难道不是大事吗？”弥青云看着自己的丈夫有些无语。
<br/>
<br/>        “那是玉儿给你们打电话说的？”张沛尧没有发现陆玉在家，话说，陆玉这两年基本不回家，怎么就会有孩子？
<br/>
<br/>        “什么玉儿不玉儿的，是陈子昂生的。”弥青云烦躁的说。
<br/>
<br/>        “你说什么？张函你......”张沛尧指着张函，气得手只哆嗦，他怎么还和陈子昂有来往？
<br/>
<br/>        “你激动个什么？陈子昂怀孕的时候小函根本不知道。现在孩子都大了。”弥青云叹了口气，悔不当初啊。
<br/>
<br/>        张函一直沉默着。
<br/>
<br/>        “小函，你给你爸爸说说怎么回事。”弥青云催促张函。
<br/>
<br/>        “能有什么事？就是当初我要和陈子昂结婚，你们不让结，逼着我和陆玉结婚，赶走了陈子昂，陈子昂走的时候怀孕了。我找到了陈子昂，也找到了孩子。”张函淡淡的说。
<br/>
<br/>        “他们如今在哪里？”张沛尧皱着眉头问。
<br/>
<br/>        “北京！”
<br/>
<br/>        “他们是怎么生活的？”张沛尧对陈子昂的了解并不是很多。
<br/>
<br/>        “某公司是陈子昂的。”张函说出了陈子昂公司的名称。
<br/>
<br/>        “你说什么？”张沛尧和弥青云同时惊问。
<br/>
<br/>        “陈子昂在来北京的那一年被落叔叔接济，在落叔叔家住了四年，落叔叔让春姨和财叔一直照顾他们母子三人，陈子昂才得以有今天的成就。”张函对他父亲和母亲说。
<br/>
<br/>        张函故意抬出落家，吴家上赶着追求落家的人，落家的分量不轻，而他们张家和落家也有渊源。
<br/>
<br/>        “你们是打算认回孩子？”张沛尧比较冷静，虽然刚刚有些吃惊，但是张函说出了落家，立马将他拉回了现实。
<br/>
<br/>        “我的孩子我当然是想要认回来了，况且陆玉这几年一直不生，难道让我们张家无后？”无论是那种状况，老人的思想一直是：不孝有三，无后为大。
<br/>
<br/>        “我也想认回来，只是怕陆玉阻拦。”弥青云想到陆玉的心思就感觉不寒而颤。
<br/>
<br/>        张沛尧：“你要想认回你的孩子，除非是陆玉愿意和你离婚。”
<br/>
<br/>        “那就离婚。”张函毫不犹豫。
<br/>
<br/>        “如果你能轻易离婚，陆玉也不可能这么多年和你凑合在一起了。”张沛尧看了一眼张函，自己的儿子他多少还是了解些，陆玉的作风不太好，他们都是多少知道点的。
<br/>
<br/>        张函对陆玉不闻不问，要么就是窝囊废，要么就是不在乎，但是这么几年，张函并不是窝囊废，所以张函一定是不在乎，这几年俩个人一直都没有孩子，有可能俩个人貌合神离，直到今天他才知道原因。
<br/>
<br/>        张函是为了陈子昂母子。
<br/>
<br/>        现在形式不明朗，围绕在吴家周围的一些势力纷纷跌落，可是落家此时想要和吴家联姻不知道是何故？
<br/>
<br/>        此时冒出来陈子昂有孩子，是有意为之还是无意的？张函目前在工作上比吴家的孩子和陆家的孩子更努力，上升的空间更大，如果爆出婚外有孩子对张函肯定是致命一击。
<br/>
<br/>        看张函和弥青云的样子是一定要认回孩子的，那样势必和陆玉及陈子昂俩个人都起冲突，最不利的就是张函了。
<br/>
<br/>        “小函，你不能认回俩个孩子。”张沛尧沉思良久。
<br/>
<br/>        “为什么？”张沛尧能想到的，张函当然也能想到，只是他需要他的父亲提出来。
<br/>
<br/>        “你如果大张旗鼓的认回孩子对孩子和陈子昂目前都不利，女人的妒忌之心是很可怕的，陆玉如果知道你和陈子昂有了孩子会怎么做？这个你有没有考虑？当初逼着你妈妈去找陈子昂，后来又背着我们去找陈子昂，只是没有找到才作罢，要不然凭你一个人能保护得好陈子昂？万幸的是陈子昂遇到你落叔叔，逃过一劫。”
<br/>
<br/>        “爸爸，我就不明白了，为什么你们非要逼着我娶陆玉，你们是有什么把柄握在陆玉的手中吗？”张函气急了。
<br/>
<br/>        “十几年前，有人送了一箱子东西让我捎给陆战君，你妈妈好奇，竟然打开了那个箱子的密码，被陆玉撞见了。”张沛尧说。
<br/>
<br/>        “那个箱子里是什么？”张函问。
<br/>
<br/>        “花币！”张沛尧和弥青云同时说。
<br/>
<br/>        “他们那里来的那么多花币？”张函接着问。
<br/>
<br/>        “甸城的一个人送的。但是我不太清楚那个人是谁，陆玉一直以为那些东西是我们的，我们又不能告诉她是陆家的，如果陆战君知道我们看了那些东西，怕是我们一家人都不得安宁。陆战君后来也试探着问过我，我一直装作不知道。”张沛尧无力的叹了口气。
<br/>
<br/>        人性有真、善、美也有假、丑、恶！
<br/>
<br/>        张函的人生过得辛苦而努力，他不像是别的家族的孩子，是被呵护着长大的，他是一路披荆斩棘努力过来的，对于人性也是看得很透。
<br/>
<br/>        年少的时候不知道世间之事纷杂，跟着父母亲闹别扭，等大一些了方知道父母亲也是不容易的，所以这一次才想和他们商量。
<br/>
<br/>        张沛尧：“孩子的事情我们要从长计议，目前要保护孩子不被暴露，这件事情还有谁知道？”
<br/>
<br/>        “其实，我也是今天才知道，冯睿说陈子昂生了一对双胞胎是江俞轩的，我寻思着倩楠马上要和江俞轩结婚了，让小函去看看是个什么情况，小函才说了实话。蔡少芬是知道陈子昂有孩子的事情，但是不知道是小函的。一想到我的孙子流落在外我就难过。”弥青云难过的说。
<br/>
<br/>        “小函，你目前要注意身份，尽量不要去靠近孩子，更不能让别人知道孩子是你的，尤其是陆家。不如我们找个机会见一下陈子昂，商量一下，先将孩子送出去。找机会再说吧。”张沛尧替张函分析着。
<br/>
<br/>        “沛尧？”弥青云看着张沛尧，她以为张沛尧会和她一条心。
<br/>
<br/>        “青云，你能每时每刻去保护孩子吗？陈子昂这几年的生意做得不错，但是，说白了一切不过是浮云。”张沛尧看着张函和弥青云说。
<br/>
<br/>        “为了保护陈子昂和孩子，别无他法。”张沛尧又郑重的说了一句。
<br/>
<br/>        “我们要保护他们母子三人，这是我们欠他们的。小函、青云你们能明白吗？”张沛尧看着沉默的俩个人苦口婆心的劝着。
<br/>
<br/>        “我明白！”张函虽然很想和孩子在一起，但是如果有人利用孩子的事情做文章，事情不会很简单，不怕万一就怕一万。
<br/>
<br/>        弥青云有些不舍，她还没有看到孙子呢。
<br/>
<br/>        “青云，我们年龄也大了，之前的豪言壮志早都消失殆尽，我们不是想求得一家人平平安安吗？只要孩子们平平安安我们也就安心了。”张沛尧安慰着弥青云。
<br/>
<br/>        一家三口难得的目标一致，终于是在利益面前选择的亲情。
