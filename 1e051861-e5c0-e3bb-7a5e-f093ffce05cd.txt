　　诡异的气氛持续了很长一段时间，最后还是李承乾实在受不了了，对老头子提出告辞，离开了甘露殿。
<br/>　　钱，李承乾也不打算要了，这是没办法的事情，就当丢了吧，谁让老李好面子呢。
<br/>　　一番无聊又无趣的折腾，李承乾回到东宫，见到早已经等的不耐烦的李孝恭。
<br/>　　“我说小子，你到底在搞什么花样，直说不行么？干嘛非要老子在这里等你一个多时辰。”刚一见面，李孝恭就鼻子不是鼻子脸不是脸的说道。
<br/>　　刚刚在老李同志的书房外面，这老货多少有些顾及，没太放肆，现在到了李承乾的地盘上，就彻底的露了本相。如果不是因为确实打不过这老货，李承乾甚至有把他乱棍打出去的想法。
<br/>　　不过说这些都没用，赚钱才是硬道理，李承乾索性也不去理会李孝恭的牢骚，直接就拉着他往东面的宜春宫走。
<br/>　　“我说，你小子到底要干嘛？有啥话直说不就行了么，你这是要憋死老子么？”
<br/>　　“跟我来就是了，给王叔你看点好东西。”李承乾故作神秘的走在前面，不断回头催促着，似乎前面真有什么好东西。
<br/>　　这也就是因为东宫面积并不大，从宜秋宫（‘兰若寺’）走到宜春宫并没有多少距离，所以在李孝恭彻底暴发之前，两人已经来到一座大殿的外面。
<br/>　　“你想让我看什么？就这破房子？老子家里有得是，不用看你的。”李孝恭一边吐槽，一边四下打量，这里虽然不是第一次来，但跟离上次来怎么也隔着四、五年了。
<br/>　　“在里面。”李承乾指挥着侍卫将大殿的门打开，带着李孝恭走了进去。
<br/>　　坛子，入眼所见全是大大小小的酒坛子，上面写着斗大的‘酒’字，足以说明它们的身份。
<br/>　　李孝恭看着这些坛子，忍不住倒吸了一口冷气：“小子，想不到你竟然存了这么多！”
<br/>　　“只是一部分，王叔要不要尝尝？”
<br/>　　“滚犊子，再拿老夫开涮，你就直接跟我回宗正寺待着吧。”李孝恭大眼一翻，没好气的说道。
<br/>　　李承乾示意侍卫去到一个角落，在那里有一个大概有人头大小的酒坛，等侍卫搬出来之后，他才轻轻一敲坛子说道：“这个不一样，这是好东西，一般人我都不告诉他们。”
<br/>　　李孝恭认真的打量着侍卫抱着的坛子，半晌才对李承乾说道：“你小子总不会说里面是酒吧？”
<br/>　　“当然是酒，不是酒难道是酒精么？”
<br/>　　“切，我还当是什么好东西，要说酒，宫里什么好酒老夫没喝过，还用得着你来显摆！”听说坛子里是酒，李孝恭顿时露出不屑一顾的神情。
<br/>　　“四娃，把酒打开，让我们大唐礼部尚书、宗正寺大宗正见识一下，咱们东宫的酒他到底见没见过。”
<br/>　　李承乾也不多说，扭头对抱着坛子的侍卫吩咐一声之后，便施施然的退到一边，比了一个请的手势，示意李孝恭亲自去试试。
<br/>　　人头大小的酒坛被放到了大殿中的一张木桌上面，叫作四娃的侍卫轻轻在封在坛口的封泥上敲了一下，将其砸碎扫到一边，露出里面封着坛口的包着红绸子的木塞。
<br/>　　没了坛口的封泥，浓郁的酒香开始四下里飘散开，纵然隔着木塞，依旧让周围十来步的距离之内酒香四溢，闻之让人垂涎欲滴。
<br/>　　李孝恭此时也顾不得自己的郡王形象了，三两步冲到桌边，一把将四娃推开，双眼死死的盯着坛子，就好像会有人上来抢酒一般。
<br/>　　“这，这是什么酒？为何酒香如此浓郁？”一只手按在木塞上面，酝酿了半天，李孝恭看看站在不远处的李承乾，呆呆的问道。
<br/>　　李承乾将两手抄在胸前，傲然说道：“它就叫‘酒’，没有其它名子。”
<br/>　　“‘酒’？”李孝恭咂咂嘴，叹了口气说道：“不错，和它相比，以前老夫喝的那些的确算不上酒，就连陛下御赐的佳酿，与其相比也要差上许多。”
<br/>　　“就凭这酒，二十万贯，外加建酒坊的地皮，算王叔你五成干股，不多吧？”李承乾嘿嘿一笑，坐地起价，除了二十万贯之外，又多了一块地皮。
<br/>　　不过李孝恭却没有计较那么多，只是哼了一声：“小子，话不要说的太满，你那酒精味道也不错，不过喝起来嘛……嘿嘿。”
<br/>　　“酒好不好王叔可以亲自尝试一下，不过，尝过之后价钱可就是二十五万贯了。”
<br/>　　“小子，你敢坐地起价！”地皮李孝恭可以不在乎，但是平空多出五万贯钱，这可等于要了他的老命了。
<br/>　　“不什么敢不敢的，小侄有言在先，不开坛，二十万贯加二百倾的地皮；开坛，二十五万贯加三百倾地皮。同样是五成干股，就看王叔你敢不敢赌了。”
<br/>　　赌？还是不赌？李孝恭犹豫着。
<br/>　　单单凭借酒香，其实已经足够诱人，李孝恭完全可以肯定这酒一定是真的，而且一定是顶级好酒。
<br/>　　可是想想李承乾这小子的累累前科，李孝恭又不能完全确定。
<br/>　　犹豫良久之后，李孝恭狠了狠心，决定二十五万贯就二十五贯，尝了之后至少可以有些保障，如果真是好酒，五万贯钱应该很快就可以赚回来。
<br/>　　至于那几百倾土地，问题并不大，最多就是以礼部的名义占上一块，然后借给李承乾使使罢了，反正又不掏他自己的钱，无所谓了。
<br/>　　于是乎，李承乾以五万贯的价钱，卖出了大唐有史以来最贵的一坛酒。
<br/>　　同样，李孝恭也喝了他有生以来最贵的一坛酒。
<br/>　　然后，整个事情变的一发不可收拾，尝过‘酒’的李孝恭死活都要把大殿中的酒装车拉走一半，美其名曰自己花了二十五万贯，已经有五成的干股，这些酒必须有他的一半。
<br/>　　李承乾怎么劝都劝不住，最后只能祭出最后的法宝‘李二陛下’。
<br/>　　声称这么多酒往宫外运，必定会引起皇帝陛下的兴趣，到时候偷鸡不成蚀把米，酒没运出去，反而被扣留在宫里，得不偿失云云，这才打消了李孝恭把酒运走的念头。
