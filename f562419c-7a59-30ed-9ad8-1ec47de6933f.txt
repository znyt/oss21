　　第683章
<br/>　　高力士向前走着，见程大雷脚步停住，回头望了一眼。
<br/>　　“大当家，是发生什么事了么？”
<br/>　　“喔，没有，走，继续带路。”
<br/>　　系统承认程大雷占领凉州城，便发布了奖励与任务。不过程大雷此刻正走在路上，也没时间仔细研究。索性先随高力士去看看，回过头再琢磨收到的系统提示。
<br/>　　被扣押的女俘，都被关押在城中一座府邸中。这原先是凉州城戴白的将军府，虽然比不得城主府气派，但也是雕梁画栋，不是普通人家可比。
<br/>　　门前有兵丁站岗，见程大雷过来，急忙行礼，将大门打开。程大雷迈进去时，忽然吓了一跳，险些退出来。
<br/>　　根据程大雷想象，抓住几十人便算不错，然而，府中押着五六百人。想想也是，帝国的文武官员都是讲究排场的，丫鬟小妾歌姬夫人女儿一大帮，全城的人集中在此地，有五六百人也不算稀罕。
<br/>　　这些人被关押在将军府，还穿着破城时的衣物，人人满腹愁容，见到程大雷时都露出惧怕神色，生怕这恶贼会突然扑过来扒光她们的衣服。
<br/>　　程大雷在帝国的名声绝对不好，在凉州城更甚，城中的文武官员有事没事都喜欢数落程大雷的不是。在耳濡目染之下，这些女人自然将程大雷视作色中恶魔。
<br/>　　程大雷宛若走入百花丛一般，一个个女眷模样都极为俊俏，丰体纤腰，模样楚楚可怜，令人心动。
<br/>　　如果普通姿色，如宋伯康这样的人也看不上，他们跑得着急，就把这些人舍下了。
<br/>　　程大雷走到将军府的大厅，坐下后也没人上茶，自己杵在那里发愣。
<br/>　　“大当家，这些女人该如何处理？”高力士又问一遍。
<br/>　　程大雷此时已经没了主意，问道：“一般是怎么处理的？”
<br/>　　“喔，像这些女人都很好办，大当家若有看得上的，自可收入房中，她们之中虽然有残花败柳，但有些还是黄花闺女，不委屈大当家。剩下的，大当家可以赏给有功之臣，诸人挑剩下的，便可发配到青楼。”高力士说起这些一点也不陌生，最后道：“大当家，你拿个主意，属下实在没办法了。”
<br/>　　程大雷恍然明白，这高力士并不是没有处理的法子，而只是有些事他不好处理，需要程大雷点头。
<br/>　　攻城为什么，还不是抢钱抢粮抢女人。程大雷虽然以军令约束手下兵丁，不许他们侵犯百姓。但很难说，这群人有多么高的道德素养。程大雷不让大家侵犯百姓，动一动敌人的女人总是没什么吧。
<br/>　　大家都知道是高力士负责这些女人，这几天内接二连三有人上门要人。如高飞虎高飞豹张肥这些人，对程大雷的忠心自然不必说，可不能因为他们对程大雷忠心，便绝了男女之念。老子豁出命打仗，要几个女人算什么。
<br/>　　对他们，高力士委实惹不起，关键是也不知道程大雷是怎样态度。才会一拖再拖，拖到现在，请程大雷发句话，赶紧把这群女人处理掉，也免得自己里外不是人。
<br/>　　按照高力士的想法，是程大雷先挑选一遍，程大雷如果想全部收下，也不算什么。之后，便是高飞虎这样的大将，然后便是稍次他们一些的武将……几轮挑选下来，估计也就不剩下什么了。
<br/>　　程大雷坐在那里，倒抽一口凉气，现在他体内善恶交战。一面，他的确体会到权力的快感，如今一声令下，便有千娇百媚的女子送入房中。被自己选中，甚至可以说是她们的福分。
<br/>　　另外一方面，这的确有违程大雷做人做事的风格。他的确杀过许多人，但多是疆场之上，不敢说自己没有错杀，可很多时候你不杀对方，就要伸出脖子让别人杀，这也是没有办法的事。
<br/>　　可如果真的把这件事做了，天下不会有人说什么，毕竟程大雷的名声已经足够糟糕，可程大雷过不去心中自己这道坎儿。
<br/>　　“难，难，难。”
<br/>　　程大雷敲着桌面，口中发出一声长叹。
<br/>　　高力士一旁看得稀里糊涂，他并不明白这有什么难的。在如今的帝国，女人本就是货物一般，世家互赠美妾不仅不算什么，反而有可能成为美谈。喔，对了，大当家定然是一个也舍不下，又不好意思说出口。
<br/>　　唉，这有什么可为难的呢。
<br/>　　“今夜，我便将她们送进城主府，伺候大当家，由大当家慢慢调教，其中还有一对双生子，还是没开过苞的黄花闺女……”
<br/>　　程大雷瞪了高力士一眼，令他闭上了嘴巴。这混蛋，我本来已经取舍不定，你还一声声诱惑，当真是混账至极。
<br/>　　“罢了罢了。”程大雷忽然长身而起，口中叹道：“做人做事，做事做人，当真是宁可得罪天下，却过不了自己这一关。”
<br/>　　高力士不明白程大雷发什么感慨，道：“大当家知道怎么处理了？”
<br/>　　程大雷点点头：“如那些丫鬟，多是穷人家的孩子，你调查清楚，如果父母健在的，便让父母接走。”
<br/>　　“大当家当真仁慈。”高力士道：“剩下的呢，给大当家送去？”
<br/>　　程大雷拍了拍额头，觉得头越来越疼，他无可奈何道：“放了，让她们各回各家，各找各妈？”
<br/>　　“放了？”高力士当真是一惊，难以相信这话会从程大雷口中说出。
<br/>　　他想了想，诚心诚意道：“大当家仁慈为怀，属下佩服，但放了她们对他们来说也未必是好事？”
<br/>　　“此话怎讲？”程大雷不解道。
<br/>　　“她们多是来自凉州以外的地方，兵荒马乱，让她们孤身上路，不是遭了野兽，就是遭了山贼。而放她们在城中居住，大多是孤家寡人，大当家觉得，她们的日子会好过么？”
<br/>　　程大雷一楞：“难道我放了她们，还不如让她们成为男人手中的玩物？”
<br/>　　“能活着已经是不错，至少落入大当家手中，她们能有一顿饱饭。”高力士摇摇头苦笑，忽然说了一句不像他口中说出的话：“就是这样一个世道，这就是她们的命，她们得认自己的命。”
<br/>(本章完)
