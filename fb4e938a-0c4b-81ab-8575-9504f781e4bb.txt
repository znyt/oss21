　　灵神界！
<br/>　　那是元始为这方真实的精神世界所起的名字，灵魂精神之世界。
<br/>　　元始的仔细打量着这个新的世界，满意的点了点头，他在这方世界，有着绝对的掌控权，世界之中的一草一木，都逃不过他的掌控，他为灵神界之主。
<br/>　　“有意思……就像，真身降临一般……”元始轻轻一指点出，击穿了天地，撼动了整个世界，虚空之中顿时出现一个巨大的黑洞，吞噬万物。
<br/>　　“嗯？有些脆弱了……”
<br/>　　元始看着眼前出现的黑洞，摇了摇头，好像有些不太满意。他用手向着虚空轻轻一抹，便将黑洞平复。
<br/>　　“世界初生，还有些脆弱，它此刻需要的是精神养料，或者信仰念力，才能够进化。”元始若有所思道。
<br/>　　“不过，如今已经能够承受大罗金仙的攻击了，大罗以上，那就再是没有办法了。也罢，那就先开通大罗金仙的权限，等以后收集足够的信仰之力，强化世界之后，那便再开通其他的。”
<br/>　　元始踏空，看着灵神界的九重天，说道：“每重天代表着一个境界，第一重天，凡尘天，对应仙阶以下，炼精化气，练气化神，炼神返虚，炼虚合道以及虚仙境界的修士。”
<br/>　　说着，元始并指在天空写下‘凡尘天’三个金色大字，之后挥手间设置几道规则秩序，隐入虚空，规定此天只能进入仙级以下的修士，若是修为超过，想要进入，便会被天地规则压制境界到凡级。
<br/>　　元始来到第二重天，接着写道‘真灵天’三个大字，改变其中的规则，只允许真仙境界及以下的修士进入此天。
<br/>　　“第三重天，天外天，天仙及以下者可入……”
<br/>　　“第四重天，玄心天，玄仙及以下者可入……”
<br/>　　“第五重天，金阳天，金仙及以下者可入……”
<br/>　　“第六重天，太乙天，太乙金仙及以下者可入……”
<br/>　　“第七重天，大罗天，大罗金仙及以下者可入……”
<br/>　　“第八重天，圣幽天，准圣及以下者可入……”
<br/>　　“第九重天，混元天，圣人或者混元大罗金仙可入……”
<br/>　　之后，玄天一一将各重天刻上名号，只要别人一进入，便能够知道此天的名字，而且他也设下了相应的禁制秩序，改变世界道则，各个境界的修士，只能到相应的天地中。
<br/>　　“嗯？放肆！本座的世界，岂能容你来放肆！”蓦然，元始一声大喝，看向虚空，眼中神光爆射，洞穿虚空，射向世界深处。
<br/>　　在那里，有天道的三千规则之力正在入侵元始的灵神界。灵神界初生，又依赖着洪荒世界而生，此刻天道便想要利用天道之力渗透进来，将元始的灵神界同化为洪荒的一部分。
<br/>　　“哼！”
<br/>　　元始作为灵神界之主，自然第一时间便察觉到了天道之力的入侵，只听他冷哼一声，元始大道轰然爆发，伴随着一道巨大的大道合流显现虚空，遮蔽九天十地，入侵新世界的天道之力，还没来得及开始掌控这个世界，便被元始大道压迫、侵蚀、同化。
<br/>　　同时，元始直接运转世界之力，将配合元始大道，将洪荒天道之力部分炼化，使得灵神界更加巩固。
<br/>　　“吾虽为世界之主，但也不可能随时来监测着此方世界，所以……灵神界需要一方天道。”元始若有所思道。
<br/>　　元始伸手一招，虚空中顿时一阵波动，紧接着，出现一道白色身影，这身影与元始有七八分相似，但他的眼神却充满淡漠之情，冰冷如冰，不带任何情绪，就好似一块寒冰，不见丝毫的感情波动。
<br/>　　这是元始的道我。
<br/>　　“正好，‘道我’更像是一方天道，没有丝毫感情，只会悟道，修道，掌道，如今融入此方世界，作为天道最好不过了。”元始看向‘道我’。
<br/>　　‘道我’点了点头，脸上没有丝毫波动。
<br/>　　元始单手一挥，‘道我’顿时便化为一团清流，融入到天地之中，消失不见。
<br/>　　“轰隆隆！”
<br/>　　只见虚空一阵颤动，紧接着，大地在扩张，在延伸，天层开始变高，世界在改变，不过一时三刻的时间，灵神界便扩大了二分之一，世界也变得更加坚固了，规则更加完善。
<br/>　　此刻的灵神界，却是已经可以容纳准圣高手了，也就是说，洪荒之中，除了少数的几个圣人之外，其他人已经全部可以进入灵神界了。
<br/>　　“本座创此精神国度的目的是，众生演化万道，由本座借鉴吸收，增进元始大道。而，欲使众生演化大道，没有比生死搏杀更好的了。毕竟，生灵的潜力是无限的，只有在真正大战之中，才能爆发出来。而，灵神界中，并不会真正的死亡，这便意味着，没有人不想来到其中试炼。”
<br/>　　“所以，本座这方精神世界，可以当作是万族的试炼之地。洪荒亿万种族之间的对战，人与神兽的对战，与仙人的对战，与天地万物的对战……这样的试炼，想必没有人不会不动心？”
<br/>　　元始轻轻一笑，一念之间，便在灵神界之中化出各种神兽，各种妖兽，当然，也有各种神秘的禁区，磨难之地，绝望之地等特殊地方。这些都是精神构成的，元始身为世界之主，一念生万物，再简单不过了。
<br/>　　“进入此界，可与洪荒万族同境界之辈对战，决胜出同级最强者，在生死之间进行体悟，增进自身战斗经验，锤炼自己的意志，打破自身极限，让自己得到升华！”
<br/>　　“当然，本座还需要在这其中加点彩头，那就更有意思了。比如，打怪掉装备。”
<br/>　　“妖兽身体之中随机藏一些简单的道术神通，功法，仙功，神器灵宝等，杀了妖兽，便有可能会得到其中的宝物，想必没有谁能够忍得住？”
<br/>　　元始神秘一笑，接着，他神念一动，挥手间便洒出亿万光雨，飞散到天地各处。在这片华丽的光雨之中，全部都是仙法秘术，神通武器等。这些东西对于元始来说没什么，但对于天下间其他修士，可就未必了。特别是对于一些没有传承的散修，若是有机会能够得到玄功秘法，他们不疯狂才怪。
<br/>　　“如今，万事具备，只欠试炼者。不过，这些试炼者也要做出限定。不是什么人都可以进来，想要接引洪荒修士进入此地，需要接引令，不然这个世界就乱套了。通过接引令，本尊能够了解到任何一个试炼者的情况，能够更好的控制灵神界的人流情况等。”
<br/>　　元始若有所思道：“接引令便相当于游戏人物的编号或者头盔，每一个人的必须都独一无二才行，防止别人冒充。”
<br/>　　尽可能的想到更多，在各方面都考虑周全后，元始才开始制造接引令。
<br/>　　制作接引令的材料当然是灵神界中的材料最好，毕竟它本身就是灵神界的一部分，天下独一无二，别人无法冒充。
<br/>　　之后，元始再用因果法则改造，将每一枚接引令，都用因果虚线串联起来，而这些因果的源头，便是交给灵神界天道掌控。这就仿佛是无尽的网线将全球的网络连接起来，因果法则无处不在，正好适合。
<br/>　　不过片刻时间，元始便制造出无尽接引令。接引令是一块寸许大小的圆印，一面刻有‘灵’字，一面刻有‘神’字，正好是灵神界。而且这些接引令就相当于一件灵宝，能够被别人炼化，一旦炼化，别人就抢不走，它会一直随你到死亡，才会消散。
<br/>　　“很好，新的时代即将来临，你们准备好了吗？”
<br/>　　……
