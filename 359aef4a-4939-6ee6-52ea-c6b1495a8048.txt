　　永恒岛上，玉虚宫中。
<br/>　　地水火风涌动，飞入太上无极元始庆云，最初最古之心紫气缭绕，同样投了进去，戊己金莲，开天灭世之象，五德凤凰、虚空金龙、火焰毕方、时空烛龙、诸天神雷、不周神山、缥缈元心、诛神大印紧随其后，被无极混沌之意一一吞噬消融。
<br/>　　就在此时，万道归一，初看之时，仿佛只有一个点，蕴藏着诸般大道、无数可能、过去未来、最初与最终的一个混沌之点。再看之时，那明明是一朵覆盖万界的宏大庆云，幽光垂下，形成水幕。
<br/>　　万象归一，元始无极。
<br/>　　至此，元始再创一印——
<br/>　　无极印！
<br/>　　逆有之太极而上，成无之混沌无极！
<br/>　　无极印与开天印刚好相反，开天印为混沌开天，一化万象，而，无极印却为万象归一，道归混沌。
<br/>　　无极印初创，代表着元始大道更进一步。万道合一的道一印都创出来了，象征着无之大道的无极印也就紧跟着出来了。
<br/>　　创出无极印固然值得高兴，但元始并没有太过兴奋，因为他的目的是与兄弟交流大道，而不是创造神通。
<br/>　　此次显化太上无极元始庆云的整个过程，只是为了给自己两兄弟一个直观的感受，庆云的演化代表着自身大道的演化，他人在观看庆云演化的过程之中，能够相互对应自己的大道，找到共同点，增进道行。
<br/>　　演化过后，就要讲道了，再不讲道，你看老子与通天的脸色都快不对了。
<br/>　　故而，元始悠悠开口道：
<br/>　　“吾有一道，名元始大道。然，何为元始？吾以为，元者，源也，始者，初也，元始则为最初本源。”
<br/>　　“元始之道，既为本源大道，又为太初之道，太始之道。”
<br/>　　“天地源于始，万物归于终。一元复始，元始大道，为诸果之因，为天地道源，为万道之源，为万象之始。”
<br/>　　“无，名天地之始，有，名万物之母。”
<br/>　　“所谓‘有’，是物质，是玄黄。所谓‘无’，是虚无，是混沌。元始之道，于‘有’中逆‘无’，于‘无’中生‘有’。”
<br/>　　“大道同归而殊途，一致而百虑……”
<br/>　　“凡道，无根无茎，无叶无荣。凡道，亿万根茎，亿万枯荣……万物生，万物枯，万物兴，万物散，万物合……皆道。”
<br/>　　“夫道者，乃天地相参，亿万合一，而后可窥其成功……”
<br/>　　“道生一，一生二，二生三，三生万物，万物负阴抱阳，阴极阳生，阳极阴生，阴阳相合，太极初成。”
<br/>　　“反之，亦可万物返归于三，三归于而，二返于一，一归道。”
<br/>　　……
<br/>　　洋洋洒洒，元始这一讲就讲了千年，千年之后，元始便停了下来，不在讲道。
<br/>　　元始所讲之道，包罗万象，各种大道，尽皆有之，但，每一种大道，他都只讲一些皮毛，最重要的是他在讲自己的元始大道。
<br/>　　在讲道的过程之中，元始也是在梳理自己的大道，大道理顺一遍之后，道行更进一步。
<br/>　　元始讲完后，老子接上。老子亦放出自己的庆云。他的庆云有三道清光，汇成三朵青莲，三朵青莲之中承载着种种大道，但最重要的只有三种，一种是阴阳之道，一种是丹道，最后一种是无为之道。
<br/>　　每一朵莲花，都蕴含着无穷奥妙，蕴含着一股清净无为的意境。
<br/>　　这是太上太极道德庆云。
<br/>　　“余以为，大道应无为，为无为，事无事，味无味。无为，则无不治。……”
<br/>　　“道可道，非常‘道’；名可名，非常‘名’。
<br/>　　‘无’，名天地之始，‘有’，名万物之母。
<br/>　　故常‘无’，欲以观其妙；常‘有’，欲以观其侥。
<br/>　　此二者，同出而异名，同谓之玄。玄之又玄，众妙之门。”
<br/>　　“有无相生，难易相成，长短相形，高低相盈，音声相和，前后相随，恒也。”
<br/>　　……
<br/>　　老子之道，大多以无为之道，讲得是无为而不治，清净淡然，处无为之事，行不言之教。
<br/>　　老子之后，就是通天，通天见两位兄长如此给力，也毫不示弱。显出奇异独特的庆云，庆云之上，三花各自凝聚成一种形状，不像元始那样，三朵庆云合一。
<br/>　　通天一朵庆云凝聚成一柄锋利惊天的虚幻长剑，剑意冲天，斩尽天下，另一朵形成一个阵道世界，世界之中，五行相生，四象涌动，雷电咆哮，神兽化形。
<br/>　　最后一朵庆云之上，却是化为朱雀、白虎、青龙、玄武四象神兽，各占一方，这是四象之道。
<br/>　　这庆云名为太上混元灵宝庆云。
<br/>　　元始若有所思，看来通天将盘古所留的开天印记全都消化了，并形成自己的大道。
<br/>　　“吾之道，为截剑道，大道五十，天衍四九，遁去其一。天道之下，自由一线生机，修道之人，本就是截取一线生机来修炼，……”
<br/>　　“夫道者，其成，先天地生，悟道乃天地相参，剑道，乃杀伐之道，欲成剑仙，须有一颗至诚之心，剑，乃心中之剑。”
<br/>　　“剑仙，一切走的都是‘剑道’。”
<br/>　　“不管是地水火风四象之道，还是金木水火土五行之道，甚至是杀戮、毁灭、终结之道……尽皆可融入剑道。”
<br/>　　“剑，便是吾之身。”
<br/>　　“剑，便是吾之命。”
<br/>　　“剑，便是吾之路，吾之道。”
<br/>　　……
<br/>　　通天这一讲，再次讲了千年，三人讲道，通过元神交流，********，无数玄妙符文在三人身边盘绕，一阵阵玄奥之意覆盖了整个永恒岛。
<br/>　　元始并没有开启声音防护，故而，这道音传遍了整个永恒岛。这却让永恒岛上的无数生灵受益，无数珍禽异兽纷纷匍匐在地，无不在静静体会着这玄之又玄的道意，一时之间灵性大增，智力不断上涨。
<br/>　　还有许多花草树木在此觉醒灵智，特别是靠近玉虚宫的几棵松树、墨竹，及池塘中的几株白莲花，玉虚宫之上的仙鹤，更是立即长大无数倍，树木郁郁葱葱，动物气息冗长，再过不久便会化形而出。
<br/>　　整整三千年，元始三人才停止讲道，元神纷纷归位，众多异兽珍禽也默默一拜，之后静静退去。
<br/>　　但元始几人却是没有立即醒来，而是静静体悟此次论道所得，直到百年之后，众人才相继醒来。
<br/>　　……
