　　楚风眠出现了。

<br/>　　跟那龙未央一行人，对立而视。
<br/>　　“此人的实力，比起七日之前，还要强大了许多！”
<br/>　　一尊龙党弟子，猛然开口道。
<br/>　　七日前，他也是在众圣堂中，亲眼见到过楚风眠击败郑罗的。
<br/>　　但是那个时候的，楚风眠身上的气息，还没有这般恐怖。
<br/>　　如今的楚风眠，比起七日之前的，要恐怖的多，楚风眠身上的气息，绝对像是一尊大人物。
<br/>　　楚风眠如今领悟到了剑道古籍中的一切，他身上的气息，几乎已经是要接近剑道之主。
<br/>　　剑道之主，是位列九域剑锋的人物，剑道上，还要超越颜古龙帝。
<br/>　　这种气息，令人感觉到，都不寒而栗。
<br/>　　“这气息，好古老，此人在众圣堂中，很有可能得到了一位先祖的传承！”
<br/>　　又一尊龙党弟子，目光凌厉，打量起楚风眠。
<br/>　　在众圣堂中，一些天资极高的弟子，进入其中，便是有可能，得到一些北邙学院先祖的传承。
<br/>　　如今楚风眠身上的气息变化，令他们想到的，只有这一点。
<br/>　　“哦？居然是又得到了奇遇？”
<br/>　　龙未央的一双眼睛，死死的打量着楚风眠，突然大笑起来。
<br/>　　“这倒是无妨，反正马上就是一个死人了，身上的奇遇越多越好，身上的奇遇越多，我可以得到的也就越多。”
<br/>　　“楚风眠？”
<br/>　　白萱看到楚风眠出现，也是脸色一愣，随后大声喊道。
<br/>　　“快走！”
<br/>　　这里，是生死台，在这北邙圣域中，只要是来到了生死台，便是生死由天。
<br/>　　龙未央一行人，抓走了白萱，逼迫楚风眠来到生死台，就是为了对付楚风眠。
<br/>　　白萱心中也是清楚无比，龙未央抓走她，目的就是为了对付楚风眠。
<br/>　　不然的话，为了对付她，根本不需要耗费如此手段。
<br/>　　如今楚风眠出现在生死台上，白萱便是明白过来这一切。
<br/>　　她心中的欣喜，一闪而过，随后却是极为慌张。
<br/>　　如今，楚风眠要面对的，可是龙党，龙未央。
<br/>　　龙未央在北邙圣域中，都是其中的佼佼者，无敌的强者，建立起龙党，在北邙圣域中都地位崇高。
<br/>　　白萱清楚，楚风眠的实力很强，足以轰杀生死境高阶的强者。
<br/>　　但是龙未央的实力，也一样很强，他曾经也搏杀过一尊生死境高阶的强者，甚至是直接虐杀。
<br/>　　楚风眠的实力，面对龙未央。
<br/>　　只怕是，输多胜少。
<br/>　　在这生死台上战斗，可就不是输赢问题了。
<br/>　　输了，便是代表着陨落。
<br/>　　没有任何一丝一毫的机会，上了生死台，也就没有留手的可能。
<br/>　　“把人放开。”
<br/>　　楚风眠却是好像没有听见白萱的话一样，根本没有理会白萱，他的双眼，冰冷的扫视这一众龙党弟子。
<br/>　　楚风眠的每一句话，如今都是冰冷无比，他的脸色，虽然平静。
<br/>　　可是任由是谁，都足以看得出来，现在的楚风眠，是真正的怒火中烧，动了真怒。
<br/>　　往日里的楚风眠，几乎都是极为平静淡漠的性格，但是正因如此，楚风眠动怒，便是令人都感觉到恐怖至极。
<br/>　　“放人？小子你是在做梦？”
<br/>　　一尊龙党弟子，阴冷一笑道。
<br/>　　“这里是生死台，你既然敢来，就已经是一个死人了？还想要让我们放人？”
<br/>　　“现在龙未央师兄，要你身上的奇遇，你识相一点的话，现在跪下，臣服我们龙党，将一切的奇遇都交出来，才可以免死！”
<br/>　　楚风眠听到这龙党弟子的话，眉头一挑，冷冷的再度开口道。
<br/>　　“放人！”
<br/>　　这两个字中，已经是充斥着不耐烦的味道。
<br/>　　“哈哈哈哈，楚风眠，一个新晋的种子弟子，蝼蚁一般的人物，也敢来威胁我们龙党？”
<br/>　　那种子弟子更是大笑起来道。
<br/>　　“别说是你今日就算泥菩萨过江自身难保，让我们放人？笑话，看来你真是失心疯不成！”
<br/>　　“不放吗？”
<br/>　　楚风眠缓缓的抬起头，一字一字的说道。
<br/>　　“既然如此，那你们就统统都去死吧！”
<br/>　　楚风眠身上的灵力，猛然爆发出来，杀意爆发，远古杀神虚影，瞬间是在他的背后，凝聚。
<br/>　　楚风眠的杀戮剑术，便是杀意越强，也就越发强大，他身上的气息，节节攀升，在这心中杀意之下，居然是跟远古杀神虚影，开始融为一体。
<br/>　　他的身躯，变的伟岸无比，高高在上，猛然一掌之间，轰击出去。

<br/>　　群龙起舞，天地裂，这一掌之威，笼罩了整个生死台。

<br/>　　这等行为，简直是疯狂到了极点。
<br/>　　北邙圣域中的种子弟子，那一个不是天之骄子，放在外面，任何一人都足以轻易击败落长恨这种人。
<br/>　　一个人，居然敢直接对他们所有人出手。
<br/>　　这一位位龙党弟子，如今都是脸色暴怒起来，这种行为，简直是丝毫没有将他们放在眼中过。
<br/>　　对他们来说，简直是一种绝对的侮辱。

<br/>　　无数龙党弟子，同时怒声大吼起来，一股股庞大灵力，轰然暴起，猛然无数武技，轰击而出，来对抗楚风眠的攻击。

<br/>　　楚风眠看着那几位龙党弟子，眼神中尽是不屑。
<br/>　　放在以前，面对这些龙党弟子的联手围攻，也许楚风眠还会忌惮而已。
<br/>　　但是自从楚风眠领悟到了九域剑术后，他的实力，暴涨数倍不止，这些人，一个个在楚风眠看来，都是一些大一点的蝼蚁。
<br/>　　想要捏死，也是随手足以。
