　　“呃，忘了。”叶少阳挠了挠头，望着翠云的背影，犹豫了一下，说道：“那个，翠云姐，我要走了。”
<br/>　　“走？你这不是刚回来吗？”
<br/>　　“这……我是说，我要走了，不在你这住了。”
<br/>　　翠云洗菜的动作停顿了一下，转头诧异地看着叶少阳，道：“你要去哪？”
<br/>　　“那个，我之前打听到，我有个师叔在江西，建了个道观，我想去投奔他。”叶少阳只能撒了个谎。
<br/>　　翠云震惊地看着他，道：“你要去江西，这可不行啊，现在各省都是不同的督军管，那边土匪还多，这一路兵荒马乱的，你怎么去。”
<br/>　　没等叶少阳开口，她又补了一句：“而且你身上一块大洋都没有，路上可怎么办？”
<br/>　　前面那些，叶少阳倒是没怎么在意，最后这句话却是让他犯难了，她说的没错，自己一分钱盘缠都没有，总不能要饭过去吧？
<br/>　　翠云站着看他，为难地说道：“我家里也没钱，不然……”
<br/>　　“不能要你的钱。”叶少阳果断地说道，提了个马扎在翠云面前坐下来，皱起眉头，喃喃说道：“我来想想，有没有什么法子赚钱。”
<br/>　　翠云也坐下来帮他一起想，问他：“你有什么特别的本事没有？”
<br/>　　“特别的……”叶少阳差点说出降妖捉鬼，摇了摇头道，“我力气还可以，干活是没问题，不过好像短时间内赚不够盘缠。”
<br/>　　“对了，你不是道士吗，不然去街上摆摊算命什么的？哦，这个好像也赚不到多少钱。”
<br/>　　叶少阳道：“是啊，我需要个短期内能赚到一笔钱的办法。”
<br/>　　翠云陪着他想了一会，突然眼前一亮，“我有个办法，啊，太危险了，还是算了。”
<br/>　　叶少阳说道：“说来听听，只要不是杀人越货就行！”
<br/>　　翠云道：“那怎么可能。不过还是算了，你又不是真的道士……”
<br/>　　叶少阳一听有门，忙说道：“我当然是真道士，你先说说怎么回事，能不能干再考虑呗。”
<br/>　　翠云只好说道：“是这样的，我们镇长家的姨太太中邪了，都说是狐狸精缠的，成天疯疯癫癫的，听说镇长找了好几个先生看过，都说这狐狸精太厉害，降伏不了，其中有个和尚还被狐狸精吊在了树上，差一点就弄死了，所以我说有危险呢。你又不会捉鬼降妖的，还是不要去了。”
<br/>　　叶少阳听她这么一说，顿时提起兴趣，道：“不满你说，其实我会捉鬼降妖。”
<br/>　　“你会？那你会的也不多，那几个先生都没办法，你能有什么办法，还是不要去了，想别的办法吧。”
<br/>　　叶少阳好说歹说，表示自己最多就是尽力而为，绝不逞能，让翠云一定带她去，翠云没办法，听他再三保证之后，总算答应下来。
<br/>　　两人草草吃了饭，叶少阳背着背包和腰带，让翠云带他去镇上，翠云也不知道镇长家在哪，只好去了镇公所打听。
<br/>　　镇公所的守兵听翠云说了来意，上下瞅着叶少阳，有些不信任地说道：“镇长到处寻找高人不假，但这看着不就是个干活的后生吗，哪里是道士了？”
<br/>　　叶少阳打了个稽首，道：“无量天尊，贫道是茅山第三十九代传人，道号少阳子，年轻虽小，本事也是有些的，烦请差爷引荐了。事成请您喝茶。”叶少阳没敢说自己是茅山掌教，不然对方打死也不会信的。
<br/>　　这守兵见叶少阳说话挺客气，点头让他等着，进去通报去了。
<br/>　　镇长一问叶少阳只有二十出头，还没穿道袍，顿时失去兴趣，让他出去打发走。
<br/>　　叶少阳得到消息之后，让守兵再去说一下，自己绝对能收了那妖精，否则分文不取。守兵再去汇报，镇长见他执着，这才召见。
<br/>　　叶少阳跟翠云一起进去，在一个敞亮的办公室里，见到了肥头大耳的镇长。翠云虽然是镇上的，也见过镇长，但还是第一次进镇长的办公室，很是紧张，不过还是主动介绍，按照叶少阳教她的，说叶少阳是她表弟，从小在茅山出家，这次是来找她探亲的，听说镇长家有妖邪作祟，特来帮忙清除。
<br/>　　镇长瞥了叶少阳一眼，对他是从哪来的显然不敢兴趣，说道：“你能捉妖？”
<br/>　　叶少阳点了点头。
<br/>　　镇长有点怀疑，说道：“我请了五个道士和尚，都失败了，有两个还差点出事。你如果只是为了混口吃喝，我劝你还是算了，看你身强体壮的，现在镇上招治安团，你也可以加入。”
<br/>　　叶少阳道：“你都找了五个，也不在乎多我这一个，总让我试试呗。”
<br/>　　镇长缓缓点头，道：“这也是个道理，不过我们当着面说清楚，如果你没能除妖，自己反而受伤，我绝对不管的。”
<br/>　　叶少阳道：“可以，签字画押都行。”
<br/>　　“那就不不必了，你表姐也在这，她听见就行，到时候万一讹诈我，我秋某人可不客气。”
<br/>　　翠云十分紧张地朝叶少阳看去，叶少阳满怀自信地点点头。
<br/>　　镇长叫了个人过来，帮着处理工作，自己带着叶少阳和翠云出门回家去。
<br/>　　镇长的家就住在镇公所不远，是一栋气派的三进门的大院子，看上去俨然是新建的，不过这种民国的建筑风格，在叶少阳的眼中就像是古董一样。
<br/>　　宅子门外，还有两个士兵把守。镇长带着叶少阳二人进去之后，先让他们在偏厅里呆着，自己先去跟夫人说一声，免得唐突了。
<br/>　　房间里只剩下叶少阳和翠云两人，连茶都没有。翠云表现的十分紧张局促，低声对叶少阳说道：“少阳，你且试试，如果不行的话，咱们就回去，他们要是不让，你就想办法溜走，我一个寡妇，他们不能把我怎么样。”
<br/>　　叶少阳听了这话，心里头暖暖的，抬头看了翠云一眼，笑道：“翠云姐放心，等我赚了钱，分你一半。”
<br/>　　翠云叹了口气。
<br/>　　叶少阳打量房间的装修摆设，看上去很是气派，尤其是桌子上放着的一个八音盒，最上面是个石英钟，做成房屋的形状。
