　　寂灭指，一指冰封天地，寂灭万物！
<br/>　　玄羽一听元始这话，便知道不好，果然，紧接着他便看到元始对他出手，他虽然气愤，但也不恼，因为他也想试一试元始的道行，是否能帮他这个忙。
<br/>　　面对元始那寂灭天地的一指，玄羽同样一指点出。这一指中带着一股锐利的锋芒，如一柄绝世凶剑出鞘，血腥之气扑面而来，不杀尽眼前一切敌人绝不入鞘。
<br/>　　浓浓的黑暗物质随着这一指化成一柄黑色宝剑，斩灭这天地，破灭这冻结的空间，斩向寂灭指。
<br/>　　针尖对麦芒！
<br/>　　两者相撞，元始一指将那黑色宝剑冰封，紧接着破碎开来，手指不停，继续向着玄羽点去。
<br/>　　这时，玄羽有一拳暴轰出，如一高黑色剑山，满山的剑气冲来，破碎了元始的这一指。
<br/>　　“不错，有几分实力。那么，便再接我一印！”元始见此，赞叹一句，接着不停，一印轰出。
<br/>　　元始的道则翻滚，元始大道加持，万道凝聚，化成一柄巨大的紫金神斧，一斧劈出，天地破裂，混沌翻滚，演化地水火风。紫气分裂，化为玄黄二气，玄者升为天，演化出日月星辰，黄者落为地，大地浮现，万千山脉拔地而起，大江大河奔腾，涌入大海……
<br/>　　元始印之开天印！
<br/>　　如今的开天印早已非当初那般简单，元始随意一劈，便可劈出大千世界，不过元始早已控制其威力，他并非是要创世，而是要用来战斗，自然不能像寻常一般。他创造的世界主要是用来困住玄羽，世界之内，便是他的领域，由他来主宰，所以世界并不需要很大，但却需要很坚固，为了接下来的战斗做准备。
<br/>　　玄羽被困入新世界之中，此界之中的一切道则都是全新的，他的实力受到了最大的压制，不能完全发挥，不过，他并没有慌张，伸手一引，将白玉仙门召来护在身前，散发着莫大天威，镇压一切动乱。
<br/>　　元始印之灭世印！
<br/>　　元始不停，又是一印轰出！
<br/>　　新开辟的世界陡然间倒塌，天人五衰降临，天地大劫爆发，劫气暴动，星辰坠落，山河破碎，火山爆发，大江翻滚……一切的一切，化为一个巨大的磨盘，磨盘搅动，将天地吞没，灭世大磨，将天地内的物质化为混沌之气，所有大道，悉数被剿灭。
<br/>　　灭世大磨向着玄羽轰来，巨大的毁灭之道，恐怖的磨灭气息，劫灭的终结黄昏……宛若整个世界都在攻击他，玄羽如今要面对的便是整个世界，白玉仙门摇摇晃晃，受到剧烈的震动，灭世大磨遮天蔽日，带着磨灭一切的力量，覆盖下来。
<br/>　　绝对不能再等了，时间拖得越久对他越不利。玄羽果断出手，抬手便是轰出一道巨大的剑光，黑色物质形成的剑光之中，带着毁灭，破灭，力量等数十种大道法则，一剑斩向上天，誓要破灭苍穹。
<br/>　　太玄破天斩！
<br/>　　玄羽不停，再次轰出一拳，带着无上霸道的气息，一种唯我独尊的无上风采，拳罡撕裂苍穹，无上拳意轰向那向他磨灭而来的大磨。到了这等境界，任何一种神通之中都会带着自己的大道意境，有了大道的加持，威力为变得恐怖无比。
<br/>　　轰隆隆！
<br/>　　斩天剑意加上霸道拳意轰击而至，一声惊天爆炸，终于将灭世大磨轰碎，玄羽的危机终于得到缓解。不过，玄羽刚刚恢复，此刻大战，他身上的气势却是萎靡不少，显然这对于他来说还是有些勉强。若是再对战一会儿，他恐怕就更不好受了
<br/>　　不过，此刻元始已经得到了他想要知道的了，他就此罢手，淡声道：“很好，你不差，不过，想要我帮忙，你还差了一点，先把你身上的伤势养好吧。”
<br/>　　说完，元始便要转身离开。这时，玄羽急了，开口问道：“我该如何联系你？”
<br/>　　“有事用此玉符联系，不过，本座不喜欢麻烦。没有事情最好别麻烦本座。”说完，元始扔了一块白色玉符给他，之后带着执尸转身离开。
<br/>　　玄羽接过玉符，眼中闪过一丝恼怒及愤恨，不过，想起自己如今的处境，还是先解决自己的问题吧。
<br/>　　玄羽收起残躯的太玄宫，突然发现门匾竟然不见了，他突然想起好像是刚才那人带走了。
<br/>　　看着元始离去的方向，玄羽：“……”
<br/>　　良久，苦笑一声，玄羽收起东西，他准备重新寻得一个地方，去恢复伤势。
<br/>　　来到时空乱流之外，玄羽发现自己站在混沌之中，周边一片虚无。
<br/>　　怎么回事？这里的世界呢？玄羽一阵疑惑，他记得此处原先是有一方世界，但此刻却消失不见了。微微一想，他便知道了，定是元始收走了世界。
<br/>　　……
<br/>　　轰隆隆！
<br/>　　混沌之中，发生一阵爆响，混沌之气翻滚不休，风暴不断，演化种种事物。上空，两道身影静静地看着下方的演化，这两人正是元始本尊与执尸。
<br/>　　良久，演化完毕，一方世界静静地立在那里，吸收着周围的混沌之气，化为先天灵气，作为补充世界的养料。
<br/>　　“好了，此界就当是洪荒的附庸世界吧，设下此界规则，立接引之门，凡是达到至尊境界的，都可以选择飞升洪荒界。”元始淡淡道。
<br/>　　言出法随！
<br/>　　他的话音刚落，便有道则衍生，在虚空之中显化白玉仙门，落下接引之光，横立苍穹之巅，一阵仙光闪烁，仙音袅袅，霞光满天，之后，仙门隐于虚空之中，消失不见。
<br/>　　“好了，也该回去了。”
<br/>　　元始转身，在前方有一个大世界，这个世界很大很大，至少是神州世界的数千倍大小，而且非常危险。
<br/>　　天地玄黄，宇宙洪荒。洪荒意味着无边无际的宇宙。
<br/>　　他远远就就能感受到天道的气息，天道高高在上，俯视众生，监管苍生。
<br/>　　这说明，天道对这个世界的影响已经越来越大了，而且，鸿钧道祖的实力也越来越强了。
<br/>　　距离下一次量劫的降临，已经不远了。
<br/>　　元始仔细看了一眼，便没有多看，直接回到洪荒世界。
<br/>　　只是，他刚回到昆仑山不久，便收到了鸿钧的传召，叫诸位圣人前往紫霄宫，。
<br/>　　“紫霄宫，鸿钧。”元始低头沉思片刻，眼光微微闪烁，不知道在想些什么。
<br/>　　“这个世界，终究是平静太久了，要起波澜了……”
