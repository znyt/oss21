　　楚风眠跟白萱刚刚进入到这众圣堂之中，顿时无数眼神，都是看了过来。
<br/>　　这北邙圣域中的种子弟子，核心弟子加起来不过一百多位，突然多出来两个陌生面孔，一下子也让周围不少弟子的视线，都看了过来。
<br/>　　“居然来了新人。”
<br/>　　“还一次晋升了两位新人。”
<br/>　　“听说外面虚空海，不是开启了一些古洞府吗？看起来这两人都是从这古洞府中得到了些好运的人。”
<br/>　　“那不是代表着他们的身上，有着不少的宝物奇遇？”
<br/>　　“哼，这两个家伙就算是晋升种子，也不过都是蝼蚁，排位末尾罢了，找个机会抢了他们，夺取了他们的奇遇！”
<br/>　　一声声议论声响起。
<br/>　　这看向楚风眠跟白萱二人的视线中，不少人的视线都是极为不善。
<br/>　　这北邙圣域中，一切的资源都要靠自己争抢，同门争抢宝物，都并非是不可能的事情。
<br/>　　能够晋升种子弟子的人物，没有任何一人是简单的人物，这在场的每一人，都要比起那落长恨还要强大不少。
<br/>　　弱肉强食，在这北邙圣域中，没有实力，只能任人欺凌。
<br/>　　“你们看？其中那个小子，不是这一届新人中的新人王吗？刚刚加入到北邙学院中，居然都晋升到了种子弟子？”
<br/>　　一名种子弟子，看向楚风眠，好像是认出了楚风眠的身份，有些惊讶的说道。
<br/>　　“哦？他就是寒月漓的夫君？”
<br/>　　“圣党的人，居然是会放他晋升到种子弟子，倒是大大的出乎意料了。”
<br/>　　“哼，夜天君此人心思缜密，很有可能是故意而为，让这小子来圣域中吃点苦头，他能够在外面一手遮天，但是在这圣域中，不过是底层罢了。”
<br/>　　“谁要出手，去教训教训他？教训教训他，只怕可以得到圣党不少的好处。”
<br/>　　“就由我去吧，正好我要跟圣党做一个交易，由我来教训教训此人，也让他知道，在这北邙圣域中，他不过是最为卑贱的蝼蚁，只能跪在地上！”
<br/>　　一尊种子弟子，突然现身，直接拦在了楚风眠的面前。
<br/>　　这种子弟子出现，直接是挡住了楚风眠面前的道路。
<br/>　　“站住！”
<br/>　　这种子弟子冷冷的开口道。
<br/>　　“哦？”
<br/>　　楚风眠看着眼前这中来者不善的种子弟子，目光闪烁。
<br/>　　看来他在这北邙学院中，还真是人人都想要欺一下。
<br/>　　不止是圣党的人，那些想要讨好圣党的人，如今也全部在打楚风眠的主意。
<br/>　　楚风眠心知肚明，脸上却是极为平静的开口道。
<br/>　　“不知阁下有什么事？”
<br/>　　“也没什么事。”
<br/>　　种子弟子目光闪烁，打量了一眼楚风眠，眼神中尽是贪婪的说道。
<br/>　　“听说师弟在外得到了不少的奇遇，为兄正好需要些财富，若是有什么王品灵石，皇品灵石的，只要借给为兄几百块，为兄就让你离开如何？”
<br/>　　几百块王品灵石，皇品灵石。
<br/>　　这可就是在刁难了。
<br/>　　虽然楚风眠财富众多，但是可也不会这样白白浪费。
<br/>　　“几百块王品灵石？皇品灵石？你以为这些是大白菜？就凭你，也敢来对付我？”
<br/>　　楚风眠脸上露出冷色道。
<br/>　　“给我滚！在不滚，你得死！”
<br/>　　“什么？”
<br/>　　周围一些围观的种子弟子，听到楚风眠的话，都是一楞。
<br/>　　这话倒是太狂妄了些。
<br/>　　刚刚晋升的种子弟子，那一个刚刚来到北邙圣域中，都是夹着尾巴做人的，只有隐忍到足够实力的时候，才会崭露头角。
<br/>　　而现在，楚风眠才不过是刚刚晋升种子弟子，便是如此狂妄，也是超出了不少人的意料。
<br/>　　这种狂妄，未免有些自信过头了。
<br/>　　刚刚晋升的种子弟子，在外面都是无敌的人物，所以有着傲气，不过这种傲气，被教训一下，就会烟消云散了。
<br/>　　“狂妄的小子，不过是刚刚晋升的种子弟子罢了，就敢如此嚣张？正好，今日我就先教训教训你，免得你招惹更多的麻烦！”
<br/>　　这种子弟子冷笑一声，傲然道。
<br/>　　“我要让你知道，你的实力，不过只是蝼蚁罢了！”
<br/>　　话音一落，这种子弟子便是猛然出手，一股灵力注入到拳风之中，一拳猛然爆轰而出！
<br/>　　“真理神拳！”
<br/>　　一拳而出，仿佛蕴含天地真理，无比可怕的灵力疯狂散发出来，这一股力量，似乎是要镇压一切，是霸道之际的一拳。
<br/>　　“真理神拳！郑罗居然参悟了这一招！”
<br/>　　“这是我们北邙学院先祖之一，真理天尊创造而出的无上武技，接近天级！”
<br/>　　“楚风眠的实力，虽然不错，但是他却还没有参悟过众圣堂中的无数武技秘术，不可能是郑罗的对手。”
<br/>　　一尊尊种子弟子，缓缓开口道。
<br/>　　这北邙圣域中，最为珍贵的，便是曾经无数北邙学院先祖遗留下来的无数武技秘术。
<br/>　　参悟其中一招半式，都足以轻易做到越级战斗。
<br/>　　没有参悟过众圣堂中的无数武技秘术，便是极大的劣势。
<br/>　　若是楚风眠参悟过，这一次的战斗，还有着悬念。
<br/>　　但是没有参悟过的话，这战斗，可就没有任何一点悬念，结局，已经注定。
<br/>　　“死吧！”
<br/>　　郑罗咆哮一声，真理神拳猛然压迫而下。
<br/>　　这一招真理神拳，的确是玄妙无比，楚风眠感觉得出，这至少是一尊踏上了生死台阶的大人物，创造出来的一招。
<br/>　　不过面对着郑罗的这一拳，楚风眠却是不闪不避，反而是同样握拳，直接迎了上去。
<br/>　　秘术？武技？
<br/>　　楚风眠身上拥有的秘术武技，甚至比起整个北邙学院的都要多，仅仅学会了这一招，就想要跟抗衡，简直可笑。
<br/>　　“郑罗，就凭你一个蝼蚁，也想要教训我？可笑至极！今日我就让你明白，你们这些老人，也没什么了不起的！”
<br/>　　楚风眠大喝一声，他的手动了，同样握拳，一拳轰击过去。
<br/>　　这一拳，甚至楚风眠就连灵力都没有动用，单单只是催动了远古龙王身的力量，以肉身，来去抗衡。
<br/>　　“找死？”
<br/>　　看到楚风眠竟然是以肉身抗衡，郑罗脸色一变，尽是怒色。
<br/>　　都不动用灵力，这简直是丝毫没有将他放在眼中。
<br/>　　北邙圣域中的种子弟子，哪一位不是天之骄子，绝代天才，什么时候被人如此看不起过。
