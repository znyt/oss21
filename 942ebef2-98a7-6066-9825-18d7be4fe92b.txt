　　各条通道，本就巨大宽阔。
<br/>　　如今再增添10倍，其庞大程度已相当惊人。
<br/>　　而那密密麻麻的妖兽，则是挤满了各条通道，且一路延绵至通道深处。
<br/>　　各支队伍现今的位置，仅是通道中间。
<br/>　　也就是说，众人若要前往通道尽头，便要走完这妖兽密布的一半路程。
<br/>　　锵…
<br/>　　炼药通道这边，萧逸一剑劈出。
<br/>　　剑出，十数条紫电游龙狂猛而出。
<br/>　　这一剑，是他的全力一剑。
<br/>　　狂猛奔袭而来的傀儡、妖兽，其巨大的冲击之势，瞬间被十数条紫电游龙剑气拦下。
<br/>　　“跟紧我。”萧逸说了一声，开始前行。
<br/>　　顾怜星、公孙火舞与十数天骄，连忙跟上，与萧逸的距离，绝不超过十米之外。
<br/>　　锵…
<br/>　　萧逸在前方开路，一剑劈出。
<br/>　　剑之所过，无一合之敌。
<br/>　　哪怕前方是无数妖兽以及数十傀儡挡路，却拦不住他脚步分毫。
<br/>　　只是，无数妖兽，终究是太过密集。
<br/>　　而周遭的各种禁制气息，亦连连攻来，顷刻便不满通道四周。
<br/>　　萧逸眯了眯眼，手指轻动。
<br/>　　道道剑气，覆于四周，自成剑网。
<br/>　　萧逸在前，最强的妖兽以及体修傀儡，都是他在应付着。
<br/>　　真正能在一侧试图攻击顾怜星等人的妖兽和禁止气息，并不太强。
<br/>　　仅仅剑气，足以护着众人。
<br/>　　嗤…
<br/>　　侧边，一头妖兽试图冲过剑网，其结果，便是在紫电剑气下瞬间分尸。
<br/>　　上品巅峰圣器之威，岂是轻与的。
<br/>　　释放而出的剑气，绝世之下，休想抗衡。
<br/>　　即便是绝世以上，等闲的，也休想奈何。
<br/>　　只要萧逸在最前方能拦着最强的体修傀儡与那些绝世妖兽，众人要安然通过通道，问题并不大。
<br/>　　……
<br/>　　萧逸这边问题不大，但其他队伍，可就没那么轻松了。
<br/>　　毕竟不是谁都有上品巅峰圣器。
<br/>　　更别说紫电圣器这样的剑类神兵利器。
<br/>　　其余队伍，往通道深处前行，但很快便陷入密密麻麻的妖兽包围之中。
<br/>　　而为首的数十体修傀儡，更是实力惊人。
<br/>　　黑云学教队伍那边，问题不大，毕竟有莫悠和青麟两个强者在。
<br/>　　叶流的队伍，也稍轻松些，毕竟他身后跟着贺幽、血柏两个少府主。
<br/>　　但除此之外，其余队伍，几乎全部陷入了苦战。
<br/>　　包括第一云、花若怜和长孙赤烈的队伍，以及冉琦的队伍。
<br/>　　“该死，带着这么多人，根本无法穿过通道。”冉琦谩骂了一声。
<br/>　　第一云回头看了眼身后天骄，“稍后若我护不住你们，便立刻放弃，离开洞府。”
<br/>　　第一云，乃是猎妖师。
<br/>　　他并不想看到身后的天骄葬身妖兽腹中。
<br/>　　不过，他会尽力，若实在不行，他也没办法。
<br/>　　……
<br/>　　炼药通道这边，萧逸一己之力抵挡着数十体修傀儡，以及最强的绝世妖兽。
<br/>　　看似游刃有余。
<br/>　　但手中，终究仅一把剑，如何能力敌四面八方如此密集的攻击？
<br/>　　锵…锵…锵…
<br/>　　一道道狂猛攻击袭来，萧逸仗剑一一挡下。
<br/>　　紫电神剑，握于手中；剑影，摄于身前三尺。
<br/>　　任他攻击再密集，萧逸的剑，只要足够快，便能一一挡下。
<br/>　　但，终究会有疏漏。
<br/>　　他，终究只一手一剑。
<br/>　　轰…萧逸一剑劈飞了一只傀儡，又一剑，震退了当先的其余傀儡与绝世妖兽。
<br/>　　可，还是有一只傀儡，霎时越过了他的剑锋，坚硬的铁手直噬而来。
<br/>　　萧逸刚刚连出两剑，根本来不及收剑抵挡。
<br/>　　嗤…
<br/>　　傀儡的拳头，终究是轰到了他的胸膛上。
<br/>　　萧逸闷哼一声，嘴角一丝鲜血溢出。
<br/>　　而这时，傀儡的攻击还未停止，化拳为手，竟是要直取萧逸心房。
<br/>　　“滚。”萧逸的剑，堪堪来得及收回，一剑劈出。
<br/>　　轰…一声爆响，傀儡被重重轰飞。
<br/>　　但这一瞬时间，已足以让傀儡将他胸膛抓出数道狰狞的伤口。
<br/>　　刚才一剑，若是稍慢些许，那只傀儡，足以将他脆弱的肉身抓得重伤。
<br/>　　“萧逸公子。”
<br/>　　“小贼。”
<br/>　　一旁的顾怜星与公孙火舞二人，看着萧逸胸膛上那接近触目惊心的伤痕，惊呼一声。
<br/>　　“别扰我。”萧逸不咸不淡地回了一声，手中紫电，握得极紧。
<br/>　　冷冽的目光，扫视着周遭的体修傀儡与妖兽。
<br/>　　他自然并非是简单地扫视。
<br/>　　他的眼眸，是要纵观全局。
<br/>　　这般扫视之下，已然能判断出每只傀儡与绝世妖兽攻来的方向，以及自己剑之所出要如何抵挡。
<br/>　　锵…锵…锵…
<br/>　　惊人的锵鸣声，连绵不断。
<br/>　　萧逸带着众人，强行前行。
<br/>　　可结果…
<br/>　　轰…又是一只傀儡越过他的剑锋袭来。
<br/>　　还是那般情况，他仅一手一剑，如何能将数十傀儡和数十绝世妖兽的攻击悉数挡下？
<br/>　　轰…
<br/>　　体修傀儡恐怖的巨力，重重轰在了萧逸身上。
<br/>　　“噗。”这一次，萧逸已然一口腥血喷出。
<br/>　　他虽也是体修武者，但现今的肉体层次，根本不足以应付这个层次的战斗。
<br/>　　嗤…身体上，再度多了几条狰狞伤痕，鲜血汩汩直流。
<br/>　　“萧逸公子。”
<br/>　　“小贼。”
<br/>　　顾怜星二人见状，咬了咬牙。
<br/>　　她们二人，虽也是十八府的少府主层次天骄，可却是十八府中最弱的二人。
<br/>　　顾怜星，自不必多说，本就体弱，前些天才恢复，自是浪费了大量的修炼时间。
<br/>　　公孙火舞，本只在十八府天骄中位列中流，之后又浪费了时间，直接落入了末流。
<br/>　　故她们二人，如今根本帮不了萧逸什么。
<br/>　　“萧逸公子，你这般太勉强了。”顾怜星沉声道。
<br/>　　“若是不行，我们众人放弃便是。”
<br/>　　“嗯。”公孙火舞连连点头道，“若仅你一人，这通道轻松便能过。”
<br/>　　对，萧逸此刻，终究是太过勉强了。
<br/>　　其余的各支队伍，哪个不是陷入了苦战。
<br/>　　萧逸这支队伍，本就除他以外，几乎是最弱的队伍，现今还要护着众人穿过通道，这对萧逸的压力，毫无疑问极大。
<br/>　　锵…锵…锵…
<br/>　　剑声不断，剑影延绵。
<br/>　　萧逸一边尽力抵挡前方攻击，一边皱眉思索，“还是太勉强了吗？”
<br/>　　“也罢。”萧逸眼眸一冷。
<br/>　　“最后说一遍，跟紧我，我压过去。”
<br/>　　“压过去？”众人一愣。
<br/>　　萧逸再也不语，只是，他的剑，在这一刻，瞬间变了。
