　　易老离去，萧逸一时呆愣在原地。
<br/>　　相处了一个月，萧逸大概了解易老的为人。
<br/>　　虽是个古怪的老头，确是个值得自己尊敬的前辈。
<br/>　　易老让自己去兑换那东西，自有他的道理。
<br/>　　萧逸甚至发现，易老似乎是有意而为之，在替自己打造着一份完美的修炼过程。
<br/>　　从第一天武道知识上的指点，到这个月的炼药知识指点。
<br/>　　每一步，仿佛都是为自己量身订造的一般，最是适合现在的自己修炼。
<br/>　　这，已经不算单纯的指点了，更像是一个师傅循循善诱地教导着弟子。
<br/>　　感慨地摇了摇头，萧逸转身离去，这一个月，他过得特别充实。
<br/>　　.....
<br/>　　外门任务处，自有外门执事在这里办公。
<br/>　　萧逸大概看了眼这里的任务介绍，大概明白了过来。
<br/>　　任务有很多，有轻松，也有困难；获取的剑点数，也不一样。
<br/>　　轻松的，如帮助剑派打扫宫殿，到后山扫台阶，栽种树木，清理杂草等等。
<br/>　　并没有任何难度，但剑点只有5至10点。
<br/>　　最重要的是，打扫、栽种期间，不得使用真气和武魂，得靠自己双手，一步步慢慢完成。
<br/>　　这就显得相当耗费时间了，将外门所有宫殿打扫完，不使用真气和武魂，少说得一个星期。
<br/>　　萧逸明白，这些任务，磨砺的是弟子的心性和耐性。
<br/>　　裂天剑派的剑点，乃是一种激励手段；任务，也有着各种锻炼弟子的用途。
<br/>　　困难的任务，便是出外历练了。
<br/>　　历练任务也分很多种，但总的来说，大多是击杀一些指定的妖兽。
<br/>　　其目的，是为了考验弟子修为实力。
<br/>　　斩杀的妖兽，只能靠自己，不得借助旁人或者家族势力。
<br/>　　这些任务，存在一定的危险性，难度更高，但剑点也更高。
<br/>　　一般在20点至40点之间；视斩杀的妖兽强弱而定。
<br/>　　“我接这二十个任务。”萧逸来到外门执事面前，指了指任务单，礼貌地说道。
<br/>　　执事看了几眼任务单，点头道，“这二十个任务，都是斩杀三级巅峰妖兽，每个30剑点。行了，记录好了。”
<br/>　　说罢，执事拿出一个冰晶球，道，“这是冰晶球，你斩杀妖兽时，记得使用。”
<br/>　　冰晶球，乃是高品炼药师的产物。
<br/>　　通过冰晶还有一些列复杂材料，炼制而成。
<br/>　　能烙印战斗画面，以及记录下双方的情况。
<br/>　　比如双方的修为、是否重伤等等。
<br/>　　这使得弟子做任务时必须靠自己，不能找其他人帮助。
<br/>　　当然，冰晶球也是有限制的，视炼制者而定。
<br/>　　若战斗时，产生的灵气波动高于冰晶球的承受限制，它就会失效。
<br/>　　简单来说，如果萧逸或者妖兽的实力，高于冰晶球的炼制者，它就会失效。
<br/>　　观这颗冰晶球的等级，应该只是普通四品炼药师炼制，效果只到洞玄三重左右。
<br/>　　萧逸接过冰晶球，而后转身离开。
<br/>　　这二十个任务，完全后，加起来能得到600剑点。
<br/>　　顺带一提，第二轮考核，前十名是有奖励的，乃是固定的剑点。
<br/>　　本来，萧逸在考核时，听说的是第二轮考核前十，能直接到内门宝库兑换物品。
<br/>　　后来听易老解释，才知道，并非直接兑换，而是奖励剑点，而后自己去挑选。
<br/>　　想来也是，内门宝库中的宝物，阶品远超外门宝库，怎么可能随便赠予呢。
<br/>　　前十名，第一名两千剑点；二到十名各一千剑点。
<br/>　　按照裂天剑派一贯的情况，第二轮考核前十之人，其天赋一般都能通过第三轮考核，并进入内门。
<br/>　　所以第二轮考核过后，并没有直接去内门宝库，而是先通过第三个考核，进入内门后再去内门宝库不迟。
<br/>　　这十人，算上第一轮考核中得到的剑点。
<br/>　　两轮考核加起来，足以在内门宝库换取极好的物品辅助自己修炼，或增强实力。
<br/>　　这是裂天剑派给予新入门弟子的馈赠，也是变相的帮助。
<br/>　　而事实上，也确实如此。今年前两个考核，前十的人，都进入了内门。
<br/>　　除了萧逸，因顾长空的从中作梗而并未能进入。
<br/>　　所以，他并未能到内门宝库换取东西。
<br/>　　这两轮考核下来，他便积累了四千五百多剑点。
<br/>　　加上这二十个任务，足以。
<br/>　　而这二十个任务，以他的实力，一月之内便可全部完成，并回来剑派。
<br/>　　易老也是知道这个情况，知道萧逸只差几百剑点就能兑换那东西，才让萧逸去来接任务的，顶多个把月就能回来。
<br/>　　否则，若是萧逸差几千剑点，就算是不停做任务，也要做好几年。
<br/>　　那还不如换别的办法突破洞玄境呢。
<br/>　　接过任务，萧逸往山门而去。
<br/>　　刚要到守山门的外门执事那汇报，这时，十数道人影，忽然拦在了他的面前。
<br/>　　为首的，竟是顾长风。
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　“小子，你让我好找阿。”顾长风冷笑一声。
<br/>　　“你以为躲起来就有用了吗？还不是被我逮到了。”
<br/>　　“躲？”萧逸眉头仍旧皱着。
<br/>　　“哼，还在狡辩。”顾长风得意道，“你到外门区域后，并没有到执事处报到，更没有到住所居住，就是怕我报复你吧？”
<br/>　　萧逸脸色一冷，算是明白了过来。
<br/>　　这一个月，自己并没有到外门弟子的正常住所，而是到独自到后山修炼。
<br/>　　裂天剑派，依山而建，大得很。
<br/>　　后山上，也荒无人烟，自然没有人知道自己去了那里。
<br/>　　这顾长风，怕是寻思了许久要找自己报仇，可是一直没找到，便以为自己躲起来了。
<br/>　　“萧逸，废话少说，今日，我们是来挑战你的，你就说你敢不敢接。”
<br/>　　“别告诉本公子，你是个孬种。”
<br/>　　顾长风冷声说道。
<br/>　　裂天剑派内，禁止同门相残，但却鼓励彼此间挑战切磋。
<br/>　　而且，胜的一方，还能从败的一方得到10剑点。
<br/>　　每个月，每个弟子，都有三次机会，挑战他人时，他人必须接受，不能拒绝。
<br/>　　这也是弟子们赚取剑点的另一个方式。
<br/>　　当然了，实力强的人，剑点越存越多；实力弱的，越来越少。
<br/>　　外门弟子，每月能得到剑派赠予的30剑点；恰好三次，若全输，就都没了。
<br/>　　而这三次机会过后，若是还要挑战，就要看双方意愿了，随便一方不愿意，都不能开始战斗。
<br/>　　这是门规所定。
<br/>　　看起来，面前这十数人，以及顾长风，这个月的三次机会都用光了。
<br/>　　否则，现在不会询问自己的意见。
<br/>　　若是自己答应，战斗便随时开始。
<br/>　　若不答应，他们只能乖乖让路。
<br/>　　顾长风见萧逸不语，讥讽道，“废物，我就知道你不敢接受。哼，跪下给本公子扣几个响头，滚吧。”
<br/>　　这时，顾长风身旁一人忽然道，“长风公子，得了面子就算了吧。我听说，这萧逸跟易老走得很近。”
<br/>　　“这个月来，早上到外门宝库寻易老，晚上又不知躲到哪里去，两人的关系怕是不一般阿。”
<br/>　　“废什么话。”顾长风不屑道，“易老算什么东西，不过是个守库人。本公子的亲哥，乃是剑堂首席，他日要承袭长老之位的。”
<br/>　　“你们还想在外门待下去，就别给本公子废话。待我哥哥成为长老，第一个就将那狗屁易老撵出裂天剑派。”
<br/>　　顾长风身旁的十数人，皆是外门中颇有名气的弟子，在外门修炼了多年，清一色的先天九重。
<br/>　　不过，顾长风有他哥哥在，自己又是内门弟子。
<br/>　　自然的，这数十人对他毕恭毕敬的，甚至马首是瞻。
<br/>　　顾长风，本身就擅长拉帮结派，尽干些乌烟瘴气之事。
<br/>　　“哼。”萧逸冷哼一声，“这些话，还是等你哥哥真能成为长老之时再说吧。”
<br/>　　“相信我，你哥哥没有这个机会。”
<br/>　　“同时也相信我，今日，你的嘴会被我撕烂。”
<br/>　　话音刚落，萧逸的身影消失在原地，瞬间出现在顾长风面前。
<br/>　　被挑战者，率先出手，便等同接受挑战。
<br/>　　“好快的速度。”顾长风瞳孔一缩，惊道，“上，给我废了萧逸。”
<br/>(本章完)
