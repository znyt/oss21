        童筱颖非常赞同地猛点头，完全把刚刚那个小女生所说过的话抛到了九霄云外。
<br/>
<br/>        其实，在童筱颖心里，不管傅昼景以前是什么样子，现在的他，是她的丈夫，是她爱的男人。
<br/>
<br/>        所以，他的全部，不管是好的一面，不好的一面，她全都接受。
<br/>
<br/>        另一边，哪怕是被一群人给围着，傅夜沉也能在众多人中找到童筱颖的身影。
<br/>
<br/>        他的目光总是有意无意地往童筱颖的身上落去，看着她拿着刀叉吃着盘子里的西餐，又见她皱了一下眉头便知道她不爱吃。
<br/>
<br/>        接着，他又看到她端起杯子喝着果汁，现在还看到她和弟弟阿景交头接耳，聊得笑容灿烂。
<br/>
<br/>        苏漫雪见傅夜沉看着那边的童筱颖发呆，于是只身走到傅夜沉的跟前，踮起脚尖，挡住了他的视线后，微笑着撒娇道：“阿沉，待会带我去坐摩天轮，好不好？”
<br/>
<br/>        “……”傅夜沉不吭声，只是冷着脸瞪了苏漫雪一眼。
<br/>
<br/>        苏漫雪自知无趣，只好假装挽住傅夜沉的胳膊，装出一副与他很亲密的样子。
<br/>
<br/>        刚刚那个还童筱颖面前说三道四的小女生，准备凑过来跟大哥傅夜沉寒暄的时候，发现大嫂又变装了不禁感到惊诧不已，她完全没发现，自己现在见到的女人和起初在洗手间外遇到的女人是完全不同的两个女人。
<br/>
<br/>        因为，现在的苏漫雪，除了骨子里的性子与童筱颖完全不同以外，其余都几近一模一样。
<br/>
<br/>        不管是外貌还是身段，包括声音，第一次见她的人，一定很难把她和童筱颖两个人区分开来。
<br/>
<br/>        寿宴散场了后，傅昼景本想把童筱颖介绍给自己父亲这边的亲戚，却发现，傅家的这群亲戚全都只为了大哥傅夜沉和大嫂苏漫雪转，完全把他和童筱颖晾到了一边。
<br/>
<br/>        童筱颖也发现了这点，抬眸看了身旁的傅昼景一眼，见他因为他们被冷落的场面而脸色不大好，于是，主动岔开话题，说道：“小卫，不如我们先走吧！”
<br/>
<br/>        “嗯！好！”傅昼景回过神来，微笑着点了点头。
<br/>
<br/>        今晚最高兴的人，只怕是苏漫雪了，因为这下子不仅仅只是傅老太爷和冷夫人认可了她，就连傅家的这群旁系亲戚也都认识了她。
<br/>
<br/>        她这个傅家大少奶奶之位，已经快要坐热了。
<br/>
<br/>        傅夜沉带着苏漫雪，不得不在各位亲戚面前走个见面过场，不经意间，他才发现傅昼景和童筱颖不知何时已经不见了身影。
<br/>
<br/>        他的目光四下探看，有些魂不守舍。
<br/>
<br/>        苏漫雪实在是看不下去了，当着大家的面，故意提高了嗓音，说道：“到时候，你们一定要来参加我和阿沉的婚礼啊！”
<br/>
<br/>        她这话一出，大家纷纷笑着点头应同：“一定一定！”
<br/>
<br/>        傅夜沉回过神来后，冷不丁地对苏漫雪说道：“今天我小叔公是寿星，你不要在这种场合喧宾夺主，显得你苏漫雪很没教养。”
<br/>
<br/>        “……”一瞬间，苏漫雪被傅夜沉怼得哑口无言。
<br/>
<br/>        周遭的气氛，也跟着急剧降温。
<br/>
<br/>        另一边，傅昼景开车载着童筱颖直接去了临海大学。
<br/>
<br/>        直到到了临海大学的校门口，那些不好的回忆，才在童筱颖的脑海里涌现。
<br/>
<br/>        这里是她前男友何明旭所在的大学……
<br/>
<br/>        以前，她每个周末，就会往这所大学里跑，就像个保姆一样，替前男友何明旭洗衣服洗被子洗床单。
<br/>
<br/>        车子刚在校门口旁的停车场里停下，傅昼景便接听了一个电话。
<br/>
<br/>        童筱颖不知道电话那头的人跟傅昼景说了什么，只见傅昼景皱起了眉头，询问：“方案为什么会出问题？刚刚才发现的吗？那么第二套方案呢？”
<br/>
<br/>        “……”
<br/>
<br/>        “这点事情，你们都办不好！算了，我马上赶回来！”
<br/>
<br/>        “……”
<br/>
<br/>        傅昼景气愤地挂了电话，抬眸看了看童筱颖，瘪着嘴，非常无奈地耸了耸肩，温和地解释道：“筱颍，今晚我可能无法陪你了，我得赶回我出差的地方去。明天要参加竞标，但是我们的方案好像出了问题。”
<br/>
<br/>        “没关系，你先去忙。我就在这里下车好了。”童筱颖非常体贴地说道。
<br/>
<br/>        “我打电话给大哥，叫他派个司机过来接你回傅家大庄园吧！”傅昼景接着说道，顺势拿起手机拨通了傅夜沉的手机号码。
<br/>
<br/>        而那边，傅夜沉掏出手机看了一眼来电显示后，几乎是秒接。
<br/>
<br/>        童筱颖连忙阻止道：“不用！你不用跟大哥说！我自己一个人在大学里逛逛，然后再打车回傅家大庄园。”
