<br/>    这个小家伙是个小男孩，看起来也不过只是三岁的样子。
<br/>    
<br/>        陈明起身，笑道：“你叫什么名字？”
<br/>    
<br/>        小男孩道：“我叫罗牧野。叔叔你呢？”
<br/>    
<br/>        陈明笑了笑，“我啊，我叫陈明。”
<br/>    
<br/>        罗牧野继续问道：“叔叔，我的父母呢，其他人呢，为什么我醒来之后，他们就都不见了？”
<br/>    
<br/>        “正在干扰中，正在判定，由于你拥有，判定无效。”
<br/>    
<br/>        弑君者，怎么可能会被这种光环干扰。
<br/>    
<br/>        陈明道：“都过去了，我不知道过去到底发生了什么，我也和你一样。在无尽的岁月之后醒来，便什么都已经沧海桑田。”
<br/>    
<br/>        罗牧野道：“就是说，现在的蝉族，只剩下我们了吗？”
<br/>    
<br/>        陈明认真地点头，“是的，只有我们了。”
<br/>    
<br/>        罗牧野愣了愣，他坐在地面上，将头埋在双膝之间哭泣。
<br/>    
<br/>        “父亲母亲说过的，有大劫将至，所有的一切都会毁灭，原来蝉族，真的毁灭了。”
<br/>    
<br/>        “都怪我太没用了.......”
<br/>    
<br/>        虽说这小家伙挺惨的，一统星空的蝉族毁灭了，只剩下他一个人，所有的亲人，族人都已经死去了，但是陈明还是心中忍不住想要吐槽，这就是所谓的主角思维吗？
<br/>    
<br/>        你就一个三岁的小孩子，你能有什么用？
<br/>    
<br/>        怎么可能会怪得到你的头上来？
<br/>    
<br/>        主角一贯都是喜欢将这些所谓的责任揽到自己身上来，也不管是不是真的和自己有没有关系，然后借着这责任的压力前行。
<br/>    
<br/>        你以为你是我陈仙师啊，是燕山的山主？你就一小孩，还都是你太没用了......
<br/>    
<br/>        奇怪的思维逻辑。
<br/>    
<br/>        不过，陈明喜欢。
<br/>    
<br/>        总好过没心没肺。
<br/>    
<br/>        陈明看向罗牧野，然后道：“对，就是你的错，你一个三岁的孩子，你那个时候就应该去对抗那些强大的敌人，让你的敌人多一个小孩的
<br/>    
<br/>        首级。”
<br/>    
<br/>        罗牧野看向陈明，怒道：“不是这样的！”
<br/>    
<br/>        陈明道：“那你说，你能干什么？”
<br/>    
<br/>        罗牧野愣住了，是啊，他能做什么呢？
<br/>    
<br/>        陈明继续道：“真相，往往是这个世界上最为黑暗残酷的，而你，应该学着承受这一切，活着本就已经是艰难，所以，何必要强行将一些不属于自己的责任背在肩上。”
<br/>    
<br/>        陈明继续道：“走了，去长眠之地。”
<br/>    
<br/>        罗牧野看了陈明一眼，或许是因为觉得陈明也是蝉族的原因，他莫名的信任陈明，他只是一个小孩子，而且是最后一个蝉族，除了眼前这个看起来也是蝉族的强者，他还能依靠谁呢。
<br/>    
<br/>        他只是一个小孩子，小孩子就应该做小孩子应该做的。
<br/>    
<br/>        陈明跨步，这个时候，罗牧野却是一阵小跑来到了陈明的身前，拦住了陈明，陈明疑惑道：“你拦住我干嘛？”
<br/>    
<br/>        罗牧野道：“叔叔，你走错方向了。”
<br/>    
<br/>        陈明的眼皮微微一颤，我本来就不知道路啊，看来，这个小家伙还是知道一些东西的。
<br/>    
<br/>        毕竟是蝉族，曾经一统过星空的强大修仙国度，一个三岁大的小屁孩都这么灵慧，应该是属于那种出生就会说话，会蹦会跳的。
<br/>    
<br/>        陈明挥手，脚下已经是浮现一朵五彩祥云，陈明看向罗牧野，然后道：“上来！”
<br/>    
<br/>        带着罗牧野，有着罗牧野指路，陈明终于是来到了长眠之地的入口，陈明听得到轻轻的蝉鸣声，陈明自然是听得懂这些话，这是蝉族的语言，陈明曾经已经学会了。
<br/>    
<br/>        陈明听到的第一句话，便是：“所有的一切都将长眠，这世间的一切，都会长眠，无论是你，无论是我。”
<br/>    
<br/>        在前方，有着一条长长的通道，这里的一切都是崭新的，应该是时间被凝固在了蝉族破灭前的那一刻。
<br/>    
<br/>        陈明看向通道四周的墙壁，可以看得到，上面绘制着留声阵法，里面传出的蝉鸣声，告诉陈明怎么走。
<br/>    
<br/>        陈明呵呵一笑，呦呵，好手段啊，除了蝉族，谁都进不去。
<br/>    
<br/>        若是走错一步，想必便是身死道消的下场。
<br/>    
<br/>        陈明照着蝉鸣声的指引，飞行了十里地的距离，终于是来到了一扇大门之前，大门如同是蝉的两片羽翼般展开，陈明刚刚落下，罗牧野便一蹦一跳地朝着里面走去，看来已经没有什么危险了。
<br/>    
<br/>        不对，蝉族在大椿的力量下消亡，前方有着一块块树皮，分明就是大椿之上。
<br/>    
<br/>        陈明走进其中，地面上什么都没有，罗牧野道：“空空旷旷的，只有我们了。”
<br/>    
<br/>        陈明跟着罗牧野继续前进，忽然看到在前方有着一具铠甲，陈明走上前去，铠甲上面有着刀痕，类似于两年蝉的，而这一具铠甲，陈明轻而易举认了出来，是瑶池的铠甲，瑶池很久以前的铠甲。
<br/>    
<br/>        看来一切都没有那么简单。
<br/>    
<br/>        陈明道：“小鬼，你听说过瑶池吗？”
<br/>    
<br/>        罗牧野道：“瑶池吗？瑶池的首领乃是我们蝉族的四大镇国大将军之一，怎么了？”
<br/>    
<br/>        原来瑶池也参与过这件事。
<br/>    
<br/>        陈明伸手摸向这铠甲，却是化作了齑粉，不复存在。岁月已经过去了太久了。
<br/>    
<br/>        瑶池曾经对抗过蝉族，那么故事的轮廓，陈明大概也是能够猜到了，大概就是有大劫降临，蝉族被迫将这棵大椿晋升到了五阶的程度，而后，蝉族靠着五阶的大椿抵抗住了大劫，但是却是无法掌控住大椿的力量，受到了大椿的反噬，这个时候，四大镇国大将军趁势起兵谋反，也就造就了如今的四大仙国。

<br/>        陈明伸手，牵着罗牧野的手，走到一座大殿之中，在大殿的墙壁上，陈明看到了一幅画，画上是陈明牵着罗牧野的手。
<br/>    
<br/>        陈明在这幅画前，愣住了，画中人的动作模样，和他们俩现在一般无二。罗牧野也是惊了惊，“咦，好奇怪啊，这画上画的是我和叔叔额！”
<br/>    
<br/>        陈明抬头扫去，在后面还有着十二幅画，但是陈明想要看清这些画的时候，却是发现，这些画已经是瞬间风化消散了。
<br/>    
<br/>        陈明似乎是明白了什么，未来，是看不到的，能够看到的，唯有现在。
<br/>    
<br/>        时间，还真是一种神奇的东西。
