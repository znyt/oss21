        傅夜沉这才想起，是刚刚被他随手扔在地上的那些玫瑰花的花枝惹的祸。
<br/>
<br/>        还好，花枝上的刺，没有断掉扎入她的脚心肉里。
<br/>
<br/>        “都是我的错，不该粗心大意地把那些花枝放在地上。”傅夜沉搂着童筱颖的腰肢，深深地自责道。
<br/>
<br/>        他紧接着毫无征兆地俯身颔首，优雅地吻了一下她的脚背。
<br/>
<br/>        “宝贝，我向你道歉。”他的声音很苏，且认真而诚恳。
<br/>
<br/>        童筱颖眨巴着大大的眼睛，怔忡地望着傅夜沉，一时间竟然没有反应过来。
<br/>
<br/>        他对她总是这般有着无尽的宠溺与呵护，从何时起，让她在不经意间为他沦陷，沉醉在他的甜言蜜语中。
<br/>
<br/>        傅夜沉一边轻轻地揉着童筱颖的脚底，一边抬头，盯住了她白里透红的小脸，关心地问道：“还疼吗？”
<br/>
<br/>        童筱颖心头一愕，猛然回过神来后摇了摇头，并将脚给缩了回来。
<br/>
<br/>        长腿拍打在牛奶浴的水面，激起一片水花溅到了傅夜沉的脸上。
<br/>
<br/>        她怔怔地看着他那张英俊的脸上，缀着牛奶的水珠，但帅气依旧，毫无违和感。
<br/>
<br/>        于是，她又淘气地掬起牛奶和花瓣，泼到了他的身上，唇边漾起一抹俏皮笑靥。
<br/>
<br/>        傅夜沉忍俊不禁，宠溺地捉住她这只淘气的小手，按在了自己的心口上，让她感受着他那强劲有力的心跳。
<br/>
<br/>        她就坐在他的大腿上，身穿的衣服全都湿透了，却依旧是大半个身子和他一起泡在了玫瑰牛奶浴中。
<br/>
<br/>        柔夷下触碰的是他的胸肌，摸上去很结实，令她觉得很有安全感。
<br/>
<br/>        童筱颖微眯起水眸，粉唇翕动，软糯地唤了声：“大哥。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “从今以后，你可不可以，为了我，保重自己？不要留下我一个人。在这世上，除了你，我没有其他亲人可以依赖了。”
<br/>
<br/>        她不希望他像以前那样，不顾自生安危地去涉险解救他人的性命。
<br/>
<br/>        “好。”
<br/>
<br/>        傅夜沉黑眸黯淡，眸光失色。
<br/>
<br/>        原来，他在她心里，只是亲人而已，所以，这就是她一直改不了口的原因吗？
<br/>
<br/>        她唤他“大哥”，是真的把他当亲哥哥了吗？
<br/>
<br/>        但是，他不想当她的哥哥，想当她这辈子、下辈子、下下辈子的老公。
<br/>
<br/>        如果可以，他希望是生生世世。
<br/>
<br/>        童筱颖并未察觉到他心里的异样情愫，忽然间，转开了话题：“如果，我说……”
<br/>
<br/>        “嗯。”
<br/>
<br/>        “那天……”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “就是那天……”
<br/>
<br/>        “那天怎么呢？”他十分耐着性子等待着她的下文。
<br/>
<br/>        童筱颖讷讷地看着傅夜沉的眼睛，还是决定把那天的事情，有必要跟他解释清楚：“就是那天，我没有威胁昼景，也没有要他跟我私奔。”
<br/>
<br/>        “嗯。”他微笑着点了下头。
<br/>
<br/>        童筱颖接着说道：“我只是去给他送抗排斥的药，希望他好好地活下去。”
<br/>
<br/>        “就这样？”他还想听她更多的解释。
<br/>
<br/>        因为，在他看来，她能跟他解释，就证明她在乎他的感受。
<br/>
<br/>        童筱颖决定一口气把话说完：“我跟昼景谎称自己怀孕，是说我怀了别人的孩子不是他的，好跟他彻底结束这段感情。我不知道自己为什么会跟昼景睡在一起，但是，我没有做任何对不起大哥的事情。大哥，我说这些，你信吗？”
<br/>
<br/>        傅夜沉凝视着童筱颖的眼睛，俊脸上露出了发自于内心的笑容，他修长的大手捏住她小巧的下巴，强迫她仰面迎向自己：“信。”
<br/>
<br/>        童筱颖愣神之际的下一秒，唇瓣便突兀地被他悍然噙住。
<br/>
<br/>        她被迫张开的小嘴，被他坚毅的温热牢牢堵住。
<br/>
<br/>        童筱颖瞪着一双大大的水眸，与这男人鼻眼相对，近到彼此能感受到彼此那暧昧的气息。
<br/>
<br/>        他直勾勾地盯着她，丝毫没因她的瞪视而退却。
<br/>
<br/>        哪怕她眸中的惊愕再明显，他也不顾她是否意愿，霸道地侵占她的理智。
<br/>
<br/>        他一直在隐忍，一直在隐忍，但终究也有忍不住的时候。
<br/>
<br/>        此刻，像是有一把火，以最快的速度攻占，燃烧着她所有理智的原野，拉着她一起与他在火势凶猛的燎原中共舞。
<br/>
<br/>        自从她回到他身边后，他的心真的不再痛了。
<br/>
<br/>        她于他而言，就像罂粟，会止痛、会上瘾、戒不掉。
<br/>
<br/>        思绪到这，傅夜沉加深了这个吻。
<br/>
<br/>        她只觉鼻间充斥着他的男性气息，在她心里的已不是小鹿乱撞，而是一种万马奔腾的状态。
<br/>
<br/>        童筱颖满面潮红，快要喘不过气来，她整个人被困在男人宽阔的怀中，挣脱不了，而且根本来不及做出任何举动，就被这个汹涌的吻给吞噬掉了。
<br/>
<br/>        他霸道的吻，不给她丝毫反抗和清醒的机会。
