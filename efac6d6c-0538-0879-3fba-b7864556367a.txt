　　楚风眠三人的遁光极快，在虚空之中穿梭。
<br/>　　几分钟的时间，楚风眠三人就来到了北邙城，这里就是通往北邙学院的入口。
<br/>　　北邙学院是建立在一处单独的空间之中。
<br/>　　本来没有北邙学院的方舟，任何武者都是无法进入其中的，但是对楚风眠等人的实力来说，撕裂空间，是轻而易举的事情。
<br/>　　楚风眠大手一挥，就在这北邙城外，撕裂开了一道空间，灵力凝聚一点，一道空间隧道便是凝聚了出来。
<br/>　　随即楚风眠三人，步入到了空间隧道之中，在其中穿行，很快北邙学院，就落入到了楚风眠的双目之中。
<br/>　　“好强的禁制阵法！”
<br/>　　楚风眠打量了一眼。
<br/>　　这北邙学院如今也是大变样了，上面的禁制阵法，都是焕然一新，这禁制阵法，就连真帝都不敢贸然硬闯。
<br/>　　但楚风眠就连八劫古帝，都说杀就杀，这禁制阵法在他的眼中，就如同是玩具一样的脆弱。
<br/>　　“我们直接进去。”
<br/>　　楚风眠一边开口，一边手心一点，一股灵力打出，从这阵法之上，撕裂开了，打来了一道通道。
<br/>　　楚风眠三人一下子步入到了其中，重新回到了北邙学院之中。
<br/>　　大摇大摆的，楚风眠没有隐藏身形，一下子来到了北邙学院的最深处。
<br/>　　如今这最深处的北邙宫，也就是剑皇殿，之前已经是被楚风眠带走了。
<br/>　　现在北邙学院的最深处，换了一座新的宫殿，看上面的灵力，虽然不如剑皇殿，但也是一件天级灵器。
<br/>　　在这宫殿之中，有着无数强横的气息，显然是整个北邙学院的强者，都聚集于此。
<br/>　　如今在这宫殿之中，一群人正端坐在一起，脸色都不怎么好看。
<br/>　　那北邙学院神秘的院长，也赫然在列，只是他却是只能坐在旁边，不敢坐在主座之上。
<br/>　　北邙学院的许多老祖，大人物，也只能坐在一旁。
<br/>　　像是风皇，夜皇，阳皇，武皇这些北邙学院的高层，如今更是只能是站在旁边，就连座的资格都没有。
<br/>　　坐在主座之上的，却是一名陌生的年轻人，看起来年纪不大，但是一身力量，雄厚到了极点。
<br/>　　在他下方坐着的几名老者，每个人的身上也都散发着浑厚的灵力。
<br/>　　看起来这一群人，如今才是北邙学院真正的主人。
<br/>　　如今每个人的脸上，都是面露这几分阴冷。
<br/>　　“这一次，我们派去去寒雪阁的人，居然都死了，到底是怎么回事？”
<br/>　　那坐在主座之上的年轻男子，脸上露出了几分怒容，冰冷的说道。
<br/>　　“五位古帝，十几名真帝，这种损失，我们陈家来到北邙学院都没有过，怎么会死在一个小小的寒雪阁里。”
<br/>　　这年轻男子的目光看向北邙学院的院长，眼神之中露出几分不善之色道。
<br/>　　“你不是说，寒雪阁里只有一名古帝，还被击伤了吗？怎么回事？”
<br/>　　“陈少主息怒，寒雪阁里的确只有寒月漓一位古帝，这件事的千真万确的。”
<br/>　　那北邙学院的院长，急急忙忙的开口道。
<br/>　　一直以来，陈家虽然入主了北邙学院，但是许多事都是由他出谋划策的。
<br/>　　现在陈家损失如此惨重，如果这陈少主怪罪下来，他也承受不起。
<br/>　　“怎么？难道你觉得，我陈家的五位古帝，就连一个受伤的小姑娘都对付不了？”
<br/>　　听到北邙学院院长的解释，一名陈家的古帝，冷哼一声道。
<br/>　　“当然不是，陈家的诸位强者，都是无敌人物，这一次的事情里，肯定有古怪。”
<br/>　　北邙学院的一位老者擦了擦汗，开口道。
<br/>　　“院长，会不会是楚风眠那个小子搞的鬼？”
<br/>　　突然之间，站在旁边的夜皇开口道。
<br/>　　“楚风眠？楚风眠是什么人？”
<br/>　　听到这陌生的名字，坐在主座上的陈少主眼神一闪道。
<br/>　　“你们还有事，敢瞒着我们？”
<br/>　　“不不不，陈少主息怒，这楚风眠，其实是寒月漓的道侣。”
<br/>　　夜皇紧张的开口道。
<br/>　　“之前他一直是跟寒月漓形影不离的，这一次多半可能是楚风眠，也在寒雪阁之中，此人的资质极高，如果得到了机缘，实力定然会暴涨，可能是因为他，才导致诸位陈家强者的陨落。”
<br/>　　“资质极高？九域的土著一个罢了，算什么东西，九域这样遍地宝藏的一个地方，被一群土著占据，的确是太浪费了。”
<br/>　　陈家的一名古帝不屑的开口道。
<br/>　　“依我看，少主根本不用拉拢什么九域的势力，等到我们陈家的大军一到，横扫整个九域，都不成问题，至于这些九域土著，只配做我们陈家的奴隶！”
<br/>　　听到这话，那北邙学院的一群人眼神之中都露出了几分怒色，但是却无人敢开口反驳。
<br/>　　现在整个北邙学院，都是落入到了陈家的手中，他们能够苟活都不易，谁还敢反抗。
<br/>　　“不管是不是那什么楚风眠搞的鬼，这一次我们陈家众人的仇，不能不报！”
<br/>　　陈少主冷哼一声说道。
<br/>　　“走，我亲自去寒雪阁一趟，等我踏平了寒雪阁，一切都清楚了。”
<br/>　　突然，就在这时，一道冰冷的声音，突然响彻而起，从天空之上传递了下来。
<br/>　　“踏平寒雪阁？我看今日，要被踏平的是你们陈家吧！阎罗星域的人，也给敢来九域？都给我滚出去！”
<br/>　　轰隆！
<br/>　　这声音一出，那在场众人的脸色都是变了。
<br/>　　北邙学院的一种人都是脸色震惊无比，这声音他们再熟悉不过了，绝对是楚风眠的声音。
<br/>　　而陈家的一群人，听到这声音的一刻，脸上的怒意已经是忍耐不住了。
<br/>　　整个宫殿之内，都是大震动。
<br/>　　“什么人？装神弄鬼的？给老夫滚出来！”
<br/>　　一名陈家的古帝，冷哼一声，凝聚灵力在他的掌心之中，猛然一掌横扫而去。
<br/>　　这一掌，横扫了整个宫殿之内，每一个角落，都被他的力量所横扫。
<br/>　　“死！”
<br/>　　这一掌即将横扫过去的一刻，突然之中一道龙爪，从天而降，跟着一掌相对。
<br/>　　猛然之间这出手的陈家古帝，脸色大变，还没有来得及收手，他的身躯就在空中直接炸裂开来，鲜血狂飙。
