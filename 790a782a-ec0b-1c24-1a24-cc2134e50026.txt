　　地元神刀一出。
<br/>　　五行刀圣身上的气势便是跟刚刚，已经是截然不同。
<br/>　　身为一尊刀修，一把好刀，足以是提升五行刀圣不少的实力。
<br/>　　五行刀圣手持地元神刀，一把对着那不灭战意斩杀而去。
<br/>　　不灭战意，毕竟只是血邪魔帝生前的一缕执念武意所化，而非是真正的武者。
<br/>　　在第一时间被压制住之后，便是没有多少反击的机会了。
<br/>　　毕竟在少帝跟五行刀圣这二人的合力之下，哪怕是真正一些半步天人境的强者，都未必足以能够以一敌二。
<br/>　　几道刀光闪过的一刻，这不灭战意便是被轰然打碎，上面的金光，逐渐的消散。
<br/>　　同时在那血邪魔帝骸骨之上，一直蕴含的强横魔意，却是与这不灭战意一同消散。
<br/>　　现在这血邪魔帝骸骨，便是再也不是那种充满威胁的敌人，而是一个毫无反抗，唾手可得的宝藏。
<br/>　　“是我的了！”
<br/>　　五行刀圣长啸一声，身形连连闪烁，一把向着那血邪魔帝骸骨猛然抓了过去。
<br/>　　这血邪魔帝骸骨，已经是没有了任何的反抗之力，这个时候哪怕是楚风眠出现，他一个人就可以从楚风眠轰杀。
<br/>　　当然不在乎还跟那少帝的盟友，先将这血邪魔帝骸骨，抓到手里再说。
<br/>　　嗖！
<br/>　　五行刀圣的身形，猛然冲到了这血邪魔帝骸骨的面前。
<br/>　　可就在他身影之前，大雄宝殿之中的几尊魔族，却是突然出手，阻拦在了他的面前。
<br/>　　这几尊魔族，都是不亚于龙摩罗那个级别的强者，如今正是少帝的帮手，来帮助少帝阻拦这五行刀圣。
<br/>　　足足是七位魔族强者，在这一刻突然突然联手，爆发出强横的力量，滚滚魔意轰击这五行刀圣。
<br/>　　“就凭你们这些蝼蚁，也敢来阻止我？”
<br/>　　五行刀圣冷笑一声，手中的地元神刀，一刀斩出，刹那之间这七位魔族强者联手的力量，都在这地元神刀的面前，被全然打破。
<br/>　　余波的威力，波及在那七名魔族强者身上，这七名魔族强者，都是被震得吐出了一大口精血。
<br/>　　哪怕是七名在九劫古帝之中都是堪称无敌的霸主，在这五行刀圣的面前，都是如此的不堪一击。
<br/>　　这便是半步天人境跟九劫古帝的差距。
<br/>　　一旦是踏入半步天人境，便不再是人了，而是一种半仙一般的存在。
<br/>　　踏入天人境，便是羽化登仙，再也跟凡人不同。
<br/>　　纵然是半步天人境，也要比起九劫古帝强横无数倍，更何况现在五行刀圣更是得到了地元神刀，实力暴涨。
<br/>　　这七名魔族联手，都被他一人轻易打破。
<br/>　　但是这七名魔族，也是早有准备，只轰击了一招，便是纷纷四散而逃，不打算跟这五行刀圣再做纠缠。
<br/>　　就当这五行刀圣，再一次准备去摄取那血邪魔帝骸骨的一刻，少帝的身影，突然显现而出。
<br/>　　这少帝的一掌，向着五行刀圣当即轰击而去，其中强横的力量，将五行刀圣，都逼得不得不退。
<br/>　　而另一边，少帝却是出手，一把向着那血邪魔帝骸骨抓了过去。
<br/>　　一心二用。
<br/>　　身为魔族之中最为耀眼的天才，少帝的实力在这一刻，展现的淋漓尽致，在逼退了五行刀圣的同时，更是要直接摄取了这血邪魔帝骸骨。
<br/>　　“可恶！”
<br/>　　五行刀圣被一把逼退开来，脸上露出了恼怒的神情。
<br/>　　他本来以为，他如今得到了地元刀圣洞府之中的无数宝藏，更是得到了地元神刀，已经是有着跟少帝抗衡的资本了。
<br/>　　可没想到少帝的实力，比起他想象之中的还要强横的多，在被逼退的一刻，便是只能够是眼睁睁的看着少帝，摄取那血邪魔帝骸骨。
<br/>　　“这转轮和尚，怎么还不出手？”
<br/>　　楚风眠在一旁看到这一幕，眼神之中都显露出了几分焦急之色。
<br/>　　他本来是想要等待这转轮和尚先出手，然后楚风眠黄雀在后的。
<br/>　　转轮和尚到底是有着什么底牌，现在的楚风眠都无法确认。
<br/>　　这转轮和尚的来历，无比神秘，他的手上很有可能有着惊天的底牌，楚风眠也不想要冒险。
<br/>　　可是现在五行刀圣的实力，还无法跟少帝抗衡，这样下去的话，那血邪魔帝骸骨定然会落到少帝的手中。

<br/>　　楚风眠看到这少帝即将是要将那血邪魔帝骸骨抓在手心之中的一块，便是不在隐藏。

<br/>　　一剑打出的一刻，漫天剑气，汇聚成一片星河，向着少帝笼罩而去。
<br/>　　贪狼，巨门，禄存，天机！
<br/>　　九玄剑术的前四式，在这一刻突然爆发而出，化为星河，笼罩在了少帝的身躯之上。

<br/>　　所以现在楚风眠，不得不出手了。
<br/>　　“楚风眠，你终于出现了！”
<br/>　　看到楚风眠现身，少帝大笑一声，直接放弃摄取那血邪魔帝骸骨，而是向着楚风眠猛然冲了过来，滚滚魔意凝聚在他的手心之中，一道魔掌，铺天盖地向着楚风眠轰击而来。
