　　第569章
<br/>　　啪
<br/>　　青釉茶杯啪地一声落地，碎成稀巴烂，楚云生的手被碎瓷割破，流出鲜红的血来。
<br/>　　熬到现在，他本人已经困了，茶水早已凉透，但他一双眼睛，却冒着灼热的光，恍惚能将茶水再煮沸一遍。
<br/>　　如此不是因为他没想到法子，而是被脑海中一个念头彻底击中。
<br/>　　只要想得时间足够长，便一定能想到所需要的办法。
<br/>　　只是这个念头太大胆，太猖狂，也太罪孽深重。
<br/>　　那么，做不做？
<br/>　　答案不言自明，从他瞳孔中的灼灼的火光，已能想到答案。
<br/>　　有些人默不作声，随时都会被忽略，永远不会被当做主角。甚至，他心里也觉得，自己是个无足轻重的小角色。可是，某种骨子里潜藏的东西，却不会轻易消失。它们藏着，窝着，却随时准备跳出来。
<br/>　　把所有人吓一跳。
<br/>　　楚云生就是这种人。
<br/>　　他口才不是很好，艺术方面也没什么天赋，沉默寡言所以没什么朋友。于是最习惯做的事情，就是默不作声的想。
<br/>　　这么多年，谁知道他想了多少坏主意。
<br/>　　但任何一个念头，肯定也没办法和此时相比。这个念头够毒、够损，够他死后在十八层地狱挖下水道。
<br/>　　书院最重要的存在是什么？是德高望重的夫子么，是以七贤十二秀为代表的书院弟子么……不，不是他们。虽然他们也很重要，但绝不是最重要的。
<br/>　　这里是万卷山，这里是万卷山上的万卷书院，最重要的永远是藏书不止万卷的藏书楼。
<br/>　　那一日，三人一起上山，之后程大雷就不怎么见得到楚云生。程大雷还以为楚云生是陷入各路诸侯安排的酒池肉林中脱不开身。
<br/>　　真相自然不是这样，事实上，楚云生进入书院后，但凡有时间，便窝在藏书楼。书院耗费心血著书藏书，都在这栋万卷楼中。楚云生如饥似渴阅读着这些前辈先贤留下的书籍。在程大雷和刘发财眼里，这可能就是一堆破纸。但在楚云生眼里，这里的每一本书都价值千金，连城不换。
<br/>　　历代读书人呕心沥血，散尽家产，为的就是留下一本，将前人的经验思考传下去。
<br/>　　此刻，楚云生立在藏书楼中。
<br/>　　对于这样的重地，书院平日不会不派人严加看守。只是今天的状况的确有些意外，为了在白沙岛排兵布阵，书院高手被抽调走了好多。又有程大雷和燕不归两尊魔头闯山，所以绝大多数人都去围杀二人。而今日守楼的，竟然一个人都没有。
<br/>　　所以楚云生一瘸一拐的，轻而易举的进入藏书楼。
<br/>　　这还真是天时地利人和呐。
<br/>　　他的手拂过一排排堆满书籍的书架，故纸堆散发着腐朽味道，此外还夹杂着樟脑药草的气味。对楚云生来说，这实在是天下第一等迷药，让人沉浸其中，不想醒来。
<br/>　　很多书都是绝版，只在书院有，往日楚云生只是听说过，如果有机会看上一眼，当真是做梦都会笑醒。
<br/>　　那本《千草谱》是一位名医，走访名山大川，写就的一份草药名录。
<br/>　　那本《域外堪舆图》乃是走访来到帝国的异国客商，画的一份海经图。
<br/>　　……
<br/>　　每一本都堪称无价之宝，秀才不出门，凭什么知道天下事，凭的不就是这一本本历代先贤心血的结晶么。
<br/>　　而楚云生竟要亲手毁掉这里。
<br/>　　他是个读书人，他也嗜书如命，他也冬卧雪，夏盗光，捧着一本辛苦得来的书如饥似渴。可如今，却要由他毁掉这里。
<br/>　　他陷入深深犹豫中。
<br/>　　不，他一点都没有犹豫，这看他着魔般的双目就知道。所谓犹豫、纠结，不过是自我安慰的情绪而已，他只是觉得自己需要纠结一下，于是便纠结一下。
<br/>　　从这个念头从心里出现，就再也无法遏制。他要毁掉这里，亲手毁掉这里……此刻，救程大雷的原因倒不占太大比重，他就是要毁掉这里，
<br/>　　有人看着温文尔雅，骨子里却是个亡命徒。满口子仁义道德，一肚子杀人放火。
<br/>　　楚云生就是这种人。
<br/>　　他从怀里掏出火石，咔嚓，咔嚓……干咽了口吐沫，心脏砰砰直跳，就如老饕面对一坛刚开封的美酒，一头色鬼面前有位一丝不挂的处子等他开垦。
<br/>　　咔嚓
<br/>　　撕碎的书页被点燃，火苗腾地一下窜起来。天干物燥，本该小心火烛的，可书院很明显没有消防安全的意识。而今夜的风又格外大些……
<br/>　　火光照亮了楚云生的脸，脸上呈现一种异样的红。如同老饕喝了美酒，色鬼心满意足……然后，他觉得自己该悲伤一下。
<br/>　　于是他就悲伤了一下。
<br/>　　万卷藏书，前辈读书人心血的结晶，就被自己一把火毁了。
<br/>　　呃……还是别悲伤了，自己赶紧溜了要紧。不然自己被困在此地，那就是引火烧身了。
<br/>　　程当家，该做的我都已做了，您的恩情我已还清，山水有相逢，咱们日后再见了。
<br/>　　楚云生不得不快点溜，若溜得慢了，被别人发现是自己干的。那么全天下读书人的吐沫星子都能淹死自己。
<br/>　　一枚被所有人忽略的卒子，拱啊拱，一字拱入田字格，终于，谁也没办法忽略他。
<br/>　　合适的时间，出现在合适的地点，于是，改变所有。
<br/>　　程大雷的境地越来越窘迫，在腹背受敌的情况下，他也陷入和燕不归同样的状况。
<br/>　　夫子余怒未消，举着剑在外面喊道：“来，给我杀了他，杀了他。”
<br/>　　张明理无语，觉得夫子还是要在意一下气质。
<br/>　　正这时候，山顶火光升起，风助火势，火借风威，熊熊大火，山头成了火炬。
<br/>　　所有人都愣住了，包括被围攻的程大雷，大家都看向着火的地方。
<br/>　　“那是什么方向？”夫子失声问。
<br/>　　“看着好像藏书楼呐。”刘发财道。
<br/>　　程大雷愣了愣，忽然捶胸大笑：“哈哈哈，老子命不该绝，老贼，看到没，本当家气数未尽，天都在帮我。”
<br/>　　夫子一激灵，作为天下读书人的魁首，他还是很在意气数之类的东西。这凭空而降的天火……莫非，程大雷当真气数未尽。
<br/>(本章完)
