        傅昼景看到童筱颖后，会心一笑，只身朝她走了过去。
<br/>
<br/>        当有几个女生你推我我推你地想要上前去找他要电话时，看到一个女人疾步扑入了这个帅男人的怀里，不约而同地止住了脚步，用艳羡的目光落在了那个女人身上。
<br/>
<br/>        童筱颖紧紧地抱住了傅昼景，完全没意识到自己已经成为了众矢之的。
<br/>
<br/>        果然，她可以对所有男人感到不适，却唯独只有他，她才愿意敞开心扉地去拥抱。
<br/>
<br/>        “昼景，我好想你……真的真的好想你……”童筱颖回想起他出差后的这段日子，她过得一点都不顺心，鼻子一酸，泪水瞬间充溢了整个眼眶。
<br/>
<br/>        只有在他的怀里，她才能感受到一丝的安心与温暖。
<br/>
<br/>        傅昼景爱怜地搂着童筱颖，隐隐地察觉到她的身子在微微颤抖，便低头看了看她，抬起手来轻轻地捧住了她的脸颊，仰面看向自己。
<br/>
<br/>        “我的傻丫头怎么哭呢？谁欺负你了？”傅昼景温柔地问道。
<br/>
<br/>        童筱颖微微摇头，看着傅昼景那双澄澈的眼睛，破涕为笑：“没有谁欺负我，只是单纯地想你了！”
<br/>
<br/>        “我也想你。”傅昼景看着童筱颖的眼睛，真挚地回应，然后微微低头，吻了吻童筱颖的眉心，安慰道，“肚子饿了吧？我带你去吃这名扬城里最好吃的美食！”
<br/>
<br/>        “嗯！”童筱颖开心地点了点头。
<br/>
<br/>        他温厚的大手，牵着她纤瘦的小手，替她拖着行李箱，往他座驾那边走去。
<br/>
<br/>        童筱颖看着傅昼景牵着自己的侧影，心口处打起了小鼓。
<br/>
<br/>        在她的眼里，他仍旧一如既往的帅气迷人。
<br/>
<br/>        如果能被他牵一辈子，直到白发苍苍……
<br/>
<br/>        童筱颖心里想着想着，情不自禁地挽住了傅昼景的胳膊，将脑袋靠在了他的肩头上。
<br/>
<br/>        傅昼景嘴角微扬，睐了童筱颖一眼后，索性直接抬手揽她入怀，搂着她继续往前走。
<br/>
<br/>        一起吃这名扬城里美食，一起又去逛了这名扬城里的夜市。
<br/>
<br/>        童筱颖吃饱喝好玩足后，刚上车没多久，便坐在副驾驶座上沉睡了过去。
<br/>
<br/>        傅昼景坐在驾驶座上开车，载着童筱颖直接去了酒店。
<br/>
<br/>        本来，他和季思妍是住一套间房，筱颖来了，自然也就没有她季思妍的份了。
<br/>
<br/>        傅昼景让季思妍自己去另一个酒店，单独另外再去开一间房。
<br/>
<br/>        季思妍故意将房间开在了傅昼景和童筱颖的房间的隔壁，等到他俩从外回来时，她也刚巧从隔壁房间里出来。
<br/>
<br/>        三人在走廊上打了个照面。
<br/>
<br/>        傅昼景本不想和童筱颖介绍季思妍，岂料童筱颖忽然拉住了他的手，轻轻地摇了摇。
<br/>
<br/>        “昼景，她……”童筱颖微微一笑，欲言又止。
<br/>
<br/>        她记得这个女人，上次，她亲自来家中取过昼景的资料。
<br/>
<br/>        傅昼景下意识地看了季思妍一眼，剑眉微蹙，而季思妍却落落大方地向童筱颖伸出手来。
<br/>
<br/>        “你好，我叫季思妍，是傅总的秘书。”季思妍彬彬有礼地自我介绍。
<br/>
<br/>        童筱颖抬起手来和季思妍握了握，微笑着回问：“你好，我叫童筱颖。”
<br/>
<br/>        “嗯，冷夫人好。”季思妍颔首，明眸里却闪过一丝狡黠。
<br/>
<br/>        童筱颖被她这一声尊称唤得心里怪变扭，于是微笑着对季思妍说道：“你叫我筱颖就行了，冷夫人的称呼，听着怪怪的。”
<br/>
<br/>        “怎么会，我觉得她叫你冷夫人挺好。你是我傅昼景的太太，自然是冷夫人。”傅昼景随之将童筱颖搂入怀中，并从裤袋里掏出房卡刷卡进门。
<br/>
<br/>        季思妍心里隐隐作痛，瞬间醋意大发地说道：“冷夫人，你不在的这段时间，傅总他……”
<br/>
<br/>        傅昼景朝季思妍瞪了一眼。
<br/>
<br/>        童筱颖的目光一直落在季思妍的身上，并未察觉傅昼景的异常。
<br/>
<br/>        季思妍对上傅昼景的目光，会心一笑地接着说：“傅总他经常跟我们这群下属说，很想吃冷夫人做的饭菜。”
<br/>
<br/>        “那回去后，我天天给你做！”童筱颖随之看向傅昼景，双手搂住他的腰肢，喜上眉梢地说。
<br/>
<br/>        傅昼景干笑着点了点头。
<br/>
<br/>        季思妍又说道：“傅总还给冷夫人买了很多礼物了，冷夫人待会不要太惊喜噢！”
<br/>
<br/>        “是什么礼物？”童筱颖看了看季思妍后，又仰头看向傅昼景。
<br/>
<br/>        傅昼景不再看季思妍，微微低头注视着童筱颖的眼睛，温柔地回答道：“进屋去，你就知道了。”
<br/>
<br/>        “嗯，那么季小姐，晚安。”童筱颖向季思妍挥了挥手。
<br/>
<br/>        季思妍礼貌性地回礼挥手，嘴角噙着意味不明的笑意。
