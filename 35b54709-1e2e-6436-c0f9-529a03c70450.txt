　　大伙又跟着纷纷点头。
<br/>　　“哎呀我不行了，这绕来绕去的，我脑子不行，别让我思考这么复杂的问题行吗，还总是吓我，你就直接说结论，是谁！”小马一个劲地催促道。
<br/>　　小白道：“小马你滚蛋，大家不是一起在分析吗，就这点线索，军师能想到这一层就不错了，哪里能具体到某个人的身上。”
<br/>　　林三生目光闪烁，一副欲言又止地样子，瓜瓜站的位置本来靠门边上，突然皱起眉头，把耳朵贴在门上，然后猛地一下拉开门，一道人影朝远处遁去。
<br/>　　有人偷听！
<br/>　　大伙连忙追出去，人早就不见了。
<br/>　　“是人，有脚步声，而且只有人类靠近，我们会全无察觉。”叶少阳分析道。
<br/>　　那么问题来了，刚才门外的人是谁？
<br/>　　大伙带着震惊讨论起来。
<br/>　　“好了，不讨论这个了，军师已经分析到这了，大家回去都思考下，散了散了。”叶少阳打发大伙离开，最后只剩下瓜瓜和包子，一左一右蹲在他两边肩膀上。
<br/>　　“出去走走？”叶少阳对一直坐在椅子上没动弹的林三生说道。
<br/>　　两人一起来到御花园。叶少阳让包子和瓜瓜去附近转转，仿佛有人偷听，然后跟林三生在园子里慢悠悠地走起来。
<br/>　　说是御花园，但这花园巨大，不夸张地说，跟个森林似的，中间还有一个巨大的湖。
<br/>　　虽然这一切都是仿造古代的，但毕竟历史上真实存在过，看上去是那么十分气派，假山怪石，楼台水榭，画面不要太美。
<br/>　　林三生带着叶少阳到处走动，不知不觉融入到了情景之中，指着前方一片潋滟的湖面，说道：“这里，我以前来过。那时候湖边还没这么多林子，但大体的布置，我还是记得的……”
<br/>　　“你来过御花园？”叶少阳震惊。
<br/>　　“建文年间，这个园子就有了，只是没这么精心打理过，那时候京城还在南京，我记得清楚，建文二年秋天，皇帝出巡到北京来，召集了一批学子讨论策论，我有幸在其中，那天，皇帝约了我们几个来这里游玩……”
<br/>　　林三生目光在湖边搜寻，找到了一个位置，伸手指过去，“就在那边，皇帝找了我们三个人坐下聊天，当时聊的便是燕王朱棣坐拥雄兵，问我们如何应对……
<br/>　　我回答说燕王必反，应当先下手削藩，皇帝听了我的意见，后来果然燕王造反，只可惜皇帝那时候优柔寡断，又念及亲情，这才导致国破……”
<br/>　　林三生触景生情，不免感慨。
<br/>　　叶少阳道：“其实我一直有个疑问，你为什么不给朱棣当官呢，反正都是姓朱的，都是太祖的后代……”
<br/>　　林三生笑道：“你不是古代人，不明白的，我们做臣子的，都是一根筋的，举个例子，如果你有个兄弟，用武力把你赶下台了，难道捉鬼联盟就能认他做老大不成？”
<br/>　　好像还真是这个道理……
<br/>　　林三生叹道：“再回首已百年身，现在何止百年过去了，那时候我一心想着复国，哪里会想到，几百年后，变成了这个样子……”
<br/>　　“好了好了，别感慨了，咱们谈正事吧。我这听你吐槽半天了。”叶少阳冲他嘻嘻一笑。
<br/>　　“不是都说好了吗，大家一起思考线索。”
<br/>　　叶少阳走到他对面，盯着他的眼睛，道：“你骗的了别人，骗不了我，军师……你已经知道是谁了是吧，为什么不能说？”
<br/>　　林三生目光闪了闪，微微低头。
<br/>　　叶少阳追问：“是不是之前在我们房间外面偷听的那个？”
<br/>　　林三生点头默认。
<br/>　　“果然……这人还真的在我们身边啊。”
<br/>　　“既然你追问，我也不瞒你，我是想到了一个最可能的人，少阳，这个人是在我们阵营，但肯定不是我们这些——星月奴不熟悉我们，鲲鹏更不熟悉，所以这个人，一定是法术界的前辈，有地位，有实力，还跟星月奴打过交道，这一点非常重要，如果鲲鹏真的查找起来，一定能找到这条线索……”
<br/>　　叶少阳听他说完，反问道：“鲲鹏是星月奴的人，她能查到的线索，咱们未必能查到吧。”
<br/>　　林三生道：“没那么费事，通过前面几条，就已经能猜到是八九不离十了。”
<br/>　　叶少阳心中一动，照他前面说的那些想了一下：法术界前辈，有地位，有实力……这样的人物，在人间还是有一些的，但是在这里，似乎想都不用想，那个名字一下子就跳了上来。
<br/>　　叶少阳深吸一口气，瞪大眼睛看着是林三生，摇了摇头：“不可能吧！”
<br/>　　“倘若真的是他，你怎么办？”
<br/>　　叶少阳懵了。
<br/>　　是啊，如果真的是他……自己要怎么办？叶少阳的心一下子就乱了。
<br/>　　“九星阁在他身上，能取下来吗？”
<br/>　　“如果是在脏器里，你怎么取？”
<br/>　　“那个，去医院什么的，应该是有办法的……”话没说完，叶少阳就知道自己想多了。
<br/>　　果然林三生补充道：“这里没有医院，想回我们的世界，就必须先灭了九星叠气阵，不然就得一直困在这。”
<br/>　　这是一个矛盾的悖论。
<br/>　　叶少阳沉默片刻，冷静下来，道：“难道只有杀人？”
<br/>　　林三生没正面回答，说道：“我们可以作法让肉身钻进洞府，但没办法让肉身钻进别人的肉身，如果仅仅只是用魂魄进去，那还是不用进去了。”
<br/>　　叶少阳缓缓点头，如果只能是魂魄形态，那自己这批人中，大部分都会废掉大半实力，然后还要在星月奴精心布置的洞府里跟她打，这不叫决斗，这是自杀。
<br/>　　“那……没有办法了吗？”叶少阳一时间有点六神无主。
<br/>　　林三生没回答这个问题，因为叶少阳已经有了答案。
<br/>　　“我到想明白之后，才真的知道星月奴有多狠……不站在任何立场地说，这一手堪称完美，因为就算我们一路解谜，到了这最后一步，却还是没办法动手……叶少阳，你不用犹豫，你是下不了手的。”
<br/>　　（明天公布一个算是剧透的消息）
