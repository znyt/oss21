　　第742章
<br/>　　这陈梦名不见经传，程大雷脑袋里根本不曾搁过这么一个人，但他竟然能与马孟起战个平手。
<br/>　　马孟起是绝世等级，陈梦也是同样境界，关键是他还不足二十岁，日后成就如何，想想也令人恐怖。
<br/>　　帝国还是有高人的，程大雷一直有种高高在上的心理，此时不得不承认，这个想法是错误的。
<br/>　　如今二人斗得不可开交，谁也占不到上风。程大雷看得目不转睛，观看这样的战斗，对他的修行也是有益的。
<br/>　　马孟起用一杆铁枪硬战平野虎陈梦，双手握枪，腰马合一，一身力气凝于一点，狠狠向对方刺去。这招直枪，只不过是普普通通的马上枪术，任何人只要学过两天把式，就懂得这招。但懂得是一回事，能够用好是另外一回事。
<br/>　　程大雷也算有些见识的，但能将一招直刺的威力发挥到极点的，唯马孟起一人而已。所谓绝世高手，不是懂得什么了不起的招数，而是能将普通的招式一遍遍重复，练到巅峰。
<br/>　　马孟起如此，陈梦同样如此，天赋是一回事，但能到他这种程度，也需要日复一日的苦功夫。
<br/>　　双斧一架，挡住了这一枪，由上一推，继而一剪，这一挡，一推，一剪便将马孟起逼退。
<br/>　　程大雷看得入神，忍不住去想，如果自己分别于二人交战，该如何应对。二人的马术、力气、眼界都不在自己之下。不过程大雷也同样有二人不及之处，他的经验要比二人多。
<br/>　　到现在，二人都打红了眼，至于为何而战，到现在也顾不上考虑，先打赢了再说。
<br/>　　马战未分出胜负，从马上又斗到步下，二人要比拳脚。
<br/>　　这是拳拳到肉的打法，谁也是不管不顾。相比之下，马超的打法更野蛮些，出招大开大阖，却又简单直接。而陈梦的功夫更加扎实，一拳一脚，都浑然天成，很明显是高手教出来的。
<br/>　　二人马战没有分出胜负，步战也未必能论出高低。再打下去，怕是要打个三天三夜，看哪一方先力竭。
<br/>　　程大雷决定管一管闲事。
<br/>　　“呔，二位停一停。”
<br/>　　马陈二人打得难分难解，似乎是根本没听见程大雷的话，也或许是听见了不在意，依旧缠斗在一起。
<br/>　　程大雷向前一步，右手接住了马超递过来的炮拳，左肩一靠，将陈梦顶在一旁，小臂挡住他的拳头。
<br/>　　一龙分二虎。
<br/>　　“呔，恰逢乱世，二位好汉有如此本领当为国家出力，何故如泼皮一般在此缠斗。”
<br/>　　程大雷说罢，脊背挺直如山，自己也觉得有几分高人风范。昔有刘玄德一龙分二虎，今日自己怎么不能收服这两员大将。
<br/>　　谁料二人冷哼一声，异口同声吐出两个字。
<br/>　　“滚开。”
<br/>　　马孟起压低身形，一拳砸向程大雷小腹。陈梦来了一招贴山靠，撞向程大雷肩膀。
<br/>　　来势汹汹。
<br/>　　程大雷倒抽一口凉气，急忙撤身，往后急退了三步，才避开二人的攻击。
<br/>　　程大雷有匹夫剑，鬼面斧，马上步下都算过得去，但这拳脚功夫委实没研究过。其实蛤蟆寨的诸多武将，都是长于战场厮杀，而不通拳脚格斗。
<br/>　　他这一避，避得险，也同样避得巧。
<br/>　　陈梦和马孟起同时眯起眼睛，本来他们不大看得起程大雷，此刻方才知道他也是个有本事的。
<br/>　　“来，打一场。”
<br/>　　“先收拾掉这多管闲事的。”
<br/>　　二人不约而同，几乎同时向程大雷攻来。三人本领都在伯仲之间，程大雷应对得狼狈不堪，现在他悔得肠子都青了。瞧热闹便是瞧热闹，自己何必要多管闲事，现在倒好，弄成引火烧身以一敌二的结局。
<br/>　　“呔。”
<br/>　　程大雷大喝一声，不得已而拔剑，连续两剑，一剑斩断陈梦衣领，一剑斩掉马超头上一缕黑发。
<br/>　　他连续往后退了几步，持剑而立，到现在程大雷也有几分火气。
<br/>　　“不要命了么。”
<br/>　　毕竟是手下留情，否则程大雷出剑杀人也就是杀人了。程大雷总归是怜惜二人年少，莫未出头便夭折在这里。
<br/>　　马超将头发系在脑后，陈梦咧了咧嘴，刚才挨的这一剑份外凶险，但二人眼里只有见强则强的战意，并没有半分惧意。
<br/>　　二人目光上上下下将程大雷打量一遍，陈梦忽地皱起眉头：“你可是姓程？”
<br/>　　在汝南城露了身份，走到此地是想藏也藏不住。程大雷喝了一声：“某家行不更名坐不改姓，正是凉州程大雷。好话与你说不听，当真要与本当家动手么。”
<br/>　　声音落地，只见陈梦乐得哈哈大笑，马孟起眼神炙热，看程大雷像饿夫看一块肥肉。
<br/>　　程大雷一激灵，这二人脑袋莫非有些不清楚，天子尚且避醉汉，如果他们真得了癔症，自己也不能和他们计较太深。
<br/>　　陈梦捡起了自己的双斧，大声道：“姓程的，你听仔细了，小爷正是平野虎陈梦，在这里候你多时，快快快，上马与我一战。”
<br/>　　马孟起持枪而立：“某内陛下的御前侍卫，统领鱼龙卫，奉陛下之命接你赴京。”
<br/>　　程大雷点点头，马孟起显得还会说话。
<br/>　　“你老实上马，让某家一枪把你挑落马下，陛下有言在先，我不会要你性命的。”马超。
<br/>　　程大雷长吸一口气，压住胸中翻腾的火气。当真是出门没看黄历，撞见这两个憨货。
<br/>　　“你们算那个，也有资格同本当家动手，先去混个十几年，在江湖上闯出了名号，再来同本当家较量。”
<br/>　　程大雷已经有些不开心，言语便不那么客气。
<br/>　　“贼人休狂，且让我一斧劈了你的脑袋。”
<br/>　　陈梦压抑不住火气，也没有什么耐性，冲程大雷奔来，专门找程大雷脑袋。
<br/>　　程大雷眯起眼睛：杀心忒重了吧。
<br/>　　长剑递出，攻陈梦的上三路，脚下移动，避开对方的攻击。那马孟起持枪立在一旁，也未趁机攻击程大雷。
<br/>　　十八路乱砍柴斧法，比程大雷斧法要繁杂些，但也有限。程大雷仗着轻身功夫与长剑锋利，瞅准对方空隙，一剑递了出去。
