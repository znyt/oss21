　　这一场战斗，简直是北邙圣域中，数年都难得一见的。
<br/>　　其中的每一刻，都被无数人注视这。
<br/>　　在其中一道身影，被打飞出去的一刻，如今便是无数人都看了过去。
<br/>　　可是下一刻，当看清这身影的一刻，周围的无数众人，一下子都是目瞪口呆起来。
<br/>　　被打飞的人，不是他们心中所想的楚风眠，而是龙未央。
<br/>　　要是这飞出去的人是楚风眠，在场的任何人都不会觉得奇怪。
<br/>　　龙未央可是北邙学院的核心弟子，超越了种子弟子的人，是北邙学院中真正的巨头。
<br/>　　而楚风眠，却只是一个小小的种子弟子，甚至是才刚刚晋升的一位新人。
<br/>　　这样悬殊的力量，如今被打飞出去的人，居然是龙未央？
<br/>　　这简直是超出了在场任何一个人的估计。
<br/>　　这种实力，真的还只是一个新人？
<br/>　　如果不是楚风眠动用的武技，是龙族武技，在场的许多人，只怕是都还以为，龙未央的对手，是夜天君。
<br/>　　因为除了夜天君外，在核心弟子之中，都没有几人，敢说可以在力量上，赢得过龙未央。
<br/>　　古傲寒，脸色极为阴沉。
<br/>　　这一幕，也远远是在他的意料之外。
<br/>　　“可恶，本来以为龙未央可以解决了这小子，没想到这小子居然真的有跟龙未央抗衡的实力！”
<br/>　　古傲寒心中暗自想到。
<br/>　　他本来便是想要坐山观虎斗，想要借助龙未央的手，轰杀了楚风眠。
<br/>　　他想要追求寒月漓，楚风眠便是一个必须要死的人。
<br/>　　尤其是现在，楚风眠身上表现出来的天资，更是令他的心中都是极为慌张。
<br/>　　楚风眠的确是一个新人，不错，在进入到北邙学院前，几乎是在北域都没有任何一点名气的人物。
<br/>　　尤其是在刚刚加入到北邙学院的时刻，楚风眠的实力还很弱小，那个时候楚风眠甚至都不是圣者。
<br/>　　这么短的时间内，如今楚风眠的实力居然是足以成长到这一步，这简直是超出了他的认知，也是让他的心中，起了必杀的心思。
<br/>　　“此子决不能留！”
<br/>　　古傲寒的目光闪烁，已经是蕴含有着几分杀意，似乎是要计算这什么。
<br/>　　不止是古傲寒，几乎这所有看到这一幕的众人，都是一样的吃惊。
<br/>　　当然最为吃惊的人，却是龙未央自己。
<br/>　　龙未央拥有的龙王血脉，足以克制一切龙族，龙族在他的面前，只是被克制的存在。
<br/>　　可是如今他的龙王血脉，在楚风眠面前，却是根本发挥不出任何的作用，反而是楚风眠的力量，居然是隐隐约约的令他都感觉到了畏惧。
<br/>　　他可是核心弟子，北邙圣域中的巨头，如今居然被楚风眠这样一个新人打飞，这让他根本便是颜面尽失。
<br/>　　心中怒意无比，疯狂的想要洗刷这等耻辱。
<br/>　　“小子，我承认你的确有着几分实力，我有些小看了你，不过今日，你还是必败无疑，能够见识我的真正实力，也算是你的荣幸了！”
<br/>　　龙未央长喝一声，突然便是仰天长啸起来，这一声长啸，便是一声声的龙吟。
<br/>　　龙未央的身躯，如今都开始疯狂的膨胀了起来，几乎是一瞬间，他的皮肤被撑破，露出来的，却是坚韧的龙鳞。
<br/>　　瞬息之间，在空中，龙未央便是化为了一条真正的龙王，血脉尊贵无比的龙王。
<br/>　　“龙王！这是龙族的龙王！”
<br/>　　“这才是龙未央的真正实力！”
<br/>　　一尊尊武者，看着天空上出现的龙王，高声议论起来。
<br/>　　化身成为龙王，龙未央身上的气息，比起刚刚，还要强大数倍，这才是将他身上血脉力量，催动到极致的地步。
<br/>　　一股股尊贵至极的龙威，围绕在他的身边，令人感觉到，都有着一种想要臣服的感觉。
<br/>　　龙，本身便是皇权的象征，龙王，更是万龙之王。
<br/>　　在这龙王面前，一些实力稍弱的武者，只怕都不敢与之抗衡，感觉到上面的气息，都会败退。
<br/>　　龙未央可以建立起龙党，成为巨头，依靠的，便是这龙王之力。
<br/>　　“龙未央，都被逼出全力了。”
<br/>　　北邙圣域的深处，几尊长老，都在喃喃自语，议论起来。
<br/>　　“楚风眠这个小子，只是一个新人，怎么会拥有如此强大的实力来？”
<br/>　　一尊长老，低声自语，似乎有些无法理解。
<br/>　　“我查到，此人是在外，得到过不少奇遇，更是得到进入过颜古龙帝的洞府，应该是大有收获，不过他的气息暴涨，却是在进入到众圣堂中的。”
<br/>　　又一尊长老，缓缓的开口道。
<br/>　　“七日前，众圣堂中，又一道传承出现的，应该是被此人得到了。”
<br/>　　“哦？众圣堂的传承？众圣堂已经是十年，没有被人得到过传承了，是谁？龙族武技，是赤龙君？还是孽龙天尊？”
<br/>　　又一尊长老，连连念出几个名字来。
<br/>　　这些名字，都是众圣堂***奉的北邙学院历代强者中，学会龙族武技的人物。
<br/>　　“都不是。”
<br/>　　这长老，缓缓的开口道。
<br/>　　“是创始者。”
<br/>　　“谁？”
<br/>　　一些长老，有些不可置信的说道。
<br/>　　“是我们北邙学院的创始者，北邙圣君！”
<br/>　　“北邙圣君？”
<br/>　　“怎么可能！”
<br/>　　“北邙圣君的雕像，虽然一直在众圣堂中，但是从未有人参悟出来其中奥秘，更何况传承，此人怎么可能得到！”
<br/>　　无数长老，皆是不可置信的大喊出来。
<br/>　　这个消息，令他们这些长老，都是震惊无比。
<br/>　　北邙学院的创始人，便是北邙圣君，是真正的无敌人物，几乎位列九域巅峰。

<br/>　　别说是传承，几乎都从未有人参悟出其中武道来，楚风眠怎么会得到传承。
<br/>　　“你说的，可是真的？”
<br/>　　一尊长老，脸色凝重的询问道。
<br/>　　“这个消息，可不能有假。”
<br/>　　“当然为真，这雕像，七日前显圣，便是只有此人在众圣堂的最深处。”
<br/>　　这位长老，缓缓的开口道。
<br/>　　“这个消息，还没有传递出去，不过眼前这战斗，诸位可是要干预一下了。”
