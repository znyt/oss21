<br/>    跟着四个劫匪弯弯绕绕，在大街小巷之中穿梭，陈明都有些不耐烦了，终于是到地方了。
<br/>    
<br/>        话说，这么久才能来到一个僻静的地方，这些劫匪真的吃得起饭吗？
<br/>    
<br/>        陈明走进四个劫匪所说的院子中，院子中确实没有什么人气，芳草萋萋，一棵棵小树杂乱无章。
<br/>    
<br/>        陈明第一个走进院子中，门外的四个劫匪却是才刚刚进来，陈明扫了一眼，很是生气，“你们就不知道把门关上吗？”
<br/>    
<br/>        刀疤脸连忙转身，将大门关上。
<br/>    
<br/>        陈明再次感应了一番，认真道：“不对，还缺一个小范围的阵法隔绝外界。只有这样，才能做到绝对安全。”
<br/>    
<br/>        刀疤脸拍了拍手，手下一个劫匪展开了一个小范围的隔绝阵法。
<br/>    
<br/>        陈明这才满意地点头，“嗯，对，这样就差不多了。”
<br/>    
<br/>        刀疤脸道：“说实话，我还是第一次见到你这样的修行者，竟然在劫匪的带领下来到了必死之地。”
<br/>    
<br/>        刀疤脸走向陈明，正欲开口道抢劫，却是感觉到一阵巨大的压力朝着他压下，不过片刻，就已经被压倒在地，再也起不来。
<br/>    
<br/>        刀疤脸艰难地看向周围，却是发现身旁的其他三人也是起不来了。
<br/>    
<br/>        而那威压的源头，便是眼前的这个看起来只有悟道境的修行者。
<br/>    
<br/>        陈明低头笑着看向被他的起势压倒在地的劫匪，“说实话，我还是第一次见到你这样的修行者，竟然亲自带着敌人来到了必死之地。”
<br/>    
<br/>        刀疤脸艰难道：“仙人饶命！仙人，在下，愿意用我的全部身家，换取自己的性命。”
<br/>    
<br/>        亲娘咧，我到底招惹了一个什么老怪物啊！
<br/>    
<br/>        陈明笑道：“我对你的全部身家没有什么兴趣。”
<br/>    
<br/>        刀疤脸的脸上露出笑意，然后他就听到陈明幽幽道：“不过，我对你的命，还真是感兴趣呢！”
<br/>    
<br/>        陈明张开手，摸出一枚苦瓜味的气血丹，直接塞到了刀疤脸的嘴里，伸手在刀疤脸的后背一拍，已经是落入了刀疤脸的口中。
<br/>    
<br/>        陈明如法炮制，给四人都喂了一颗气血丹。
<br/>    
<br/>        刀疤脸看向陈明，惊惧道：“你给我们吃了什么？”
<br/>    
<br/>        陈明道：“断肠丹？离魂丹？七日绝命丹？管他呢，反正不是什么好东西。”
<br/>    
<br/>        陈明收起威压，抬腿在一块青石上坐好，“我给你们三个时辰的时间，我要关于步留痕的所有消息。”
<br/>    
<br/>        刀疤脸尚有余悸地看了陈明一眼，实在是被陈明吓怕了，陈明此人，看起来不过是悟道境，然而他展开威压，却是让他们觉得如同仙人一般强大。
<br/>    
<br/>        而且，对方既然隐藏修为，必定是什么见不得光的仙，加上刚刚给他们喂得毒丹，更是害怕不已。
<br/>    
<br/>        “大人在此稍等，我们马上给大人将步留痕的所有消息弄到。”
<br/>    
<br/>        不过半个时辰的时间过去，刀疤脸已经带着三个劫匪回来了，“大人，我们从红袖招酒馆得到了消息，步留痕隐藏了修为，他实际上已经是天仙修为，如今打伤了天庭的七位天仙，已经带着女儿朝着瑶池掌控的星河逃去！”
<br/>    
<br/>        陈明道：“你们能够保证消息是准确的？”
<br/>    
<br/>        刀疤脸道：“这红袖招乃是瑶池秘密做东，瑶池和天庭因为这三条星河的掌控权一直不对付，绝对是真的！”
<br/>    
<br/>        陈明微微摇头，“还不够，我还要更详细的消息！”
<br/>    
<br/>        想要控制住一个人，必须先让他绝望。
<br/>    
<br/>        绝望的人，只需要一根稻草，便足以收服。
<br/>    
<br/>        而现在，步留痕明显还没有走到绝望的这一步。
<br/>    
<br/>        陈明没有兴趣去推动步留痕走到绝望中，他现在只是想要旁观。
<br/>    
<br/>        不过，为了保证消息的实时性，不至于步留痕人都没了陈明才得到消息，陈明还是决定跟着步留痕的路过去。
<br/>    
<br/>        赤龙星河府。
<br/>    
<br/>        金色的殿堂之中，在正中央有着一张金色的桌案，桌案之后，一位金甲将军看向大殿之下站着的众多天庭仙官，“一个步留痕而已，你们难道还抓不住他吗？”
<br/>    
<br/>        其中一位仙官拱手道
<br/>    
<br/>        ：“瑶池，瑶池似乎是插手了！”
<br/>    
<br/>        金甲将军呵呵一笑，道：“瑶池还真是会恶心人啊，天庭的叛徒，他们都有兴趣插一手吗？”
<br/>    
<br/>        一位仙官道：“大人，请再给我们一些时间。”
<br/>    
<br/>        金甲将军道：“记住这里是哪儿，这里是赤龙星河，我们天庭掌控的赤龙星河，若是在赤龙星河中，步留痕都逃走了，那斩仙台，你们也可以去上面试试刀还利不利了。”
<br/>    
<br/>        众多仙官齐齐拱手道：“我等，这就前去。”
<br/>    
<br/>        步留痕已经逃了一个月了，天庭的眼线遍布整条星河，若不是他有着天仙的修为，恐怕早就已经落入了天庭的手中。
<br/>    
<br/>        步留痕实际上是想要逃入天庭掌控的赤龙星河和瑶池掌控的玉龙星河之间的孽龙星河。
<br/>    
<br/>        这一条星河，混乱之极，天庭和瑶池在其中争斗不休，各种势力盘根错节，最是好隐藏其中。
<br/>    
<br/>        只要逃到了孽龙星河，那么天庭就拿他暂时没有什么办法。
<br/>    
<br/>        不过可惜的是，整条赤龙星河都已经被封锁住，他也只能强行杀出一条路来。
<br/>    
<br/>        而现在困在他面前最大的难关就是，镇守着通往孽龙星河入口的守星大将，乃是一名天仙中期的天庭仙官，加上这一位仙官可以借助星辰的本源力量。
<br/>    
<br/>        他想要通过这最后一颗星辰，简直难上加难。
<br/>    
<br/>        他最为擅长的仙术名叫八步登龙，速度奇快无比，而他，便是靠着这仙术逃了这么久，若是能够再修成一层八步登龙，那么他就可以破空而行，不必借助仙路也可以行走在星空中！
<br/>    
<br/>        可惜的是，他修行了上百年，却是还困在最后一道坎上，迟迟无法突破。
<br/>    
<br/>        步留痕是一个身形消瘦，却颇为高大的中年男人，两眼显得有些外突，炯炯有神，带着一些食物回到房间中，在房间中，楚楚可怜的步生莲看向他，“父亲，我们真的已经无路可走了吗？”
<br/>    
<br/>        步留痕看向生的沉鱼落雁的步生莲，道：“不，我们还有机会，瑶池一直都是天庭的对头，若是瑶池插手，我们便有机会逃走！”
