        这枚钻戒，价值不菲。
<br/>
<br/>        她一身着装包括手提包，都是国际限量版。
<br/>
<br/>        而她跟他离婚的时候，她是净身出户，她也没有背着他藏私房钱。
<br/>
<br/>        就算她接私单，也不可能在这么短的时间内，赚到这么多钱去买这一身奢侈品。
<br/>
<br/>        所以，她的的确确没有骗他。
<br/>
<br/>        只是，他仍旧不相信她已另嫁他人的事实。
<br/>
<br/>        “筱颍，你是不是被哪个男人包养呢？”傅昼景哑着声音质问道。
<br/>
<br/>        他居然这么想她，难道，在他的眼里，她就只能是被男人包养的那种类型吗？！
<br/>
<br/>        童筱颖瞪向傅昼景，欲言又止，顿了顿后，她才冷笑：“对！我是被男人包养了！你不是跟我离婚了吗？所以，我又回到了‘夜惑’，卖了我的初夜！跟你在一起这么久，我很荣幸，你不愿意碰我，才让我如今能卖个好价钱。”
<br/>
<br/>        “你就这么作贱你自己吗？”傅昼景歇斯底里地怒吼了起来。
<br/>
<br/>        童筱颖趁机甩开了傅昼景的手，看着傅昼景渐渐惨白的俊脸，吃力地翘起嘴角，不知羞耻地说道：“你忘了吗？我在‘夜惑’里卖过唱啊！自然认识一些有钱有势的公子哥们。现在，你知道我有多恶心了吧？”
<br/>
<br/>        她知道，他有情感洁癖。
<br/>
<br/>        她这样作贱自己，是让他放弃自己最快的办法。
<br/>
<br/>        但是，她低估了他对她的爱。
<br/>
<br/>        傅昼景的身影在她面前越放越大，将她完全笼罩在了他的阴影之中。
<br/>
<br/>        童筱颖缓缓抬眸，对上了他那双深如黑潭般的眼睛，讷讷地张嘴：“我……”该走了！
<br/>
<br/>        “筱颍，我爱你，不要用这种方式赶我走！”他先声夺人，打断了她的话。
<br/>
<br/>        她直勾勾地看着他，一时怔愣着，不知道自己接下来该怎么做是好。
<br/>
<br/>        “筱颍，我爱你。”他的大手，温柔地拉住她的手。
<br/>
<br/>        掌心的温度透过肌肤相触而传了过来，这份炙热让她下意识地想甩手，他却紧紧地抓住她的手，不让她挣脱。
<br/>
<br/>        “筱颍，我爱你。”傅昼景一遍又一遍地向她告白，另一只手揽住了她的腰肢，带动着她的身子往自己身上贴紧。
<br/>
<br/>        他一边说着，一边深情款款地凝视着她。
<br/>
<br/>        明明是爱的告白，但慢慢地……这表白的意境就变味了。
<br/>
<br/>        童筱颖的双手用力一推，将自己跟面前的男人推出了一道陌生的距离。
<br/>
<br/>        他的告白，她不再唯心所动。
<br/>
<br/>        傅昼景脚尖落地，稳住身子，难以置信地看着童筱颖，脸色越来越惨白。
<br/>
<br/>        “记得吃药，好好保重自己的身体，你还有尚轩，好好活下去。从今以后，你不要再联系我了，我也不会主动联系你。我与你，回不去了，已经结束了，不可能了，不要再痴心妄想了。”童筱颖放下了手，碎碎念叨着，黯然神伤地转过身去。
<br/>
<br/>        “筱颍，不要走……”傅昼景疾步上前，双手一探，从她腰间环过，十指相扣在她腰腹上，将她再次搂入了怀中。
<br/>
<br/>        当这个怀抱贴上背脊，腰腹间被紧紧搂住之际，童筱颖身子一僵。
<br/>
<br/>        “筱颍，不要走……”他有气无力地说。
<br/>
<br/>        童筱颖柳眉微蹙，分开腰腹间紧扣的十指，就在她勉力转身之际，傅昼景高大的身躯突然向她压了下来。
<br/>
<br/>        “小卫？”童筱颖脸色一变，心跟着揪了起来。
<br/>
<br/>        傅昼景整个身子软瘫在了她的身上，脸色惨白得十分吓人。
<br/>
<br/>        童筱颖拉起傅昼景的手臂，架在自己的肩头上，吃力地扶着他进了屋，将他安置在了木床上，然后又跑了出去将地上的那几瓶药捡了起来。
<br/>
<br/>        她将药片倒在了手心里，又去倒了水，再次回到了傅昼景的身边。
<br/>
<br/>        “小卫，快把药吃了！”童筱颖将手中的杯子，放到床边的凳子上后，又将傅昼景的身子扶了起来，半倚在自己的怀中，“小卫，把药吃了，好不好？”
<br/>
<br/>        傅昼景却微微摇了摇头，气若游丝地说道：“筱颍，和你在澳洲生活的那三年，是我过得最幸福的时光。周一到周五，我工作，你念大学。到了每个双休日，我们在一起快乐的度过，一起坐在沙发上看电影，一起研究美食，一起讨论建筑的设计。你偶尔会让我感到无趣，我们也会吵架，但很快又会和好。离开你之后，思妍动不动就拿性命威胁我，我身心都很疲惫，直到我从她那儿逃出来，回到了这里，我才知道，其实你一直在等我，等我来找你，但是，我来晚了，你却已经走了。筱颍，有你在身边的日子，才是我最大的幸福。”
