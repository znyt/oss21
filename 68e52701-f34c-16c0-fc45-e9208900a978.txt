　　一头巨大妖兽，猛地朝他袭来。
<br/>　　萧逸仗剑而出。
<br/>　　这时，他手中之剑，正是那把‘斩星’。
<br/>　　萧逸刚要将面前妖兽斩于剑下，脑海却陡然生疼。
<br/>　　“嘶。”萧逸痛呼一声。
<br/>　　眼前画面，忽然变得无比‘狭隘’。
<br/>　　所能见到之画面，也仅仅是他与面前这头妖兽。
<br/>　　萧逸明白，这是因为他所经历的，是‘斩星前辈’的战斗。
<br/>　　至于画面之内其他人的战斗，其他东西，与他无关。
<br/>　　锵…
<br/>　　巨大妖兽噬来，萧逸手中‘斩星’挡于身前。
<br/>　　巨兽，身躯庞大而坚固。
<br/>　　斩星竟是未能损它分毫。
<br/>　　“好强的妖兽。”萧逸先是一惊，随后脚步连退。
<br/>　　“万花斩星。”
<br/>　　萧逸暴喝一声。
<br/>　　十朵剑花，轰然出现于空气中。
<br/>　　萧逸一剑劈出。
<br/>　　剑花瞬间绽放。
<br/>　　剑落，妖兽分尸而死。
<br/>　　……
<br/>　　不知过去了多久，萧逸再次睁开了眼睛。
<br/>　　看了眼两旁的剑气之路。
<br/>　　他已经走出很远的距离了。
<br/>　　回头看去，早已看不到一开始的剑碑。
<br/>　　前方，怕是要不了多久就会到达第一层空间的尽头。
<br/>　　他已然明白这次参悟的所在。
<br/>　　所谓参悟，便是让自己带入这位前辈的身份中。
<br/>　　自己的一身手段、武技等等，丝毫在参悟状态中用不得。
<br/>　　所能用的，只是这位前辈生前的手段，那个画面阶段的手段。
<br/>　　其考验意义，便是当给你同样的机缘，同样的功法武技，同样的手段下，你能否做得更好。
<br/>　　比如在战斗画面中。
<br/>　　自己代入这位前辈的身份，拥有着同样的剑技功法。
<br/>　　自己能否更快地结束这场战斗？
<br/>　　又比如在修炼画面、参悟武道画面之中。
<br/>　　自己在同样的情况下，参悟同样的武道，能否比这位前辈更快地感悟成功，掌控成功？
<br/>　　这就是参悟的考验。
<br/>　　同样的剑道之路，自己去走一遭，能否比斩星前辈更快更好。
<br/>　　若是在这些战斗画面中，自己败了。
<br/>　　或者在这些感悟画面中，自己参悟失败了，用的时间更久了。
<br/>　　那么，这此参悟，便会失败。
<br/>　　剑心，亦会在那一瞬间失守。
<br/>　　这也代表着，参悟者，会瞬间迷失其中，再无法醒来。
<br/>　　参悟之路，便就此止步，只能等半月之后，被强行轰出这片空间。
<br/>　　当然了，虽说在剑帝碑空间之内只待了半月。
<br/>　　但，在参悟状态中的迷失，可是动辄千年万年。
<br/>　　这等迷失，可是很可怕的事，亦是极难度过的岁月。
<br/>　　“呼。”萧逸再次深呼吸一口气。
<br/>　　眼睛，开始闭上。
<br/>　　脑海一白，再次进入了参悟状态。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　参悟状态中，已过数千年。
<br/>　　这时，萧逸面前的画面，忽然变得莫名。
<br/>　　越是到后面的画面，越是奇怪莫名。
<br/>　　萧逸甚至觉得，这些画面有些‘浑浑噩噩’、‘模糊’。
<br/>　　当然，萧逸看得很真切。
<br/>　　这里，似乎是一片虚空之地。
<br/>　　周遭漆黑一片，唯面前一团巨大火焰，横亘虚空。
<br/>　　“星幻之火？”萧逸瞬间反应了过来。
<br/>　　随后，开始盘膝感悟起来。
<br/>　　不知过去了多久。
<br/>　　萧逸明显已经参悟了其中武道。
<br/>　　他试图去掌控这片火焰。
<br/>　　但他发现，他根本什么都做不了。
<br/>　　原因，只有一个，那便是在斩星前辈的经历中，斩星前辈从未想过要掌控乃至得到这团星幻之火。
<br/>　　斩星前辈，在经历中未曾做过的事；萧逸在参悟状态中，亦做不到。
<br/>　　斩星前辈的经历中，想做，却做不到的事，萧逸可以去经历，可以去尝试能否做到。
<br/>　　但斩星前辈想都没想过的事，萧逸根本经历不了。
<br/>　　当时斩星前辈的心境，应该是知晓自己绝对掌控不了这团火焰，故没有打算尝试去掌控。
<br/>　　画面，到这里结束。
<br/>　　萧逸从参悟中醒来。
<br/>　　这一段的参悟经历，萧逸过得特别快。
<br/>　　因为，在参悟状态中，感悟那团星幻之火时，萧逸的速度，远比当年的斩星前辈要快得多。
<br/>　　萧逸自己本身就掌控多种世间强悍火焰，又是个控火天才。
<br/>　　自然的，把他代入到斩星前辈的身份中，参悟这团星幻之火异常快。
<br/>　　他的参悟速度越快，画面过得越快，现实中的时间流逝就会越慢。
<br/>　　想来，他刚才并未真正‘渡过’多少时间。
<br/>　　另外一点。
<br/>　　他终于明白斩星前辈的那句话，‘若你有本事，自己会发现的’。
<br/>　　确实，刚才这个画面中，这个阶段的斩星前辈的实力层次，只要萧逸日后达到这个层次；那他萧逸日后肯定也会到达这片画面中的地方。
<br/>　　他已经大致将星幻之火存在的画面记下来。
<br/>　　日后，若是他能到达那片不知名的奇怪地域，一眼便能知晓。
<br/>　　……
<br/>　　萧逸再度闭眼，再度参悟。
<br/>　　参悟状态中，再度过去了千年。
<br/>　　画面，忽然定格。
<br/>　　面前，一头奇怪的妖兽，忽然出现。
<br/>　　妖兽的奇特，萧逸无法形容。
<br/>　　但，那狰狞的面容，阴蛰到极点的气息，萧逸却蓦地感到有些熟悉。
<br/>　　“无尽黑海的妖兽？”萧逸陡然想了起来。
<br/>　　对，面前这头妖兽，样貌似极了当年他在东域遇到的黑海妖兽。
<br/>　　那个无尽黑海，毒气四溢，一望无际的剧毒之海。
<br/>　　不过，面前这头妖兽的气息，远比他当年遇到过的妖兽强得多。
<br/>　　早在之前画面中遇到‘星幻之火’的那个阶段时，斩星前辈的实力就远远超过了绝世强者。
<br/>　　那么，现今画面阶段遇到的妖兽，自然强得多。
<br/>　　画面周遭，同样是一个辽阔的战场。
<br/>　　战场之内，杀戮无数。
<br/>　　但很快的，这些画面，又再度变得‘狭隘’。
<br/>　　萧逸能看到的，也就只有面前这头妖兽。
<br/>　　“上古大战？”萧逸皱了皱眉。
<br/>　　他记得，当年上古时期结束的标志，便是人类强者与妖兽皇者的大战告终而落幕。
<br/>　　也是那场大战，打得天地失色，大陆四分五裂。
<br/>　　东域直接被分离出去，甚至于天地力量破碎。
<br/>　　东域那边，直接连一条完整武道都无法凝聚，只剩残存的一丝丝天地武道力量。
<br/>　　难道说，斩星前辈这个阶段的画面，所进行的战斗，便是当年那场大战？
<br/>　　容不得萧逸多想。
<br/>　　面前狰狞妖兽，已然袭来。
<br/>　　“万花星火。”萧逸暴喝一声。
<br/>　　一剑劈出，剑气纵横，剑气之上火焰汹涌。
<br/>　　那是斩星剑道与星幻之火一道的融合。
<br/>　　一剑出，剑气凌厉，火焰，直接撕裂了大片空间。
