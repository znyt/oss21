周萍不禁有些唏嘘感叹，有时候人的命数就是这样，原本夏妍妍可以站在海城最显赫的权贵陆寒霆身边，成为人人艳羡的陆太太。
<br/>
<br/>        但是一场替嫁，改变了所有人的命运。
<br/>
<br/>        周萍也不知道这是意外，还是冥冥之中的命中注定。
<br/>
<br/>        “夏妍妍，你也回去好好工作吧，这件事到此为止！”周萍转身进了办公室。
<br/>
<br/>        ……
<br/>
<br/>        夏妍妍不知道当时周萍为什么会用那种目光看着她，好像有点同情，有点遗憾…她只知道研究院的论坛迅速恢复了平静，一夕之间所有的帖子都被删，关键词被锁，有一只神秘的大手将这一切都压了下去。
<br/>
<br/>        能有这么大本事的人，除了陆寒霆不会有别人了。
<br/>
<br/>        周萍也雷厉风行的肃清了研究院所有的流言蜚语，大家都回归了工作岗位，不敢再提夏夕绾出轨的事情。
<br/>
<br/>        夏妍妍感觉闹了这一场，都是自己自导自演，她连夏夕绾的头发都没有伤到。
<br/>
<br/>        不过这更加加重了她对夏夕绾的仇恨，她将所有的希望都放在了幽兰苑那个鬼夫身上。
<br/>
<br/>        夏妍妍给李玉兰打电话，“妈，我总觉得怪怪的，好像夏夕绾和陆寒霆之间有什么事情是我不知道的。”
<br/>
<br/>        李玉兰上一次差一点就把命给丢了，修养了很长时间才恢复过来，她身体好了，自然又到了作妖的季节了，“妍妍，你不要想太多，这个夏夕绾已经替嫁进幽兰苑冲喜了，就凭这一点她就无法跟陆寒霆在一起，妍妍，陆太太的位置是你的，你一定可以嫁给陆寒霆的！”
<br/>
<br/>        想到夏夕绾已经嫁人的事情夏妍妍一颗心总算是安定了下来，“妈，你有没有催爸去幽兰苑看看那个鬼夫女婿？”
<br/>
<br/>        “妍妍，你放心，我现在和你爸就在去幽兰苑的路上呢，今天一定会见到那个鬼夫的。”
<br/>
<br/>        夏妍妍勾唇，“妈，那就交给你了，等你好消息。”
<br/>
<br/>        “好。”
<br/>
<br/>        李玉兰将电话给挂断了，她现在在车上，夏振国在开车，经历了上一次的丑闻，夏振国虽然没有离婚，但是已经将她打入冷宫了，不跟她交流，也没有再踏入过她的房间。
<br/>
<br/>        李玉兰知道自己已经没有靠山了，当务之急还是要收拢夏振国的心，让自己站稳脚跟。
<br/>
<br/>        “振国，以前的事情都是我的错，都是我咎由自取，我已经诚心忏悔了，不过我的心始终在你和这个家的身上，所以希望你能再给我一次机会。”
<br/>
<br/>        夏振国这段时间过得如鱼得水，但是他没有忘记李玉兰带给自己的奇耻大辱，所以他冷哼了一声，懒得搭理李玉兰。
<br/>
<br/>        他的反应在李玉兰的预料里，三言两语是哄不回他的心的，不过，她最擅长的就是收拢男人的心，所以她有的是时间。
<br/>
<br/>        现在还是以妍妍为主，妍妍是她最大的依仗了。
<br/>
<br/>        只要妍妍成功嫁给陆寒霆，嫁入陆氏当少奶奶，那她风光的日子才是开始。
<br/>
<br/>        现在李玉兰还沾沾自喜，觉得自己很有先见之明，幸亏她让夏夕绾替嫁给了幽兰苑那个鬼夫，要不然现在麻烦就大了。
<br/>
<br/>        这时幽兰苑到了，车停了下来，夏振国和李玉兰下车。
<br/>
<br/>        这是他们第一次来幽兰苑，幽兰苑有雕花大门拦着，进不去，夏振国按了门上的门铃，很快就有女佣出来了。
