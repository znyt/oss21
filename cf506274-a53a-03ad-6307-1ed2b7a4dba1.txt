　　“噬...火...百...刃...”
<br/>　　当慕容家执事全力一拳轰向萧逸时，耳中蓦然想起一道深沉的声音。
<br/>　　这道沉重的声音，听在他耳中，仿佛重若千钧。
<br/>　　沉重的力量，仅仅是因为这四个字所代表的含义。
<br/>　　而那含义，称之为‘玄阶武技’。
<br/>　　噬火百刃，萧家最出名的武技，紫云城无人不晓。
<br/>　　当他听到噬火百刃四字时，心中已经想到了‘玄阶武技’。
<br/>　　而玄阶武技，对于一个后天武者来说，只能用恐惧去形容。
<br/>　　他的脸色，瞬间从狞笑和不屑变为恐慌。
<br/>　　然而，这恐慌，又瞬间定格，因为，他已经死了。
<br/>　　“轰”的一声。
<br/>　　噬火百刃忽然从萧逸手中打出，重重劈在慕容家执事身上。
<br/>　　那散发着炙热高温的火刃，先是将他的身体劈出一道巨大的伤痕，随后轰然爆炸，将他炸得血肉模糊。
<br/>　　当他跌落地面时，已经成为一具毫无生机的尸体。
<br/>　　“慕容显执事，他...他竟然被秒杀了...”另一个后天三重的执事脸色大变。
<br/>　　慕容华更是脸色阴沉，“萧逸他，竟然练成了萧家最强的武技，噬火百刃，情报错误，看来我们要小心了。”
<br/>　　萧逸看了眼地下的尸体，眼中暗暗点头，“玄阶武技，果然强大。”
<br/>　　噬火百刃，虽然只是玄阶低级。但，这可是先天武者都还在使用，且没能大成的武技。
<br/>　　其难度如此高，威力自然不必多说。
<br/>　　虽然只有一道火刃，但慕容显本身没有防备，又在这么近的距离里被打中，自然直接被秒杀了。
<br/>　　当然，最重要的原因，还是因为萧逸本身的修为，他是后天一重，与慕容显不过差了两重。
<br/>　　而他由于体内的冰火气丹调动真气的速度远超其它后天境武者的气丹，则变相地让他的战力不亚于后天二重。
<br/>　　也就是说，他和慕容显的真正差距，只有一重。
<br/>　　而这区区一重修为差距，远远弥补不了噬火百刃的强大。
<br/>　　“该死，一定要杀了萧逸，为慕容显执事报仇。”剩下的那个后天三重执事愤怒地说道。
<br/>　　“不能大意了，使用武技，我们联手上，尽快斩杀萧逸。”慕容华吩咐道。
<br/>　　两人话语刚落，脚步一动，联手朝萧逸攻来。
<br/>　　“联手吗？哼。”萧逸怡然不惧，一个跳跃，从马上跳了下来。
<br/>　　“豹形。”萧逸心中喝了一声，快若疾风，瞬间消失在原地。
<br/>　　慕容华一惊，“区区后天一重，其速度竟不亚于我？”
<br/>　　萧逸忽然出现在另一个后天三重的执事背后，一拳打出，双爪成虎形。
<br/>　　“小心。”慕容华提醒一声，然而，迟了。
<br/>　　另一执事被萧逸重重地轰了一拳，吐血被击飞。
<br/>　　“好猛的力度，这小废物怎么回事。”那执事满脸不可思议。
<br/>　　萧逸虽只是后天一重，但在冰火气丹的增幅下，他调动灵气的速度和猛度丝毫不亚于后天二重。再加上形意五绝，他的战力达到了后天三重之上，仅次于后天四重。
<br/>　　嗖、嗖、嗖、嗖，萧逸不断移转腾挪，避开慕容华，专攻另一个执事。
<br/>　　噗、噗、噗、、那执事猛地吐出一口又一口的鲜血。他只觉，萧逸每次打向他的拳头，都势大力沉，让他体内一阵气血翻滚。
<br/>　　“萧逸，有本事和我大战一场，这般阴险，算什么英雄。”慕容华见另一执事不断受伤，心中大怒。
<br/>　　“阴险？”萧逸冷笑道，“你们三人都是成名已久的后天境武者，联手埋伏，欲要杀我这个少年。论阴险，我和你们差远了吧。”
<br/>　　话虽这样说着，但萧逸还是放弃了对另一执事的攻击，转而攻向慕容华。
<br/>　　“哼。”慕容华冷哼一声，心头一喜，暗道，“小废物，果然是少年心志，稍微被我言语一激便攻向我了。”
<br/>　　“啪、啪、、”
<br/>　　两人对轰了一掌。
<br/>　　慕容华岿然不动，萧逸则被震退了数米。
<br/>　　“萧逸，你不是我的对手，你死定了。”慕容华见自己一掌震退萧逸，顿时自信十足。
<br/>　　“呵。”萧逸站稳身子后，淡淡一笑，“算了，大概知道我现在是什么层次的实力了，也该结束战斗了。”
<br/>　　“什么？”慕容华顿时一惊，“你...你竟然敢拿我们来给你试手？”
<br/>　　“现在才发现吗？”萧逸冷笑一声。
<br/>　　“狂妄。”慕容华怒喝一声，“别以为你炼就了噬火百刃就了不起，区区一道火刃，远远比不上我小成的碧波水吟箭。”
<br/>　　慕容华大手一挥，数十根蓝色水箭从天而降，密密麻麻地朝萧逸飞去。
<br/>　　“噬火百刃。”萧逸低喝一声，手中凝聚出一道火刃，脱手而出。
<br/>　　一道火刃，对上数十根水箭。
<br/>　　炙热的红芒，与冰冷的蓝光碰撞在一起。
<br/>　　霎那间，冰火消融。最终，火刃消失，数十根水箭也消散无踪。
<br/>　　虽然萧逸的噬火百刃才刚刚练成，只有一道；而慕容华的碧波水吟箭练了数十年，已至小成，距离大成不远矣。
<br/>　　但，玄阶武技和黄阶武技的差距太大了。
<br/>　　炎龙大陆，武技无数，但却仅仅被分为天地玄黄四个阶级。可想而知，每一个阶级间的差距，必然是极为巨大的。
<br/>　　玄阶武技，哪怕是低级，也比黄阶巅峰武技强得多，彼此间的差距可容天差地别来形容。
<br/>　　这也是为什么三大家族各自拥有的唯一一本玄阶低级武技是他们最大的底蕴，从不外传。
<br/>　　只有家族长老和个别精英执事才能学习。
<br/>　　而黄阶巅峰武技，却是一般的族人都能学习。
<br/>　　两者根本不是一个层次的。
<br/>　　若萧逸如今的噬火百刃能有小成，可以打出数十道火刃的话，瞬间就能秒杀这后天四重的慕容华。
<br/>　　“这小子很强，我们联手。”慕容华脸色凝重地对另一执事说道。
<br/>　　“嗯。”另一执事沉重地点了点头。
<br/>　　“碧波水吟箭。”两人同时发动了武技攻击。
<br/>　　萧逸怡然不惧，他已经试出如今自己的战力，准备结束战斗了。
<br/>　　萧逸心中低喝一声，“升龙。”
<br/>　　他的气势瞬间大涨，境界从后天一重达到后天三重。
<br/>　　慕容华两人顿时脸色一变，“怎么可能，这小子的境界凭空增加了。不对劲，他是后天三重了，我们不是对手，快跑。”
<br/>　　萧逸后天一重的境界都能和他们打成平手，如今后天三重，他们连招架的可能都做不到。
<br/>　　两人心中瞬间想到了这一点，立刻收回了武技，转身便逃。
<br/>　　“想跑？”萧逸眼中迸发出骇人的杀意，“慕容家的，你们一直想置我于死地，这次便先杀你们三个执事，收点利息。”
<br/>　　话音刚落，噬火百刃脱手而出，以极快的速度打向慕容华两人。
<br/>　　嗖的一声，火刃直接从逃跑的两人被后劈过。一声爆炸巨响，两人被炸得血肉模糊，跌落地面后，生机全无。
<br/>　　萧逸在升龙的增幅下，达到后天三重的境界，加上冰火气丹的增幅，所打出的噬火百刃威力凭空增加了一大截，瞬间秒杀了这两人。
<br/>　　其实，如果这次的战斗，慕容华是那些精英执事，修炼了慕容家的玄阶低级武技的话，情况一定会不同。
<br/>　　就算萧逸能杀他们，必然也会很困难，而不是像现在那样轻松。
<br/>　　萧逸在三人的身体上摸索了一番，搜出了三个乾坤袋。
<br/>　　对于武者来说，一般身家都会随身带着。毕竟，自己的财富，没有比放在自己身上更安全的地方了。
<br/>　　打开三人的乾坤袋，里面的东西大致都差不多，都是些材料、几颗后天丹，还有一千两左右的银两。
<br/>　　“慕容家的执事，身家倒还算丰厚。”萧逸笑笑，毫不犹豫地将三人的‘贡献’笑纳了。
<br/>　　随后，萧逸的眼睛一变，太阴太阳之眼出现。
<br/>　　两个无形的漩涡凭空而现。
<br/>　　漩涡出现在三人的尸体身上，将三人的武魂力量全部吸收。
<br/>　　值得一提的是，两个后天三重的执事，武魂只是橙阶；反倒是慕容华，竟然是黄阶武魂。
<br/>　　难怪之前另外两个执事言语中对慕容华似乎颇为恭敬。
<br/>　　三个武魂的力量，被注入到控火兽和冰鸾剑武魂中。
<br/>　　而后，萧逸将那数十山匪的武魂也一并吸收。虽然那些只是最低等的赤色武魂，但架不住量多。
<br/>　　当所有武魂被吸收，控火兽武魂堪堪进阶为黄色武魂；至于冰鸾剑武魂，则没有任何变化。
<br/>　　萧逸重新骑上了骏马，目光冰冷，“我才刚出紫云城，立刻就遭到慕容家的埋伏了。五长老，这其中你功不可没吧，这笔账，等我回到萧家再慢慢跟你算。”
<br/>　　想罢，萧逸一甩马鞭，绝尘而去。
<br/>(本章完)
