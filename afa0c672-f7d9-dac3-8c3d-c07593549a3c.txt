当时他就站在二楼的雕花栏杆处看着，看着跪在雪地里的她。
<br/>
<br/>        她不认识他。
<br/>
<br/>        他却很早就认识她了。
<br/>
<br/>        所有的记忆都倒退回了很多年前，那时的陆司爵二十岁，柳璎珞十二岁。
<br/>
<br/>        他第一次见她，是在医院里。
<br/>
<br/>        她一定不知道，她十二岁的时候，他就见过她。
<br/>
<br/>        那天她妈妈在医院里，需要动手术，要很大一笔钱。
<br/>
<br/>        柳招娣的妈妈来了，将一叠钱用力的砸在她稚弱的身上，骂她是贱人生的小贱人。
<br/>
<br/>        那些钱散落了一地，她低腰去捡。
<br/>
<br/>        回廊里很多围观者，对着她指指点点，她却面无表情，清冷淡漠的将地上散落的钱一张又一张的捡了起来。
<br/>
<br/>        在人群里，她很特别。
<br/>
<br/>        他多看了两眼，那天她穿了一件白裙子，低腰捡钱的时候乌发的秀发披散下来，他看到了她露出的半张侧颜，尚且稚嫩却已堪称绝色。
<br/>
<br/>        那时的陆司爵已经20岁了，20岁的男孩子，长大了，他的朋友都开始谈恋爱，暗地里还偷偷摸摸的将片儿给他分享了，但是，他毫无兴趣。
<br/>
<br/>        直到遇到她，这个叫做柳璎珞的女孩儿。
<br/>
<br/>        他的眼睛流连在她半个侧脸，她还小，但是稚嫩的五官已经清丽绝色，纵然见惯了美人的他，双眸里也闪过惊艳。
<br/>
<br/>        当时一张钱掉在了他的脚边，被他踩到了，她过来，是半蹲着捡钱的姿势，她将小脸抬起来，一双清冷的翦水秋瞳望着他，“先生，你踩到我的钱了。”
<br/>
<br/>        这是她跟他说的第一句话。
<br/>
<br/>        他将脚移开，她捡了钱，然后就走了，当时她就跟他擦身而过，但是她的目光没有再他身上多停留一秒。
<br/>
<br/>        陆家的君主陆司爵，站在哪里都是最瞩目的焦点，那些女孩子看着他就两眼冒着粉色泡泡，但是她抓紧了手里的钱，头也没抬的就走了。
<br/>
<br/>        第一次见面，她看都没有多看他一眼，她眼里都是手上那叠少到可怜的钱。
<br/>
<br/>        但是，她仿佛在他心里落了根，他回去就让管家去打听，管家说她叫柳璎珞。
<br/>
<br/>        柳璎珞。
<br/>
<br/>        并不是柳招娣。
<br/>
<br/>        陆司爵那时就知道柳家有两个女儿，进行了一场换位人生。
<br/>
<br/>        柳璎珞的生母是一个卖艺不卖身的歌女，被柳父醉酒后强占的，当时怀了孕，生下了柳璎珞。
<br/>
<br/>        柳璎珞的生母有心脏病，常年住院，为了母亲，她进了柳家，成为了柳招娣的影子和血灌子，柳家给她钱，给她母亲治病。
<br/>
<br/>        柳璎珞。
<br/>
<br/>        好美的名字。
<br/>
<br/>        像她的人。
<br/>
<br/>        后来，他26岁，已经一手缔造了陆氏帝国神话，常年在洛杉矶，有一天，那个跟他分享片儿的哥们找他玩，两个人在酒吧的豪华包厢里
<br/>
<br/>        “司爵，你都好些年没有回国了，我跟你说，这些年帝都城可热闹了，帝都出现了一个绝色的第一美人，这位第一美人还是一个天才少女，刚创立Fly珠宝。”
<br/>
<br/>        “这个天才少女就是柳家的女儿…柳招娣。”
<br/>
<br/>        柳招娣？
<br/>
<br/>        时隔六年，他再度听到她的名字。
<br/>
<br/>        他知道柳招娣就是柳璎珞。
<br/>
<br/>        他比任何人都更早的知道了她的存在。
