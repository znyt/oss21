　　第81章
<br/>　　桃园结义：年少的英雄在桃园相聚，为共同的理念蹈生赴死，桃花终会落去，豪杰的名字永远流传。
<br/>　　程大雷看着这行系统提示，陷入久久的沉默中。隐藏属性是很稀有的，直到如今，程大雷碰上的也不过寥寥几人。而这种三人凑在一起，才能激发隐藏属性的事情，更是从未遇到过。
<br/>　　看着眼前三人，程大雷脑海中不由浮现那三个流传千古的名字：大爷姓刘命备字玄德，家住大树楼桑，二爷姓关名羽字云长，家住山西解良县，三爷姓张名飞字翼德，家住涿州……
<br/>　　这样的名字，这样的隐藏属性，眼前这三位，该不会和那传说中的三位有什么关系吧？
<br/>　　“你们……是哪里人？”程大雷试探着问。
<br/>　　“范阳郡。”刘悲答。
<br/>　　这就对得上了，程大雷深吸一口气，再问：“那么一位姓张名飞字翼德的，你们认得么？”
<br/>　　三人面面相觑，一头雾水。
<br/>　　程大雷有些困惑，也有些失望，又问：“你们为何要在蛤蟆寨落草？”
<br/>　　“村子里恶霸做恶，被我们兄弟杀了，官府如今正通缉我们。因听到程当家的名声，所以特来投奔。”关鱼答。
<br/>　　程大雷看了关鱼一眼，心道，我就知道是你小子杀了人。传说中，关云长就是杀了恶霸地主，后来才改行卖绿豆的。
<br/>　　“你们三个谁杀的人呐？”程大雷明知故问。
<br/>　　结果没料到，张肥和关鱼的目光都落在刘悲身上。刘悲才十七岁，可一副苦大仇深的表情，气质好像七十一岁。
<br/>　　“唉，不值得呐。”
<br/>　　还真的是蔫人出豹子，以后真不能以貌取人。
<br/>　　“大当家，我忽然想起来，你说的张翼德，好像是我的祖祖祖……爷爷……”张肥道。
<br/>　　程大雷一听，瞬间来了精神，接着又盘问许多，才总算知道三人是怎么会事。
<br/>　　这个世界的历史没有三国，自然也没有刘关张，汉末之后不是三国鼎立，而是十九国混战，直到大武一统，立国一百二十年直到今日。
<br/>　　但刘关张的名字没有在历史中抛头露面，却并不代表他们没有存在过。眼前这三人，就是刘关张的后人。
<br/>　　据刘悲所说，他的祖先是个很有理想的人，与关张二人的祖先桃园结义，打下三柄兵器：鸳鸯剑、青龙偃月刀、丈八蛇矛。立志要扫平天下叛逆，匡扶汉室。
<br/>　　但事情不是很顺利，攻打黄巾贼时就吃了瘪，未立寸功，后十九国混战，投奔谁谁玩完，最后刘大耳在乱世中闯荡了大半辈子，却是一无所获，六十岁那年和关张回到范阳郡定居。
<br/>　　刘大耳是卖草鞋的，关红脸是卖绿豆的，张胡子是屠户，三人家里，张飞相对富裕些。当年少年踌躇满志，梦想着仗剑闯天下，归来时已是两鬓见白，沧桑满眼。每天张飞打开门卖猪肉，关羽推着小车在他门前卖绿豆，过半晌，担着担子卖草鞋的刘备就来了。
<br/>　　天黑之后，三人便聚在一起喝酒，日复一日。关张是看得开的，这刘备却是未必。每日酒醉，总要吵吵些『吾乃大汉皇叔』『倘若天道助孤，世上又有几人配称王』之类的话。
<br/>　　村里的孩童常常以此取笑他，到后来便常常能见到一个白发苍髯的老头呼喝着什么。
<br/>　　活到七十二岁，善终。
<br/>　　关张二人也相继去世，三人的后人便一直在范阳郡定居，后代子孙世世在桃园结拜，刘家的子孙永远为大哥，倒不是以年龄论的。
<br/>　　事情已经过去好多年了，关于先人的名字，眼前这刘关张三人也记不真切。今天是提到了，你一句我一句才把祖先的往事补充完整。
<br/>　　到他们这一代，张肥学了酿酒，关鱼继承祖先遗志，平素喜读兵书，刘悲却是和一个姓华的人学了医术。
<br/>　　听完这些，程大雷心中唏嘘不已。浊世洪流，人在其中太手足无措了。若是给他个舞台，那个半疯半癫，被顽童嘲笑的白发老头，便是威名千古的大汉昭烈帝。
<br/>　　谁能相信，那个在长街之上，口呼『我视天下英雄如土鸡瓦狗』的老者，他腹中的十万豪情都是真的。
<br/>　　目光又落在三人身上，程大雷心中想着，这三人未来能成长为那传说中的刘关张么？
<br/>　　的确，三人目前的等级还很低，而且都是生活职业，可按游戏术语来讲，那就是三人的成长性是极高的，上限是传说等级的。
<br/>　　未必没有可能啊。
<br/>　　“日后，你们就留在山寨，大家以兄弟相称，不必在意……”程大雷说着，忽然一顿，微倾着身子问道：“你们……认得一个姓诸葛的人么？”
<br/>　　“……”刘备。
<br/>　　“……”关鱼。
<br/>　　“……”张肥。
<br/>　　程大雷安排三人下去休息，让三人先熟悉山寨环境，也不用着急给三人安排工作。这三人可都是成长性极高的宝宝，累坏了怎么办……
<br/>　　这时孙九才将苏樱的信递给程大雷。程大雷草草读了一遍，意识到这是好事。辣椒只靠湖底捞一家酒楼出售，收入是不错的，但毕竟销路太窄。而辣椒是调味品，也并不是主食，只靠一家酒楼也消耗不了太多。打开销路是迟早的事，只是没想到今天就有人找上门来。
<br/>　　程大雷把柳芷唤来，由她代笔给苏樱回信，信中只是说：价格方面完全可由苏樱做主，供应方面不用顾忌，对方要多少，自己这里都可以提供的。
<br/>　　写罢之后，程大雷让信鸽把信传给苏樱。
<br/>　　山寨的辣椒又收割一茬，产量并不是很高，亩产在三千斤左右。程大雷记得前世辣椒的亩产量都可以上万的，可能是他种植不得法，或者是气候原因。辣椒毕竟是新物种，山寨懂得种植的也没有，只能慢慢摸索。
<br/>　　如今山寨大概有三万余斤的辣椒，满满的存在仓库中。
<br/>　　当初，还将一些辣椒苗外包出去，只不过其他山寨都以为自己忽悠他们，没几家当真。却也是有几家山寨取了一些辣椒苗回去中，日子算下来，当也该收获了。
<br/>　　程大雷让徐神机在山下贴出告示：蛤蟆寨开始收购辣椒，每斤辣椒换一斤粮食。
<br/>　　
<br/>(本章完)
