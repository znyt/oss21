　　大量的武者来回进出归墟天，也让归墟城变的无比繁华了。
<br/>　　毕竟归墟城可是归墟天唯一的入口，眼前这一座归墟城，虽然才建立起来不足半年时间。
<br/>　　但是其中繁华，已经足以是跟九域之中的许多存在了千年，万年的大城相比了。
<br/>　　“倒是大变样了。”
<br/>　　楚风眠笑了笑。
<br/>　　他上一次来归墟城，还是一个小家伙，神力境的武者，转眼过去，楚风眠已经是成就了准帝。
<br/>　　无数的武者妖神，在这归墟城中进出。
<br/>　　西域如今是一片混乱，但是在归墟城，倒是还有些秩序。
<br/>　　“先进去看看再说。”
<br/>　　楚风眠想了一下，现在也不着急直接进入归墟天，现从这归墟城里看看再说。
<br/>　　对现在九域的变化，楚风眠还是听说居多，究竟是如今情况如何，也应该楚风眠自己来判断。
<br/>　　楚风眠走进城门，两名武者便是拦了过来。
<br/>　　楚风眠看了一眼这两名武者，眉头不由一皱，这二人的身上，都有着浓郁的血腥味，而且一身功法，都是魔功。
<br/>　　全都是六道魔门弟子。
<br/>　　看来这归墟城，居然是落到了六道魔门的手中了。
<br/>　　“进城，每个人一块极品灵石。”
<br/>　　一名六道魔门弟子看了楚风眠一眼，语气冷淡的说道。
<br/>　　现在的九域也已经不再像是之前一般，物资匮乏的，大量的灵石，也随着一座座古老宝藏的开启，而开始出现。
<br/>　　极品灵石，王品灵石这种在以前极为少见的灵石，现在已经是大量的出现了，都是硬通货。
<br/>　　当然皇品，甚至是圣品级别的灵石，就算是在许多古老宝藏之中，都很少出现，依然是价值非凡。
<br/>　　楚风眠不动声色间，手心一动，拿出了一枚极品灵石直接递了过去。
<br/>　　这两名六道魔门弟子便是退开，放任楚风眠进入了。
<br/>　　看来这些六道魔门弟子，还没有认出楚风眠来。
<br/>　　不过楚风眠出名的时候，是在九域剧变之前，九域剧变发生的事情太多了。
<br/>　　以至于楚风眠这个曾经九域天才榜的第一人，如今认识的人，也是寥寥无几了。
<br/>　　正好，倒是也省了不少的麻烦。
<br/>　　楚风眠步入到了归墟城之中，一座座商会，一眼就足以望得见。
<br/>　　什么万器阁，通天阁等等，曾经九域的大商汇，也因为九域剧变，也变得更加繁华了起来，大量的武者进出其中。
<br/>　　而除了一座座商会，还有着一条街道的两侧，几乎全部都是坐在地上，摆摊的武者。
<br/>　　他们所卖的，多数都是从归墟天之中，得到的宝藏。
<br/>　　归墟天的宝藏，许多都是数十万年前，甚至是还要更加古老的，虽然被人得到，但是却难以参悟出其中的奥秘。
<br/>　　这样的宝藏，卖给各大商会，也没有很高的价格，毕竟这东西一日没有被认出来，也就一日没有价值。
<br/>　　所以许多的武者，宁可是摆摊去卖，也不愿意卖给商会，想要卖出一个高价。
<br/>　　“这位前辈，这是数十万年的一件古灵器，我在一座废墟之中翻到的，只要十枚皇品灵石，就卖给前辈。”
<br/>　　楚风眠刚刚走过去，旁边一名武者，便是急忙的对着楚风眠吆喝道。
<br/>　　他的手中，是一件罗盘，上面刻画着极为古老的纹路，散发着微弱的灵力，好像是沉睡的灵器一样。
<br/>　　楚风眠看了一眼，便是没有理会，这罗盘根本就是一个假货，一件寻常的灵器罢了，只是以一些手段造假出来的。
<br/>　　也许蒙蒙其他人还可以，可楚风眠却是足以一眼看的出来。
<br/>　　“前辈，您在看看？这可是古灵器……”
<br/>　　那名武者还想要在说些什么。
<br/>　　楚风眠眉头一皱道。
<br/>　　“滚！”
<br/>　　“前辈打扰了。”
<br/>　　那武者脸色一变，急忙的收起罗盘离开了，准备去找下一个目标。
<br/>　　在这归墟城内的摊位，所卖的东西，许多是真正的古灵器，一些从废墟之中出土的宝藏。
<br/>　　但是也有大部分，是一些武者故意弄出来的假货，用来蒙蔽人的。
<br/>　　不时可以听见一声声的争吵。
<br/>　　楚风眠也看了一些摊位上的东西。
<br/>　　这一些摊位上摆放的，的确是从归墟天之中出土的宝藏，许多都是年代久远。
<br/>　　但是越是年代久远的灵器，或者是灵药，都越是有可能失去灵力。
<br/>　　毕竟数十万年的岁月，足以让帝尊都陨落，灵器也不例外，会失去灵力。
<br/>　　当然像是祖龙至尊剑，甚至是人皇罐这个级别的灵器，并不会因为时间而消逝。
<br/>　　但是一些寻常的天级灵器，却是撑不住这么久的时间。
<br/>　　一旦是失去灵力，再好的灵器，圣药，现在也就是一团废物，没有任何的意义。
<br/>　　“看来还是要去商会看看。”
<br/>　　楚风眠一边走着看着，一边心中想到。
<br/>　　一些好东西，应该都是被各大商会给收走了。
<br/>　　楚风眠现在财富无限，如果可以从各大商会之中，多买一点圣药，甚至是诸天奇物的话，也足以助楚风眠提升实力突破。
<br/>　　就在楚风眠准备离开的时候，突然他的目光，看向了其中一个摊位。
<br/>　　在这个摊位之上，摆放这的，是一本古籍，一本看起来无比破旧的古籍。
<br/>　　在这古籍之上，都没有一点点的灵力散发而出，仿佛就是一本腐朽的古籍一样，根本不像是武技，功法。
<br/>　　可楚风眠看到这古籍的第一眼，便是就被这古籍给吸引住了。
<br/>　　在他的心中，好像是感觉到，这古籍有着一种在疯狂吸引着楚风眠的感觉，让楚风眠必须要买下他。
<br/>　　“这一本古籍，怎么卖？”
<br/>　　楚风眠走了过去，开口道。
<br/>　　这摊位的主人，是一名老者，一身破破烂烂的，看起来已经是接近奄奄一息的躺在那里。
<br/>　　楚风眠的走了过去开口，他还是没有一点的反应。
<br/>　　“这一本古籍，怎么卖？”
<br/>　　楚风眠加大了一点声音，再度开口道。
<br/>　　“年轻人，你要买这老骗子的东西？”
<br/>　　那老者还没有开口，从旁边突然传来了一道声音，是临近这老者的一名年轻男子。
<br/>　　他也摆放着一个摊位，看了楚风眠走了过去，对着楚风眠开口道。
