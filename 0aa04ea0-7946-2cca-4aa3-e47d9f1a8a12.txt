<br/>        “那我等你的消息。”
<br/>
<br/>        “好！”
<br/>
<br/>        明司寒跟裴萱妍挂了电话后，便一直握着手机，两眼盯着手机屏幕盯得发直。

<br/>

<br/>        突然，一面镜面墙壁出现了一个黑洞，里头传来脚步声，很快，那脚步声的主人，便出现在了明司寒的视野里。

<br/>        中年男子的左腿是假肢？！

<br/>        “这位是乔治先生。”带领明司寒过来的男人，毕恭毕敬地介绍。
<br/>
<br/>        明司寒主动迎上前去，十分礼貌地向乔治先生欠了欠身。
