　　第65章
<br/>　　青牛山下，一少年从落叶城方向快步行来。他穿着粗布衣服，脸上像是蒙着一层黑灰，看上去其貌不扬。但若是仔细看，就会发现她一双眼睛黑露露的，有汗水流到脖颈，露出其下干净素白的肌肤。
<br/>　　少年竟是一个少女假扮的，她走在山道上，眼睛警惕地看着四周，这条路并不太平，她生怕会遇到山贼。
<br/>　　“小姐也不知怎地想的，她现在这样都是那恶人害的，现在出了事，却让我去找那恶人。”
<br/>　　这少女正是苏樱的丫鬟小蝶，她这次过来，要去的正是蛤蟆寨。
<br/>　　在程大雷大闹黄家之后，黄员外就忙不迭的同苏家退了亲，这是好事，却也是坏事。整个落叶城都传，苏樱是蛤蟆大王的女人，谁如果敢娶她，小心那蛤蟆大王上门。
<br/>　　如此，苏樱身上就背了两个人设，一是克得夫家灭门的寡妇，二是被山贼糟蹋过的女人。
<br/>　　这两条，条条要人命。
<br/>　　重病的苏家老爷，一怒之下将苏樱赶出家门，给她的，不过一座残破不全，没有生意的酒楼。
<br/>　　如今主仆二人的日子苦不堪言，小姐一个女儿身，只得换做妇人打扮，以黑纱罩面在外面抛头露面。生意差到极点的酒楼，根本支撑不起主仆二人的生活。而且平时还有些浑人闲汗在酒楼里惹是生非，打小姐的主意。
<br/>　　这一切，岂非都是拜那恶人所赐。
<br/>　　可偏偏小姐还让自己来寻他，难道那杀人不眨眼的恶贼竟会帮助小姐吗？小蝶心中想不通这些事，小姐让自己来，自己自然就来了。至于自己会不会被那恶贼留下杀掉，也就顾不得了。
<br/>　　……
<br/>　　那扭蛋机再次出现，骨碌碌旋转，随着咚的一声，一物从里面落了出来。
<br/>　　嘟，恭喜获得绝世的鬼面斧一柄。
<br/>　　程大雷找到自己的大斧，猛地一提，竟发现提不动。就这么一会儿，他发现自己这柄大斧似乎变了模样。
<br/>　　通体黝黑，即便斧刃也是黑色的，当迎着阳光注视时，好像斧面上有一只只恶鬼的面孔，看上去狰狞可怖。
<br/>　　鬼面斧：绝世兵器，重108斤，传说中马家村牛铁匠所制，附带技能：热血。
<br/>　　程大雷查看系统界面，获得这样的信息，看来猜得不错，系统奖励的战斧将自己原先的破斧取代了，也可以说，原先的斧头升级成绝世等级。
<br/>　　糟糕、普通、精致、顶级、绝世、传说，这既是人物的等级，也对应物品，能达到绝世，在整个帝国也算得上屈指可数的神兵利器。
<br/>　　至于鬼面斧附带的热血技能，应该是战斗力方面的增幅。
<br/>　　照理来说，自己这次抽奖的奖励还是不错的。可问题是，程大雷正寻思着如何把兵器换了，怎么偏在这时候抽到一柄品质相当不错的斧头。
<br/>　　还有一个问题，程大雷也不得不考虑，那就是——这斧头自己目前拿不动了。
<br/>　　原先的斧头种五六十斤，程大雷勉勉强强能抡得动，他上阵的机会本来就不多。可现在斧头升级后，重量达到可怖的108斤，程大雷也勉强能提起来，但若说抡着他上阵打仗——委实有些强人所难。
<br/>　　看来自己还得想办法提升一下力气。
<br/>　　目前只能暂时搁着，唯一能做得，怕也就是扛着它吓唬吓唬人。
<br/>　　这时柳芷推门进来：“大当家，你饿不饿，要不要我煮面给你吃？”
<br/>　　柳芷最近努力让自己像个丫鬟，她是抱定嫁鸡随鸡嫁狗随狗的心思，只不过，她在这方面实在没什么天分。端茶倒水，洗衣做饭……简直一窍不通，尤其是做饭，每次都令程大雷怀疑，对方是不是怀恨在心，想要下毒害死自己。
<br/>　　“饭就不吃了，我教你的女子体操学得怎么样？”
<br/>　　“嗯，我一直很努力在学。”
<br/>　　“操练起来，我看你最近有没有用过。”
<br/>　　“嗯。”
<br/>　　柳芷将裙摆提起，露出一双粉色藕丝面的绣鞋，虽然以抱定为奴为婢的心思，可这样站在程大雷面前，她还是有些羞怯的。
<br/>　　她起手以笨拙的姿势，模仿程大雷所教的那些古怪姿势，脸上泛起津津汗珠。
<br/>　　“大当家，这些有什么用呢？”
<br/>　　“哈，强身健体的，人呐，永不能松懈，那句话怎么说来着，这世界有人比你聪明还比你努力……”
<br/>　　“嗯，我会努力的。”
<br/>　　“记得，以后要好好吃饭，尤其是豆制品，你这个年纪，最重要的事情就是发育。”程大雷循循善诱，毁人不倦。
<br/>　　正这个时候门突然开了，一『少年』站在门口，睁大眼睛看着汗津津的柳芷。
<br/>　　“你们在做什么！”
<br/>　　程大雷愣了一会，才认出这唇红齿白的少年正是小蝶。
<br/>　　“你怎么过来了？”
<br/>　　小蝶没有回答，迈步走进屋子，目光狐疑的盯着柳芷。
<br/>　　她是认得柳芷的，柳家和苏家有些交情，两家的小姐算是熟识，这是真正的闺中密友。
<br/>　　“她怎么会在这里？”
<br/>　　“哈……”程大雷拦住准备开口的柳芷，总觉得太阳穴跳个不停，脸皮有些发烧。
<br/>　　小蝶目光在屋里转了一圈，床上的两床被褥是瞒不住人的，其他就更不用提了。
<br/>　　她忽然鼻头一酸，紧跟着一句话吓了程大雷一跳。
<br/>　　“妄小姐对你朝思暮想，你却在这里风流快活。”
<br/>　　“咦，她也在想我么！”
<br/>　　最后总算是问明白苏樱那边的情况，程大雷心下惭愧，这几天忙得连轴转，真的没顾得苏樱那边。
<br/>　　“这有什么，我今晚就差人去落叶城，把你家小姐抢过来。”
<br/>　　“呸，你想的美，我家小姐才不会陪你做山贼呢。”小蝶啐了一口：“再说，自从你上次大闹黄家后，全城都在通缉你，你根本进不了城。”
<br/>　　“这样啊……”程大雷叹了一声，眉头微微颦起。
<br/>　　嘟，『抢夺压寨夫人』第三段任务触发：帮苏樱经营酒楼成为落叶城第一。
<br/>　　脑海中收到系统提示，程大雷眼珠一转，忽然对柳芷道：把黄总管和廖甲廖乙叫来。
<br/>　　柳芷去了，程大雷目光转到小蝶身上，用自以为磁性的声音道：“小蝶呐……”
<br/>　　“你想干嘛？”小蝶一激灵，原因是程大雷的眼神实在太猥琐了。难道她支开柳芷，就是为了对自己用强。
<br/>　　果然，坏蛋露出了狰狞的面孔。
<br/>　　程大雷从屋里翻出一匹花布：“喏，拿去，做两件新衣服……这里的事情，可不要告诉你家小姐喔。”
<br/>　　
<br/>(本章完)
