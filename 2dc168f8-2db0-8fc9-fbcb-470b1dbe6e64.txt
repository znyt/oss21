陆寒霆并不知道自己将车开的有多快，当卡车刺目的双闪灯打了过来，他瞬间觉得双目刺痛，出了一身的冷汗。
<br/>
<br/>        年少时曾经几次走入绝境，面对死亡他没有任何的恐惧，甚至会觉得有一些解脱，可是现在的他突然瞳仁骤缩，大手迅速打了方向盘，自救。
<br/>
<br/>        他不想死。
<br/>
<br/>        现在的他一点都不想死。
<br/>
<br/>        他变得很惜命。
<br/>
<br/>        因为，他有了陆太太。
<br/>
<br/>        自从有了陆太太，他对这个人世间充满了留恋，脑袋里电光火花之间，都是跟她在一起开心甜蜜的瞬间。
<br/>
<br/>        没有遇到她之前，他并没有觉得自己的病有多么糟糕，日子过得也不好也不坏，后来遇到了她，开始贪恋一切抓在手心里的温暖，他想让自己变好，他还跟她说过，以后的人生一半把肾给她，一半养她和孩子。
<br/>
<br/>        可是，现在一个都没有实现。
<br/>
<br/>        陆寒霆猛地打了方向盘，劳斯莱斯幻影豪车跟卡车几乎是擦身而过，很快豪车撞上了护栏和墙壁，砰一声溅起了一路的火花然后才停了下来。
<br/>
<br/>        劳斯莱斯幻影是世界级豪车，在遇到意外车祸时它的防御功能也是最好的，现在整辆车都报废了，但陆寒霆也躲过了一劫。
<br/>
<br/>        他趴在方向盘上，身体经过剧烈的撞击就像散架了一样，视线里都是黑的，什么都看不见。
<br/>
<br/>        很多人跑了过来，打开了驾驶车门，“先生…先生…你可以听到我说话吗？”
<br/>
<br/>        陆寒霆两耳嗡嗡作响，没有给这些人回应，他伸出满是鲜血的手，到副驾驶座上一阵摸索。
<br/>
<br/>        “先生，我们已经打120了，你现在伤的很重，最好不要乱动，你想要找什么？”
<br/>
<br/>        众人看着陆寒霆，男人一身笔挺的黑色西装，现在哪怕是出了车祸也严谨矜贵的丝毫不乱，加上这辆几千万的豪车，一看就是身份显赫的权贵。
<br/>
<br/>        现在他的身上都是血，分辨不出哪里受伤了，面对这些救助他视若无睹，只是很执着的在摸索着什么东西。
<br/>
<br/>        “先生，你是不是在找手机？”
<br/>
<br/>        这时有一个女人将已经掉落在门缝里的那只手机递给了陆寒霆。
<br/>
<br/>        陆寒霆接过了手机，轻易的就拨出了夏夕绾的手机号码，她的手机号码在什么位置上他烂熟于心，根本就不需要看。
<br/>
<br/>        电话拨出去了，那端悠扬的手机铃声响了一遍，迅速被接起了，众人听到那端传来了一道清丽轻甜的嗓音，“喂，陆先生。”
<br/>
<br/>        那边还传来了别人的笑声，“夕绾，又是陆总的电话啊，看来陆总很粘人啊，几分钟就来一个电话，不会是不放心你吧？”
<br/>
<br/>        众人看着满脸是血的男人，只见陆寒霆扯了一下苍白的薄唇，嗓音低哑的唤了一声，“绾绾。”
<br/>
<br/>        那端的夏夕绾很快就听到了他声音的异常，耳畔的吵杂声都消失了，应该是她跑到了一个安静偏僻的地方，紧张的问，“陆先生，你怎么了，你的声音怎么怪怪的？”
<br/>
<br/>        “绾绾，你现在就回来，好不好？”陆寒霆攥着手机，喃喃道。
