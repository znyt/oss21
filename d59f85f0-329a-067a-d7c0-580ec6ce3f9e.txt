　　面对群情激愤的众人，秦吉胜又能说什么，原本他就在举棋不定的犹豫之中，现在看到众人这样，把心一横：“好，那就出兵，各部马上回去整顿兵马，白天休息一天，晚上……晚上子夜之后，全军出战，给唐军来个闪电攻击！”
<br/>　　“噫！”一群小鬼子兴奋的摩拳擦掌，对于刚刚山背大兄和秦吉胜两人讨论的那些东西就好像没有听到过一样。
<br/>　　在他们看来，二十三万装备精良的军队，去打十几万毫无准备的唐军似乎一点问题都没有，因为他们熟悉这里的地形，加上又是夜战，唐军一定会因为准备不足而产生慌乱，到时候可就是他们一展身手的时候了。
<br/>　　至于说埋伏，这个小鬼子们并不担心，有道是只有千日作贼，没有千日防贼，这种不知道什么时候才会发起的进攻根本就没有办法去防御和埋伏。
<br/>　　就好像我们去下套子去套野兔，只有你在确定了野兔出入的路线，知道它一定会在某一刻出现的时候，才会布下套索。
<br/>　　可是如果是一片平地，哪怕你明知道这里将来可能会有兔子经过，但是你也不会把套索布在那里，因为你虽然知道兔子会从这里过，但是却不知道它什么时候会从这里过。
<br/>　　抱着这样的心思，小鬼子们各自回去准备不表。
<br/>　　却说当天晚上，月黑风高，端得是一个偷袭的好日子。
<br/>　　二十三万倭人军队在天黑之后便已经全都从床上爬了起来，等到午夜子时正好吃过了晚上的‘断头饭’，整理好行装，准备出征。
<br/>　　……
<br/>　　唐军大营中还亮着火把，无数巡营的哨兵在营地中走来走去。
<br/>　　大部分的大唐骑兵还都在休息，这是一天中人们睡眠质量最好的时刻，很少有人会在这个时间还是那么精神。
<br/>　　但这些人却并不知道，一场危机正在悄然降临。
<br/>　　突然，一阵尖锐的竹笛声响起，巡营的哨兵闻声纷纷一愣。
<br/>　　但是，还没等他们反应过来发生了什么的时候，一片黑云已经笼罩了下来，覆盖了几乎四分之一的营地。
<br/>　　“啊……”凄厉的惨叫声随着黑云压下在唐军大营中回荡。
<br/>　　“敌袭！反击！反击！”“快去马厩，把马赶出来……”。
<br/>　　两轮箭雨过后，唐军大营几乎像炸开了锅一样，无数人影在营中奔走，但很快就倒在了那铺天盖地的箭雨当中。
<br/>　　“放火箭，火箭！三轮速射，然后总攻！”秦吉胜和山背大兄此时也在小鬼的偷袭的大军之中，借着缴获来的望远镜看着模模糊糊的唐军大营，在发现唐军阵脚大乱的时候，果断的下达了总攻的命令。
<br/>　　长箭如织，雨点一般砸向唐军大营，无数人惨叫着四散奔逃向着背对着小鬼子的方向逃走，这些都是山背大兄王在望远镜中看到的情景。
<br/>　　秦吉胜同样看到了如此场面，而且不但如此，他还看到了那些骑着哈士骑冲进唐军大营的骑兵正在肆意杀戮，冲天大火在唐军大营中开始燃烧。
<br/>　　这还是倭人第一次在面对唐军的时候取得如此大的战果，这让秦吉胜和山背大兄两人几乎忘记了彼此间的仇恨，两只手紧紧的握在了一起。
<br/>　　而既然已经取得战果了，那自然就要乘胜追击，看着那些逃遁远去的身影，倭人骑兵果断的追了上去。
<br/>　　“陛下，我们也过去看看吧！”看着唐军大营中的熊熊大火，秦吉胜实在是很高兴，对身边的山背大兄邀请到。
<br/>　　“朕也正有此意，服部将军，请！”“请！”
<br/>　　两个小鬼子的最高领导互相谦让着，并骑而行，目光虽然一直在看着前方，但却都将一半的注意力留在了对方的身上。
<br/>　　仗打到现在这个样子，获胜已经是板上钉钉的事情了，接下来他们需要做的事情那就是消灭内部不和谐的声音。
<br/>　　……
<br/>　　唐军的大营中到处都充斥着一股子烤肉的味道，味上去让人十分不舒服。
<br/>　　无数军帐此时正在熊熊燃烧着，帐篷下面估计一定全都是尸体，否则绝不可能发出这样的味道。
<br/>　　大队的倭人骑兵已经冲过了这片区域，向着远处的黑暗不断前进，那里还有大量大唐的溃兵，不将他们全部杀死不足以泄恨。
<br/>　　秦吉胜十分开心，他万万没想到这一次的进攻会如此顺利，没有遇到任何的阻碍。
<br/>　　看来骄兵必败这个成语并没有说错，一个月左右的攻无不克已经让大唐铁骑忘记了自己是在异国的土地上战斗，所谓的天时、地利、人和几乎一样不占。
<br/>　　但就算是这样，能够如此快的杀进唐军大营也是秦吉胜所没有想到的，他已经有了付出一半战力的准备，可没想到却没有费多大力气就已经完成了既定目标。
<br/>　　营地中到处都散落着兵器和护甲，那些三尺长的横刀上面带着干涸的血污，这应该是那些靺鞨人不养于保养兵器的原因造成的。
<br/>　　钢弩什么的被丢的到处都是，显然是仓促间那些大唐的靺鞨骑兵找不到箭矢，所以连弩也都丢掉了。
<br/>　　至于护甲，应该是为了跑的快一点吧，反正也来不及穿，不如丢掉还能减轻一些负重，跑的也能比同伴快一点。
<br/>　　看着地上散落的东西，秦吉胜分析着一切可能的经过。
<br/>　　身边已经开始有士兵在打扫战场，那些钢弩被收了起来，这一次他们没有丢掉，也没有破坏掉，因为这一次是倭人大获全胜，说不准还能找到箭矢，那样的话这些钢弩就能派上用场了。
<br/>　　只不过，让人有些奇怪的是，唐军大营里虽然到处都是血迹，可是却没有什么尸体，这让秦吉胜有些担心，总觉得有些奇怪。
<br/>　　而与秦吉胜不同的是，山背大兄王显的很兴奋，一路上指点江山，意气风发，就好像这一仗是由他指挥的一样。
<br/>　　这让秦吉胜有些不满，不过他却没有说出来，只是在心中留了一个疙瘩。
<br/>　　大战还没有结束，一切都还没有结果，唐军虽败但却还没有到山穷水尽的程度，接下来秦吉胜还要乘胜追击，努力将唐军彻底消灭干净。
<br/>　　想着想着，秦吉胜忽然抽了抽鼻子，有些疑惑的四下看了一眼，最后将目光停在不远处的一道营地中的排水沟里面。
