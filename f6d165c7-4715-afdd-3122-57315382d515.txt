　　第152章
<br/>　　快马来到青牛山，徐神机将信送到程大雷手中。
<br/>　　“大当家，幽州那边来的。”
<br/>　　“喔。”程大雷将信接过来，随口问道：“人现在在哪儿呢？”
<br/>　　“我让他在飞虎寨等着，他说要等大当家的回话。”
<br/>　　程大雷把信打开，粗略扫了一遍。
<br/>　　“大当家，信上写什么啊？”
<br/>　　“让我们抽一千兵，去帮他们打伏虎山。”程大雷随手将信丢到火炉里。
<br/>　　“这……”徐神机道：“大当家，这兵咱们不能给他派啊，明显是想让咱们狗咬狗，可如果不派兵，怕他会直接过来打我们。这……这……大当家你说该怎么办？”
<br/>　　“吃人家一颗枣，就得给人拉出一棵树，这就是江湖道的规矩。得，现在人家要咱还树来了。”程大雷一挥手：“来人，给本大当家取纸笔来。”
<br/>　　宣纸铺开，笔蘸浓墨，略沉吟片刻，笔落在纸上，唰唰给李善言写了一封回信。
<br/>　　『尊贵的军师大人……』
<br/>　　徐神机揣着程大雷的回信来到飞虎寨，将信交给那兵丁。说也奇怪，这兵丁接过回信，二话不说骑上马就走。
<br/>　　……这么着急干什么，我还准备招待招待呢。徐神机冲人家背影喊：“不留下吃饭啦？”
<br/>　　腾！
<br/>　　对方走得更快了。
<br/>　　马不停蹄，连夜返回幽州城，将信递到李善言手中。
<br/>　　李善言沏了杯茶，一边品茶一边看程大雷给自己的回信。
<br/>　　『尊贵的军师大人：
<br/>　　是那位运筹帷幄，算无遗策的李军师善言给我写的信么？我让山寨的马夫看，我让山寨的厨子看，我让所有兄弟一起看，仍不敢相信军师竟会亲自给我写信。这字里行间蕴藏着无上智慧，每一个字于我都是无上妙言，我逐字看，翻来覆去的看，我想要裱在墙上，让我的后人知道，他的祖先也有这样的荣耀，收到过军师的亲笔书信』
<br/>　　李善言端茶的手颤抖，身上一层层泛鸡皮疙瘩。他终于明白，裴喜之那满口胡话是从那里来的了。
<br/>　　他继续往下看。
<br/>　　『自从归顺幽州王以后，我每日都活在荣耀中，感觉吃饭也香了，走路也有劲了，一口气上五楼都不喘气了。这样的恩情，熊老大竟不知珍惜，山寨弟兄知道后，人人同仇敌忾，誓与熊老大不死不休，一千弟兄整装待发，随时听军师吩咐』
<br/>　　李善言点点头，这程大雷说话虽然肉麻，但看来还算懂事，至于肉麻……大概是初入官场，还不懂得怎么拍马屁，有些用力过猛。
<br/>　　『……不过，山寨五千余人，老弱妇孺便有三千，如今天寒地冻，山寨缺衣少粮，大家都是在山里扒树皮，挖草根度日。他们若出去打仗，山寨老弱便没人扒树皮，挖草根了，当然，为幽州王做事，大家责无旁贷。有些老人已在准备绳索，宁愿自尽也不成为幽州王的累赘。但我想，幽州王仁道为怀，必不会让这一幕发生。所以我想向幽州王借一些物资，物资清单随信附上』
<br/>　　李善言此时才注意到信封中还有一张纸，他抽出来后，在桌上打开。
<br/>　　『稻谷十万斤』
<br/>　　李善言吓了一跳，程大雷这是饿疯了不成，张嘴就是狮子大开口。
<br/>　　『皮甲两千副
<br/>　　军靴一千双
<br/>　　军马五百匹
<br/>　　铁弓一百张，羽箭十万支
<br/>　　……』
<br/>　　李善言是一口茶都喝不下去了。本以为程大雷因为山寨缺粮，所以想要些口粮过冬，这于情于理却也不算什么。可程大雷开下的这份物资清单……你是要攻打幽州城么！
<br/>　　这该不会是耍自己吧！
<br/>　　李善言想了想，揣着这封信连夜去了杨龙停的王府。
<br/>　　杨龙停看过这封信，沉默了好久。
<br/>　　“他是得了失心疯么！”
<br/>　　“我……也不太能明白他怎么想的，特意禀告城主，由城主决断。”
<br/>　　杨龙停想了想，道：“他若不想出兵，却也在情理之中，毕竟是他自己的人，自然不想给我们去用。只是不知他心底究竟什么态度……这样，仓库还有几百双军靴，差人与他送去。告诉他，什么时候杀了熊老大，其他物资补齐。”
<br/>　　“真要给他物资？”李善言。
<br/>　　“试试他会不会出兵而已，我还没有富到拿自己的钱养山贼的地步。”
<br/>　　杨龙停在房中踱了两步，忽然道：“伏虎山那边情况怎样？”
<br/>　　“罗铁已去办这件事，点了他一千兵马，在加上绿林道的匪人，共两千人，这两日便该有消息。”
<br/>　　“伏虎山地形复杂，多是山岭，不易骑兵进击。罗铁武勇虽有，谋略却不如何，那边的事情你可要多盯着些。”
<br/>　　蛤蟆寨热火朝天，杨龙停也并没歇着。准备粮草，操练兵丁是每一日都在进行的。攻打伏虎山则算是当下的一件大事，其中也有练兵的用意。
<br/>　　这一日，罗铁领兵两千，浩浩荡荡兵发伏虎山。
<br/>　　在罗铁想来，伏虎山不过是乌合之众，自己大旗所到之处，便可将对方杀得片甲不留。
<br/>　　结果，惨败。
<br/>　　仗着地利之便，伏虎山诱敌深入，将对方兵马分割，各个击破，两千兵马丢盔弃甲，活下来逃回幽州城的不足千人。
<br/>　　熊老大能在幽州境内作威作福这么多年，外人当他是个莽夫，可真若只是个莽夫，他能活这么久么？
<br/>　　熊老大是有脑子的。
<br/>　　本来杨龙停是想借这场仗露脸的，结果不仅露了脸，连屁股都露出来，裤衩都没剩下多少。
<br/>　　看着站在自己面前的罗铁，如果不是因为对方是故人之子，杨龙停真一刀劈了对方的心思都有。
<br/>　　真是烂泥糊不上墙啊，想要你在人前露脸，日后委以重任，结果你是让我的脸结结实实砸在地上。
<br/>　　杨龙停大怒之下，将罗铁一撸到底，将他从参将直接撸到运粮的马夫。日后能不能爬起来，就看你自己了。
<br/>　　终是有那点儿故人之情，杨龙停还是给罗铁留了重新站起来的机会。
<br/>　　罗铁兵败之后，杨龙停亲自统兵，依旧点一千兵马，包括罗铁原先领的残兵败将。
<br/>　　一日之内，点齐兵马，杨龙停兵发伏虎山。
<br/>　　
<br/>(本章完)
