        &lt;h3&gt;第四卷第十一章  温馨江家人&lt;/h3&gt;
<br/>
<br/>        江家大佬江爸爸到了西城，江俞轩和姐姐江俞琪亲自去机场接到了西城江俞轩的家里。
<br/>
<br/>        这个家还是一如既往的只有江俞轩一个人住，江爸爸看了看空荡荡的房间，对江俞轩说：“房子布置得很温馨，只是少了一点烟火气息，没有女主人的家不算家哦。”
<br/>
<br/>        江俞琪嗤笑一声：“咳，他现在都不知道谁是女主人了。”
<br/>
<br/>        “以前不知道还有情可原，现在不知道是不是太过荒唐？”江爸爸看了一眼江俞轩。
<br/>
<br/>        江爸爸注重养生，也经常锻炼，生活习惯很好，五十岁的人，看着就像四十岁不到，身型清瘦，眉眼凌厉却不失温柔。
<br/>
<br/>        “倩楠一直在北京，我们最近一直没有联系。”江俞轩解释着。
<br/>
<br/>        他想到过年的时候和父亲谈心，那个时候如果自己没有犹豫不决，剥开心扉和父亲谈一谈会是什么效果？
<br/>
<br/>        父亲的那句话似乎在表达他自己的意思？想一想自己确实有点荒唐，虽无夫妻之实却有了夫妻之名。
<br/>
<br/>        江俞轩沉默了，不过江爸爸到没有在他的婚姻上继续纠结，而是转变的话题，聊起了西城的发展和规划，父子三人倒是其乐融融。
<br/>
<br/>        “有人给我说，西城原本能拿下两个项目，而你坚持只拿下一个项目，我想先问问你的意思？”
<br/>
<br/>        江俞轩和江俞琪才明白爸爸突然到西城的目的，他们姐弟是被人打了小报告了。
<br/>
<br/>        “西城的两个项目我们确实是有能力拿下来，只是里面问题众多，高新的项目还是争取过来的，西岭的项目是万万不能动的。”江俞轩耐心的解释着。
<br/>
<br/>        “嗯？”
<br/>
<br/>        “这几年空气污染越来越严重了，有些城市已经开始大力整顿环境污染的问题，西岭是西城的自然屏障，而在西岭大力开发修建别墅是违反自然规律的，目前看着是一个赚钱的项目，后期后患无穷。我们作为开发商不单单只是看重眼前利益，还要为今后的发展做规划。
<br/>
<br/>        我曾经写过一篇文章，论西岭开发的重要性，论自然环境对人类发展的重要性，西岭如果开发是影响真个生态链的，这篇文章发展在西岭商报上，可惜被人拿掉了，爸，我们不能因为赚钱而破坏一个地方的生态环境。”
<br/>
<br/>        “爸，我也同意轩轩的说法，所以在开会的时候我们已经讨论过这件事情了，谁给你打报告了？”
<br/>
<br/>        “这件事情你们给集团写一个报告，要有理有据，用事实说话。我明天就走了，晚上准备给我吃什么？”江爸爸将自己的一双儿女放在西城，实在是有些想念他们，便找个机会过来看看。
<br/>
<br/>        “你想出去吃，还是在家里吃，要不带你出去吃西城的小吃？”江俞轩看着自己父亲问。
<br/>
<br/>        他还以为父亲能呆几天，谁知道父亲明天就走。
<br/>
<br/>        “吃什么都行，我也就是想过来看看。”江爸爸淡淡的应到。
<br/>
<br/>        “我还说介绍你和杨老头认识一下，杨老头人不错。”江俞轩看了一眼父亲。
<br/>
<br/>        “杨东？”江爸爸看了看江俞轩姐弟俩，他总是会时不时的想起这两个兔崽子，可是反观他们好像对他不感冒啊。
<br/>
<br/>        “是啊，你认识吗？”
<br/>
<br/>        “我不认识，听说过，西城轰轰烈烈的改变离不开杨东的努力。”
<br/>
<br/>        “那我打电话看看他今天有没有时间，然后在决定吃什么好不好？”
<br/>
<br/>        “问吧，问吧。”江爸爸审视的看着江俞轩，这么急急的介绍杨东和他认识是有什么想法？
<br/>
<br/>        江俞轩拨通了杨东的电话。
<br/>
<br/>        “杨总，您好。”因为有父亲在旁边，江俞轩不敢对着杨东喊“糟老头子”了。
<br/>
<br/>        “哈哈哈，今天这么客气？”杨东哈哈大笑。
<br/>
<br/>        “我想问你晚上怎么解决？要不晚上一起？天天去您家蹭吃蹭喝，来而不往非礼也。”
<br/>
<br/>        “今天晚上？”
<br/>
<br/>        “是啊，提前约着太见外了，像我们这种关系有时间就一起吃没时间就算了。”
<br/>
<br/>        “我问问你嫂子啊，我今天没有应酬，泽瑞好像也还没有走，如果可以了就一起吧。”杨东对江俞轩的邀请倒是不推辞，在他的心目中，江俞轩既是他的朋友也是一个小兄弟，完全就是一家人一样。
<br/>
<br/>        “好啊，我等你电话。”
<br/>
<br/>        江俞轩挂了电话然后对江爸爸说：“可能晚上会和杨东一家人吃饭，爸，你不介意吧？我和杨东这几年关系非常好。”
<br/>
<br/>        “哼，杨东年龄和我差不多大吧？”
<br/>
<br/>        “是啊，和你差不多大。”
<br/>
<br/>        “我说江俞轩，你和杨东都成朋友了，那么亲密的，为什么和我就很别扭呢？”江爸爸语带不满。
<br/>
<br/>        “噗嗤。”江俞琪忍不住笑了，这酸不拉几的口气是怎么回事？
<br/>
<br/>        “哈哈哈哈，老爸，我今天好像没有打翻醋坛子，怎么满屋子的酸味？”
<br/>
<br/>        “你......”江爸爸傲娇的转过头不理他们姐弟了。
<br/>
<br/>        “爸，我这是看你来了给你找个年龄相当的朋友陪你说说话，再说了，这几年你儿子也得亏了人家关照，我是您的儿子，哈哈哈，您的儿子啊，话说，咱俩这关系谁能铁得过？”
<br/>
<br/>        江俞轩说完冲江俞琪挤了挤眼睛，嘿，他们的老爹还真有意思。
<br/>
<br/>        杨东打来电话说不用出去吃饭，就在家里吃，让江俞轩去他家，江俞轩有点为难了。
<br/>
<br/>        “你爸爸来了吧？”杨东笑着问江俞轩。
<br/>
<br/>        “什么都瞒不过你啊。”
<br/>
<br/>        “叫上你爸爸和你姐姐，你候姐又给咱们下厨了，我和你爸爸可以聊聊天，你和泽瑞可以聊聊天，你姐姐就陪你候姐下厨，现在就过来。”
<br/>
<br/>        “你没有在单位吗？”
<br/>
<br/>        “我现在就回家，一会见啊，这一次是专门招待你老爸，小子，这一次是你蹭你老爸的饭。”
<br/>
<br/>        “我这天天蹭吃蹭喝好像不太好，要不我就不去了，让我老爸去？”江俞轩笑着说。
<br/>
<br/>        “那你将他送过来，你回去吧。”
<br/>
<br/>        江爸爸听着江俞轩和杨东的对话，看着他儿子神采飞扬的样子，心里是满足和幸福，孩子大了，已经得到别人的认可，从江俞轩和杨东的对话中，可以看出来俩个人的关系真的不错。
<br/>
<br/>        杨东在业界是出了名的严谨、严厉，一般的人在杨东的面前都是看着杨东的脸色行事，可是他的儿子却和杨东谈笑风生，这关系还真不是一般的好，能得到杨东的赏识，看来江俞轩已经成气候了，有些事情可以放手让他去做了。
