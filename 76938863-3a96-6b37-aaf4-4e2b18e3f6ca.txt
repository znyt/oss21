　　离开万金府，萧逸御空而离。
<br/>　　不过半个时辰，便已飞离了万金城。
<br/>　　整个万金地域，异常辽阔。
<br/>　　除了万金城外，自然还有大大小小无数小城。
<br/>　　萧逸随便在一座小城上落下，寻了间客栈，就此住下。
<br/>　　客栈房内。
<br/>　　嘭…嘭…嘭…
<br/>　　一个个屏障与禁制，瞬间布下。
<br/>　　萧逸特地去万金府一趟，一是护金卓回去，二则是买大量妖兽精血。
<br/>　　如今妖兽精血已买，自然是该修炼了。
<br/>　　萧逸盘膝坐于床上，顺着修罗战体第八层的功法，开始了修炼。
<br/>　　一边修炼，一边内视体内。
<br/>　　此时，体内一颗深黑色的‘小水珠’，异常醒目。
<br/>　　再看真切些，这些‘小水珠’，竟有足足6颗。
<br/>　　小水珠上，散发着的一股股毁灭气息，让人触目惊心。
<br/>　　那些便是修罗之力。
<br/>　　当初未有修罗战体后5层时，肉体力量卡在第七层巅峰，也就是天极境巅峰。
<br/>　　那时，体内的修罗之力，足够999滴。
<br/>　　当时萧逸便猜测，若要突破肉体力量，这999滴修罗之力，必是关键。
<br/>　　前些天，在修罗殿总殿得到后5层后，他果然证实了自己的猜测。
<br/>　　在修罗之体突破第七层巅峰，踏入第八层时。
<br/>　　也就是肉体力量突破天极境巅峰，踏入圣境时。
<br/>　　体内的修罗之力，瞬间达到1000滴。
<br/>　　也是那一瞬间，1000滴修罗之力，就此融合，乃至升华。
<br/>　　于是便有了这些‘庞大’的黑色小水珠。
<br/>　　当然，所谓庞大，只是相对于以前那一滴滴细小的修罗之力而言。
<br/>　　如今，体内这些黑色小水珠，随便一颗，其力量澎湃度与强猛度，远超从前千颗。
<br/>　　这就好比以前是1000个小球，如今1000个小球融合为一个巨大的球。
<br/>　　自然的，威力强猛了许多。
<br/>　　且球只是比喻，整颗黑色小水珠，相对从前，乃是质的飞跃。
<br/>　　此时，体内六颗黑色小水珠，便代表着他的肉体力量，达到圣境六重。
<br/>　　果然，修罗战体第七层，是个坎。
<br/>　　毕竟，这是天极境与无极境间的差距。
<br/>　　“无极境，武道无极。”萧逸皱眉自语着。
<br/>　　武道无极，这是这天地规则所定。
<br/>　　萧逸相信，这不仅适用于正常的武道修炼，亦适用于体修。
<br/>　　也就是说，他的肉体，也早在突破圣境那一刻起，踏入了武道无极的范畴。
<br/>　　而修罗战体，最大的力量来源，便是这些修罗之力。
<br/>　　武道无极，便也是修罗无极。
<br/>　　这些更加强大的修罗之力，会成长到什么地步，萧逸充满了期待。
<br/>　　萧逸的心神，渐渐收敛。
<br/>　　随着第八层功法的运转，萧逸手掌亦开始虚摄。
<br/>　　三枚乾坤戒，装满了圣境层次妖兽精血的一个，开始源源不断涌出一瓶瓶血玉瓶。
<br/>　　几乎是玉瓶出现的瞬间，里头妖兽精血，在萧逸的吸摄下，纷纷飞跃而出。
<br/>　　一滴滴妖兽精血，在巨大的数量下，化作一条精血长河。
<br/>　　妖血长河，不断从萧逸手中消失，进入萧逸体内。
<br/>　　萧逸亦明显感觉到，自己的肉体力量在不断上升。
<br/>　　同时，体内修罗之力，不断自主凝聚着。
<br/>　　不过一个时辰，乾坤戒内的所有妖兽精血，被萧逸吸摄一空。
<br/>　　当然，那里装满着的，是圣境层次的妖兽精血。
<br/>　　“圣王境，成了。”萧逸脸色一喜，自语一声。
<br/>　　堪堪在最后一瓶妖兽精血吸收殆尽之时，肉体力量再次突破。
<br/>　　此时，体内修罗之力，已达10颗。
<br/>　　不得不说，修炼修罗战体所需的妖兽精血，果真是个恐怖的数字。
<br/>　　乾坤戒内，如此大的范围，圣境级别的妖兽精血玉瓶，堆积如山。
<br/>　　可却在如此短短时间内，便吸收一空。
<br/>　　这还是圣境，那圣王境呢？
<br/>　　念及此，萧逸脸上的喜意，化作了微微皱眉。
<br/>　　“呼。”萧逸轻呼出一口浊气。
<br/>　　手中乾坤戒，又是光芒一闪。
<br/>　　一瓶瓶装着圣王境层次妖兽精血的玉瓶，凭空而现。
<br/>　　萧逸顺着修罗战体第九层的功法，再次修炼起来。
<br/>　　一如之前那般，一滴滴妖兽精血，被吸摄而出，随后化作血河。
<br/>　　萧逸一边修炼，一边吸收，一边也在内视体内。
<br/>　　明显可以感觉到，圣王境层次的妖兽精血澎湃度，远超之前的圣境层次妖兽精血。
<br/>　　萧逸的肉体力量，几乎是以肉眼可见的速度，不断飙升着。
<br/>　　又是数个时辰后。
<br/>　　待得乾坤戒内妖兽精血被吸收一空。
<br/>　　萧逸的眉头，紧紧地皱了起来。
<br/>　　因为，此时的肉体力量，仅仅是达到圣王境八重。
<br/>　　虽说圣王境层次的妖兽精血蕴含力量远超圣境妖兽精血。
<br/>　　但同时，修罗战体第九层与第八层间，同样差距极大。
<br/>　　原本达到圣王境后，体内有10颗修罗之力。
<br/>　　可之后的提升，几乎是每凝聚出10颗修罗之力，肉体力量才算突破。
<br/>　　也就是说，他现今肉体力量，达到圣王境八重。
<br/>　　体内修罗之力，亦足足80颗。
<br/>　　“圣王境层次的妖兽精血，不够了。”萧逸皱了皱眉。
<br/>　　他显然是微微低估了修罗战体所需的庞大妖兽精血。
<br/>　　不过也当然，毕竟第七层以后的功法，他是前些天才在总部得到。
<br/>　　自然的，对于后续层次所需的修炼之物，他也只能大概猜个准。
<br/>　　这，也造成了如今的窘迫，差些许才达到圣王境九重。
<br/>　　也就是说，距离突破第九层，还差一重多些左右罢了。
<br/>　　圣境层次的妖兽精血，圣王境层次的妖兽精血，已被他吸收一空。
<br/>　　事实上，他还有为数极多的圣皇境层次妖兽精血。
<br/>　　不过他想了想，并未打算吸收。
<br/>　　妖兽精血层次不同，效果也迥然不同。
<br/>　　如他之前圣境肉体力量时，圣境层次与圣王境层次的妖兽精血，都能发挥完满功效。
<br/>　　而现今圣王境肉体力量时，再吸收圣境层次的妖兽精血，则效果大打折扣。
<br/>　　最佳的效果为圣王境层次与圣皇境层次。
<br/>　　这个层次，大概保持在与自己肉体力量一致，或者高一境界效果最好。
<br/>　　既是如此，他宁愿将圣皇境层次的妖兽精血，留待肉体力量达到圣皇境之后再吸收。
