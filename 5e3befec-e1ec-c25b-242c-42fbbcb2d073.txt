　　第236章
<br/>　　“是呐，一个为人所不齿的戏子而已，哭也罢笑也罢，无非是大人们的玩物。”楚青衣的声音越来越虚弱：“因为得到太少的缘故，别人给一点一滴也得记着，哪怕是用命来偿。”
<br/>　　“你这样死得不值？”
<br/>　　“戏子的生命是活在台上的，有人一声都遇不上一出好戏，一旦遇上便要将自己一生心血耗尽。”
<br/>　　“我那一剑，是此生最完美的一个亮相，虽然只有一剑。”
<br/>　　楚青衣的声音越来越低，到最后竟浅浅唱起来。
<br/>　　“惜起残红泪满衣，劝君莫作有情痴，天地无处着相思……”
<br/>　　“花若再开非故树，云能暂驻亦哀丝……”
<br/>　　他终究不能再唱下去，没有完成最后一曲，头靠在程大雷肩头，就此香消玉殒。
<br/>　　程大雷无奈的摇摇头，将他从肩上斜下去，此时程大雷距离城墙缺口大概三十步，想走却也能走得脱。
<br/>　　“我替你报仇。”
<br/>　　程大雷合上他的双目，握剑逆行，烟花乱绽，铛铛声不停响起，打落一地兵器。
<br/>　　前方是扑过来的周声声，周声声是用棍的高手，但不代表他不懂得用枪，只是用枪没有用棍顺手而已。
<br/>　　一枪袭来，却是用棍的打法，点，砸，削，程大雷潇洒避过，欺近周声声的身体。
<br/>　　有禁卫军扑来，程大雷一记直刺刺穿他的手掌，将他手中的钢刀打下来。
<br/>　　再刺，便是攻向周声声。
<br/>　　“抱歉。”
<br/>　　直来直往，一剑穿喉。
<br/>　　血从周声声喉咙渗出，他睁大眼睛，身体扑通一声倒地，临死前口中冒出一句话。
<br/>　　“何必……抱歉？”
<br/>　　“大概是杀你晚了吧。”
<br/>　　程大雷摇摇头，目视周围，只见烟雾渐渐散去，徐神机大概已经带人走脱，连林冲都不见了，演武场内层层精兵，将程大雷团团围住。
<br/>　　“程大雷，你已是死路一条，莫再做顽兽之争。”李乐天高声道。
<br/>　　程大雷看看周围，忽然嘴角浮现一丝笑意。
<br/>　　“诸位可听说过一种踏风而行的功法？”
<br/>　　众人一愣，加紧提防。
<br/>　　“且看我身无彩凤双飞翼。”
<br/>　　程大雷身体凌空而起，真如同肋生双翅般踏风而行。
<br/>　　“下去吧，你飞不起来。”
<br/>　　忽然一个人影从城墙上跃在程大雷头顶，一掌向程大雷拍来。
<br/>　　程大雷举拳硬接其一掌，被拍了下来。见一个白袍僧人落在自己十步外，长袖飘飘，
<br/>　　这招从天而降的掌法，莫非是……如来神掌？
<br/>　　特么，还真飞不起来呐。
<br/>　　程大雷目视周围，看四面八方无数手持强弩的兵丁瞄准了自己。
<br/>　　程大雷义无反顾的出手，不仅出手，两只手一起出，高举在空中。
<br/>　　“我……投降。”
<br/>　　咦！
<br/>　　李乐天明显怔了怔，怎么投降得这么痛快，我已经要下令放箭了好不好。
<br/>　　他明显被噎了一口气，最后无奈的挥挥手：“抓了，打下天牢。”
<br/>　　……
<br/>　　“咱刑部天牢，不知关过多少人，我不问你在外面有多少钱，也不管你当过多大官，只要到了这里，是龙你得给我盘着，是虎你得给我窝着，一切都得听我燕三，我燕三让你活，你就能活，我燕三不让你活，你就活不了。”
<br/>　　秃头燕三大摇大摆走在天牢中，身后跟着一个刚送进来的犯人。
<br/>　　“犯了什么事啊？”燕三斜睨对方一眼。
<br/>　　“杀人。”
<br/>　　“哼，最看不起你们这些杀人犯，杀了几个人呐？”
<br/>　　“有……”犯人顿了顿：“总有那么几个吧。”
<br/>　　“哼，没本事的才杀人呢，长安城的大人物都用不着杀人，怎么把你这种小人物关进天牢了。”燕三来到一个牢房前，道：“进去吧。”
<br/>　　牢房十分狭窄，三面墙壁，一面铁栅栏，只有一扇小窗透气。
<br/>　　“进去是进去了，想出来可就不容易喽。”
<br/>　　燕三把门锁了，回头看到一个着锦衣的男人立在自己面前，面白无须，穿着丝绸质地的衣服，他没有穿官服，刚才在押送囚犯的队伍里，竟没有注意到他。
<br/>　　看他这个样子……很像是从宫里来的。
<br/>　　锦衣男人拿出一面令牌，只是扫了一眼，燕三得腰立刻弯下去：“您吩咐。”
<br/>　　“莫要对他用刑，一日三餐不要亏待，他想吃什么就让他吃什么，只要不是出去，其他要求尽量满足。”
<br/>　　呃……燕三腹诽：这他妈是坐牢还是住店呐。
<br/>　　“大人，他不就是一个犯人么？”
<br/>　　“不该你问的不要问。”锦衣男冷喝一声，面向那犯人时弯了弯腰：“爷，我先走了，您有什么需要，尽管吩咐牢头。”
<br/>　　程大雷无所谓的摆摆手。
<br/>　　锦衣男走后，燕三回头打量这个犯人，左看右看，上看下看，也没看出有什么稀奇。
<br/>　　“是不是感觉我很平平无奇？”程大雷问。
<br/>　　“嗯呐，你究竟是什么人？”
<br/>　　“去，给爷打盆洗脚水，让爷先解解乏。”
<br/>　　夜来，程大雷靠在稻草铺成的床上，百无聊赖的发呆。
<br/>　　咕咕，咕咕
<br/>　　窗外突然响起布谷鸟的叫声，程大雷顿时一精神，用手指敲了敲墙壁。
<br/>　　不多时，一个脑袋杵在小窗外，是飞天蛤蟆白元飞。
<br/>　　“大哥，你没事吧？”
<br/>　　“暂时没事。”程大雷问：“外面什么情况？”
<br/>　　“官府在大肆搜捕，不过听您的吩咐，大家都已悄悄撤出城，现在在葫芦山落脚，如今城里只留了几个弟兄。”
<br/>　　“那就好，有伤亡的弟兄，多给他们家人留些银子，咱们能做的就这些。”
<br/>　　“大哥放心，咱们伤亡并不大，有十几个兄弟受了伤，咱们大部分都被大哥救过命，没有大哥我们早就死了。这次大哥陷进天牢，兄弟们肯定会救大哥出来。”
<br/>　　“还有一个弟兄也被抓了进来，他叫林少羽，你查查他被关在哪里，也要救他出来。”
<br/>　　“是，外面的弟兄正在想办法，不过天牢防卫很严，即便我进来也险些被……”飞天蛤蟆一怔：“大哥，有人来了，我先撤了。”
<br/>　　说完后，飞天蛤蟆不声不响消失，感觉像一只夜里行走的蝙蝠。
<br/>　　几乎他刚走不久，程大雷就听到了脚步声，程大雷不由感慨：这飞天蛤蟆的耳力真好。
