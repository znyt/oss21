<br/>　　“他不是没有去找过，而是没有谈拢。”
<br/>　　比洛多正打算回答，莱昂抢在了他的前面代替他回答道：“首先是酬劳，要想让那些特种部队动手，并且还是在有无人机和卫星的帮助下，美国政府提出的酬劳不会是一个小数目，而且这个酬劳跟钱无关，应该是这里的矿藏，铜、锂、铁这些东西远比美金对美国政府有吸引力。”

<br/>　　比洛多眼眸低垂，没有做任何的反驳，显然莱昂所说的那些并不是他的胡乱猜测，而是实情！

<br/>　　比洛多知道，罗昊嘴里所说的“惹上了大麻烦”不过是他的一个说辞而已，如果说是在十年前，比洛多是会相信罗昊的话，可是放在今天，比洛多是不会相信的。

<br/>　　罗昊这么说，不过是想要得到更多的利益！
<br/>　　比洛多沉默了片刻，随后说道：“如果我能当选下一届的总统，我愿意跟你们国家签下一笔价值2亿美金的军火交易，其中一亿美金支付现金，剩余的一亿美金用等价值的矿藏来抵充。”
<br/>　　丫的，等的就是你这句话！
<br/>　　罗昊立刻咧开嘴笑得无比灿烂，对着比洛多伸出手，说道：“比洛多先生，希望我们合作愉快。”
<br/>　　小狐狸！
<br/>　　比洛多在心中嘀咕了一句，伸手跟罗昊握了握，问道：“对于行动，你有什么计划？
<br/>　　刚才在莱昂对比洛多没有选择跟美国军方合作进行分析的时候，罗昊在心中对整个行动已经有了一个初步的规划后，说道：“营救人质和斩首巴坦伦的行动同时进行。毒液，你带天蛇、玄龟和布谷鸟对巴坦伦进行斩首行动，我们负责营救千雪。”
<br/>　　“没问题，如果在巴坦伦没有特别防备的情况下，对他进行斩首行动的成功率会高出很多。”莱昂说道：“现在巴坦伦应该只是把我们当成了是比洛多先生请来保护他女儿的雇佣兵，还不会想到我们的目标是他和布克尔特。”

<br/>　　“比洛多先生。”
<br/>　　罗昊看向比洛多，说道：“我需要你的部队配合我们营救人质，另外也对布克尔特的cocaine种植园和提炼加工基地采取行动，既然动手了，就要从根本上摧毁布克尔特的制毒贩毒集团！”
