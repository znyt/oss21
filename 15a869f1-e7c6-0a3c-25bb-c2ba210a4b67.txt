        “好好好，让你这小丫头就是！”傅炎廷笑不拢嘴地说。
<br/>
<br/>        连星儿这才悔棋拿回棋子重新下。
<br/>
<br/>        傅夜沉的母亲见状，还会打电话跟童筱颖念叨，说星儿这小丫头就只会讨老爷子开心。
<br/>
<br/>        下完棋后，一家人用晚餐时，连星儿坐在傅炎廷的身旁，也会很宠曾爷爷，替曾爷爷夹菜，还会很细心地替自己的曾爷爷挑鱼刺和鸡骨头。
<br/>
<br/>        小丫头嘴甜又贴心，傅炎廷巴不得这丫头天天待在自己身边。
<br/>
<br/>        冷太太一边给连星儿盛汤，一边打趣地问道：“小丫头，要不跟着奶奶，住在这傅家大庄园里如何？”
<br/>
<br/>        “学校里的住宿费浪费了太可惜了，要不，我每周双休日回来陪奶奶和曾爷爷。”连星儿笑着说道。
<br/>
<br/>        冷太太将汤碗轻轻地放到连星儿的手边，笑着说道：“就你嘴甜！两边都不得罪。来，尝尝猪蹄汤，补补你这满脸的胶原蛋白。”
<br/>
<br/>        “奶奶，你教我做针线活，好不好？”连星儿端起汤碗喝了一口后，接着说道。
<br/>
<br/>        冷太太不禁好奇起来：“怎么突然想做这个呢？”
<br/>
<br/>        以前，她倒是对童筱颖有这样的要求，但是却从来不要求自己的小孙女学这些。
<br/>
<br/>        隔代宠可不是空穴来风，冷太太对童筱颖的要求很严格，而在连星儿面前，就好像是变了一个人。
<br/>
<br/>        “我想做一个一模一样的公仔，送给我的好朋友。”连星儿微笑道。
<br/>
<br/>        冷太太倒是二话不说就答应了。
<br/>
<br/>        傅炎廷则叮嘱道：“学针线活要小心扎到手。”
<br/>
<br/>        “等我学会了，我给爷爷也做一个软软的靠枕，好不好？”
<br/>
<br/>        “我就知道，我家小丫头是不会忘了我这个老头子的！哈哈——”傅炎廷笑不拢嘴地摸了摸下巴上白花花的公羊须。
<br/>
<br/>        冷太太听着也跟着笑了起来。
<br/>
<br/>        连星儿随后的这几天里，都是白天在学校，傍晚时回傅家大庄园，晚上跟奶奶学针线活，并在庄园里过夜，第二天再回学校。
<br/>
<br/>        就这样连续一个星期下来，校内网的论坛上莫名其妙地多了一些关于她的流言蜚语。
<br/>
<br/>        那些偷拍的照片里，清晨时，连星儿哈欠连天地走进校园；傍晚时，连星儿走到离校不远处的路边上了一辆限量款的豪车。
<br/>
<br/>        议论纷纷的标签是：连星儿、被包养、做小三……
<br/>
<br/>        如果不是邢乐游提醒，明司寒还真不知道连星儿竟然在校内就被人黑得体无完肤。
<br/>
<br/>        不过，也就一分钟的事情，校内网上所有与连星儿有关的舆论和照片，全被人清得一干二净，就连底下辱骂连星儿的那些小号也一并被封了。
<br/>
<br/>        寝室里，张子晋忍不住偷偷瞄了一眼此刻正在用笔记本电脑的明司寒，又与邢乐游面面相觑，彼此心照不宣。
<br/>
<br/>        上次仲正浩和姚莹莹的事情，就让他们笃定，这个连星儿在明司寒心里有着特殊的地位，这次，他们就更加肯定了。
<br/>
<br/>        “乐游，你说着连星儿到底是什么来头？该不会真的是被某个富豪包养了吧？”张子晋故意拉高了音调，跟邢乐游闲聊。
<br/>
<br/>        邢乐游下意识地看了明司寒一眼，见明司寒不吭声，回答道：“反正，现在的富豪和富二代就喜欢找这些刚踏进大学校门的大一新生‘谈恋爱’。不足为奇！”
<br/>
<br/>        “你们最好不要随便在背后非议连星儿的事情，搞不好，她家里会随时给你们送来法院传票。”明司寒淡淡地说道。
<br/>
<br/>        邢乐游和张子晋顿时一脸好奇地看着明司寒，异口同声：“连星儿是什么来历？”
<br/>
<br/>        “她来头不小，圈子里没有哪个富豪或者是富二代敢包养她，所以，你们能闭嘴的就闭嘴。”明司寒不痛不痒地说。
<br/>
<br/>        张子晋不禁打破砂锅问到底：“寒哥！你就跟我们坦白吧！连星儿到底是什么来头？满足一下我们的好奇心！”
<br/>
<br/>        “是啊！正巧，正浩那小子不在，就我们三个人，我们保证不说出去！”邢乐游信誓旦旦地说道。
<br/>
<br/>        明司寒瞥了这两个好奇心爆表的室友一眼，云淡风轻地回答道：“她是傅氏集团老板的女儿，她的曾外公一家子在军政界有一定的地位。”
<br/>
<br/>        顿时，张子晋和邢乐游不约而同地咽口水，静默了。
<br/>
<br/>        同时也纷纷狐疑地看向明司寒。
<br/>
<br/>        话说回来，他明司寒家里的长辈在军政界也是有头有脸的人物吧！
<br/>
<br/>        难道……
<br/>
<br/>        “寒哥，你怕是要跟连星儿上演一场《罗密欧与朱丽叶》的戏了？”邢乐游调侃道。
<br/>
<br/>        张子晋也忍不住笑了起来。
