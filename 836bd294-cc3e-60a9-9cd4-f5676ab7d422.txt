　　站在魔兽山脉中一处山巅，只见一头蓝鹰落在山头上，小医仙摸了摸它的脑袋，道：“小岚，你就待在这里吧，保重。”
<br/>　　慕清霄走上前来，取出几颗魔核，其中二阶三阶各有几颗，见到魔核的蓝鹰，瞳孔中满是渴望，毫不客气的吞了下去。
<br/>　　几颗低阶魔核，足够蓝鹰进阶为三阶魔兽，在这魔兽山脉中也有自保的能力，至于以后会不会成长，就要看它自己的造化了。
<br/>　　“戾！”
<br/>　　蓝鹰叫唤了一声，似乎在与小医仙告别，双翅一展就离地而起，向着魔兽山脉中急驰而去。
<br/>　　“我们也该启程了。”
<br/>　　话语刚落，慕清霄就搂上小医仙的纤腰，脚尖轻点，两人就离地而起，向着距离此处最近的城市，黑岩城疾驰而去。
<br/>　　半空中，靠在慕清霄的怀中，小医仙玉手捂着粉唇，美眸中尽是不可思议。
<br/>　　自己竟翱翔于天空之上，而且近在咫尺的青年没有借助任何外物，但也没有看到能量羽翼，难道他是斗宗强者，可是未免太年轻，太恐怖了些。
<br/>　　“你......你是斗宗？”
<br/>　　小医仙简直惊呆了，粉唇轻启，试探性的问道，芳心跳个不停，那个女子不希望，自己喜欢的男人是一位顶天立地的强者。
<br/>　　慕清霄微微一笑，道：“你这小妮子倒也太高看我了，不过，相差的也不远。”
<br/>　　相差不远，也就是说，斗皇！
<br/>　　天呐！
<br/>　　小医仙的小脑袋简直转不过弯来，前一秒她以为慕清霄最多也就斗师层次，却没想到是一位高高在上的斗皇，这可是加玛帝国巅峰级别的强者啊！
<br/>　　如此年轻的斗皇，简直不可思议！
<br/>　　难怪他有如此自信，娇躯轻轻靠在温暖的怀抱中，小医仙芳心涌上前所未有的安全感，闻着他身上散发出的淡淡清香，有些迷恋上了这股味道。
<br/>　　慕清霄以正常的速度，搂着小医仙御空而行，约莫十几分钟的时间，站在高空隐隐的就看见了黑岩城那颇为巨大的轮廓。
<br/>　　黑岩城是坐落在加玛帝国东部省份的一座大型城市，其规模比起乌坦城来，还要更加雄伟一些，算起总体实力，这座名为黑岩城的城市，在加玛帝国的所有大型城市之中，都能名列前茅。
<br/>　　在一个不引人注目的角落，慕清霄搂着小医仙轻轻落下，随后牵着她向黑岩城中走去。
<br/>　　小医仙也是第一次出来，面前的巨大城市，根本就不是青山镇能比的，美眸扑闪扑闪，有些兴奋，道：“这里就是黑岩城吗？”
<br/>　　“仙儿，时间也不早了，我们在黑岩城中暂住一晚，明日在启程，如何？”
<br/>　　听到慕清霄如此亲密的称呼她，小医仙心中甜蜜不已，自然是百依百顺，乖巧极了，任由他牵着向城中走去。
<br/>　　进入城市，需要交纳一金币的费用，交纳费用，通过有些黑暗的城墙通道，片刻后，视线豁然开朗，喧哗鼎沸的人声，铺天盖地的灌耳而来。
<br/>　　望着周围拥挤的人群，一片繁荣昌盛的景象，慕清霄牵着小医仙，道：“我们先找一家酒店吧，难得来一趟，今晚就好好逛逛吧。”
<br/>　　“嗯，好。”
<br/>　　走在街道上，慕清霄和小医仙都吸引了不少异样的目光，毕竟青年神骏，女子倾国倾城，想不吸引目光都难。
<br/>　　当然，对于某些异样的目光，两人直接无视，在街道上打听一番后，便在黑岩城中找到了一家上好的酒店。
<br/>　　酒店生意不错，环境也极好，倒是上上选，至于价格就有些昂贵，但对于慕清霄和小医仙来说，根本算不得什么。

<br/>　　在楼下吃过晚膳后，两人就一起上了楼。
<br/>　　小医仙的房间就在他的隔壁，慕清霄冲过澡，换上一套相同的白袍，腰间依旧挂着翡翠佩饰，一举一动都透露着不凡，任谁一看都将他当成大家族的公子哥。
<br/>　　“咚咚...”
<br/>　　敲响房门，几秒后，小医仙将房门打开，慕清霄眼中闪过一丝惊艳。
<br/>　　此时的小医仙，换上了一套崭新的白色长裙，银色长发垂直而下，倾城柔美的脸颊上带有一丝嫣红，一眸一笑都格外诱人。

<br/>　　他确实被小医仙给惊艳到了，倾国倾城的脸蛋，配上柔美的气质，绝对是女神级别。
<br/>　　小医仙一颤，伸出莲藕般的玉臂，轻轻的搂在他的腰间，柔软的娇躯靠在他温暖的胸膛上，她毕竟是女儿家，脸皮薄了些，不像慕清霄，做任何事情都随心所欲。
<br/>　　但面对自己喜欢的人，她也会尽量用自己的方式表达出内心的情感。

<br/>　　然而，她的年纪也不过十八岁而已。
