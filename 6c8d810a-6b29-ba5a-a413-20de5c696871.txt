　　“战！遮天手！”
<br/>　　三方对峙着，谁也不让谁，玄天最先出手，他那浩大的武相金身抬手之间，莫不带着无双神威，巨大的力量轰破苍穹，浩大的手中宛若遮蔽九天一般，金光璀璨，力量凝聚，一掌向着妖族的星辰真身拍去。
<br/>　　同时，巫族的盘古真身也动手了，浩大的巨人，一道狂暴的拳头，压迫虚空，充满煞气，好似约定好一般，同时对着星辰真身轰去。巫族也不傻，知道两族最大的敌人是妖族，所以他们联手对付妖族的星辰真身。
<br/>　　“可恶，尔等欺妖太甚？吼！”
<br/>　　帝俊见到两方联手，陡然狰狞一吼，不由得脸色剧变，那浩瀚的星辰真身一手持着屠巫剑，充满杀戮的气息，一手持着星辰剑，无穷星光缭绕其身，仿佛星空古帝一般，大吼一声，分别对着玄天的遮天手与巫族的拳头砍去。
<br/>　　轰！
<br/>　　两道巨响轰然而起，却是玄天的遮天手直接湮灭星辰剑光，毫无阻碍的拍在了星辰真身的身上，同时帝俊的屠巫剑斩在了盘古真身的拳头上，将盘古真身震退。
<br/>　　屠巫剑是妖族杀戮无数人族炼制而成，专门对付巫族肉体的凶剑，天生克制巫族，对于祖巫凝聚而成的盘古真身，同样克制。
<br/>　　光芒璀璨，恢弘无边，惊天动地之时，帝俊喷出大口鲜血，疯狂的疾驰后退，眼中充满愤怒，还有不甘。
<br/>　　若是对付一道真身，他还能应付，但同时对付两人，而且两人都是非常棘手的人物，他一时之间有些招架不住。
<br/>　　“大哥，你怎么样？啊！玄天小儿，接我一剑！”
<br/>　　东皇太一见状，顿时脸色狂变，从神魂受伤之中清醒过来，蓦然怒吼而起，眼中满是杀意，全力催动手中凶剑，杀向玄天。
<br/>　　混元剑本身是要进化成先天至宝的存在，就因为出生较晚，还没等完全孕育出来，便被盘古开辟混沌之时重创了，之后就一直吸收着天地间的无尽凶煞之气进化，成了极品先天灵宝，威力自是不俗。被东皇太一炼化之后，更是凝练成一柄杀气惊天的凶剑。
<br/>　　如今太一的全力一击，剑气凶戾，划破虚空，向着玄天斩来，他自然是不敢小觑，对于太一这个对手郑重对待。
<br/>　　“诛天矛！”
<br/>　　玄天的武祖法相挥手之间出现一道金色长矛，长矛对天，神光闪烁，带着神魂的气息，对着东皇太一对轰而去。
<br/>　　东皇太一受过一次伤害，自然明白这不容小觑，他吸取教训，用一部分力量守护住自己的神魂，长剑与诛天矛对抗起来。
<br/>　　“嗡！”
<br/>　　诛天矛洞穿虚空，神光涌动，直接躲过混元剑的攻击，射向太一。
<br/>　　不好！
<br/>　　太一暗道一声，心念一动，顿时身上出现一道金色的铠甲，护住心脉，之后手中长剑转势斩向金色长矛。
<br/>　　“噗嗤！”
<br/>　　太一终究是被诛天矛刺穿半边身体，虽然被他挡住了大多数攻势，但诛天矛也威力也不容小视。这时，一击过后，诛天矛所凝聚的力量也逸散开来了。
<br/>　　不是东皇太一的混元剑威力不够强，也非他的铠甲防御力不够，只是，双方的修为差距，太大了。
<br/>　　这还是玄天手下留情，若是他使出全部实力，这一矛便可洞穿东皇太一一切防御，诛杀他。东皇太一的实力是准圣中期巅峰，而玄天的实力却已经是准圣圆满了，而且还不是一般的准圣圆满。
<br/>　　“噗嗤！玄天，只恨我妖族没有早日将你人族屠灭，否则哪还有今日之事？吼！”
<br/>　　东皇太一急速倒退，口喷鲜血，肩膀处一个巨大的血洞还正留着金色的鲜血。
<br/>　　而这个时候，帝俊同时也操控着星辰巨人与盘古真身大战在一起。帝俊手持屠巫剑，克制祖巫，再加上祖巫这不是真正的盘古真身，终究是落入了下风。
<br/>　　“吼，帝俊！再来！”
<br/>　　盘古真身怡然不惧，手中法则幻化出一柄虚幻的开天神斧，与帝俊大战起来。
<br/>　　同时，三族之中，其他修士，一样大战起来。人族武皇与妖族白泽等妖神，鲲鹏、伏羲等妖圣大战，劫魔殿的无数劫魔，不论什么，看见就杀，满身血气，劫气充斥着全身，他们的实力在无尽杀戮之中增长起来，除非是一举灭了他们的神魂，否则这些劫魔是一下子杀不死的。
<br/>　　“杀！杀！杀！……”
<br/>　　整个战场就如同一台巨大的绞肉机，每一刻，每一秒都在吞噬无数生命。一条条山脉般的水龙在咆哮，一颗颗流星般的火球在轰炸，一把把锋利的刀剑在切割……
<br/>　　“轰隆隆！”
<br/>　　无数法术、神通、灵宝狂暴的攻击，整片天空都变得动荡不已，一道道空间裂缝在不断延伸，一片片厚重的大山被生生掀起，一颗颗星辰被生生打爆……
<br/>　　冰霜、火焰、飓风、毒雾、流星、洪水、等等，仿佛末日来临，整个世界都陷入了倾覆的危机之中。
<br/>　　“杀！！！”
<br/>　　一声声惨烈的杀喊声汇聚成一曲末日的哀歌，这次战争比之前任何一次都惨烈，战场中所有人都清楚，自己没有退路，退一步，就是死！只有将眼前的敌人全部杀尽，自己方能获得生机。
<br/>　　这一刻，所有修士都化身暴戾的凶兽，完全不畏生死，为了生存，杀！杀！杀！为了种族，杀！杀！杀！，为了未来，战！战！战！
<br/>　　这一刻，没有任何同情，没有任何理智可言，人们全部杀红了双眼，拖着断刀，提着残剑，浴血奋战，杀出一条血淋淋的道路，杀出一个辉煌的明天……
<br/>　　“杀！”“杀！”“杀！”……
<br/>　　天地间一片血红，狂暴的杀气形成一道道血色的飓风，撕裂一层层乌云……
<br/>　　半城烟沙，血泪落下，残骑裂甲，铺红天涯！
<br/>　　“人、巫、妖三族之间的战斗，还真是惨烈，三个种族都是十分恐怖的势力，整个种族汇聚在一切的战力，即便是我已经成圣了，都感到心惊胆颤！”
<br/>　　西方灵山，菩提树下，准提圣人通过眼前的水镜术，看到三族之间的大战，感慨道。
<br/>　　“再厉害有能如何？就算再厉害的势力，也不能永立不倒，就如太古、远古时期的那些势力，凶兽王朝、龙、凤、麒麟三族，哪个不厉害？一个个如今却都已经消失在历史长河之中。唯有证道永恒，才是我辈修士所追求的，其他一切都是虚的，终究会消散在这天地之间……”
<br/>　　接引圣人眼神闪烁，在一旁淡淡说道，
<br/>　　“就如同你我如今已经成圣，不成圣，终为蝼蚁！三族再强又如何，还不是被我等视为蝼蚁，覆手可灭。”
<br/>　　“也是，不过，在天道之下，我等也为蝼蚁啊。”说到这里，准提圣人也不得不感慨一句。
<br/>　　“师弟，慎言。”接引看了一眼天上，提醒道。
<br/>　　“善哉！”
