　　U盘里存储的，全都是有关于龙隐部队战术中队队员的资料，这些资料并不齐全，但也是有着很高的参考价值，以及利用价值。
<br/>　　这个存储容量达到32G的U盘中，只有这十几兆的龙隐部队战术中队队员的文件资料，而且只有龙隐部队战术中队的。
<br/>　　这个U盘是詹姆斯在离开货轮时，让人留在船长办公室的，一般人不可能收集到这样的资料，只有美国人强大的情报系统才能做到，而即便是依靠美国人如此强大的情报系统，也没有完全收集到龙隐部队的资料，只因为这个神秘的特种部队实在太神秘了。
<br/>　　U盘中的资料，布克尔特已经不知看过了多少遍，他在选择，选择合适的，能用来为西伦科特复仇的目标。
<br/>　　布克尔特并不知道西伦科特是死在詹姆斯手中的，除了当时参加行动的人以外，没有人会知道西伦科特是被詹姆斯射杀的。布克尔特把这笔账算在了龙隐部队头上，因为西伦科特会有今天的结果，在布克尔特看来全都是因为当年阿富国的事情，如果没有龙隐部队的多管闲事，西伦科特所做的事情就没有人会知道，他也不会加入到“暗月”来寻求安全！
<br/>　　“韦斯利！”布克尔特紧握着拳头，淡蓝色的眼睛中透露着如毒蛇一样阴冷、凶狠的目光，冷声说道：“你也必须要死！”
<br/>　　布克尔特又一次从头开始浏览U盘中这份几十页的资料。罗昊？不行！
<br/>　　布克尔特不蠢，他不会头脑发热直接去找资料上那些特战队员的麻烦，那个辽阔的东方古国被称为“雇佣兵禁地”，即便是在精锐的雇佣兵也不敢踏上那片土地，更何况，是他手下那些武装毒贩，直接跟资料上这些身经百战，从死人堆里爬出来的特种部队队员面对面，简直就是跟死神面对面无异。所以，只能从他们身边人下手！
<br/>　　云熙，罗昊的妻子？
<br/>　　布克尔特第一次看到资料上云熙照片的时候，就有一种惊艳之感，即便是南美洲最美的“选美皇后”都没有她那么漂亮，这是一种只有东方女人才有的美丽，美丽但是又不张扬。
<br/>　　但是在看过云熙的资料后，布克尔特就把她排除了。资料上显示云熙进入过“炼狱”学习战斗技巧，并且跟上一届“鬼王”有着不明确的关系，在北非对当地的反政府武装势力和军阀进行过斩首行动，根据资料上的不完全统计，云熙在北非的时候总计狙杀目标139个，并且数次从当地反政府武装和军阀的围剿中逃脱，在北非，她被称为“沾血蔷薇”！
<br/>　　如果不是因为资料上的照片和数据，以及云熙在北非战斗时，穿着吉利服拿着狙击步枪的照片，布克尔特很难把这个漂亮的女人跟让北非反政府武装势力和军阀心惊肉跳的“沾血蔷薇”联系到一起。
<br/>　　布克尔特跳过了云熙，他可不会跟这样一个女人交手，如果把这样一个女人作为复仇目标，自己手下有多少人都是白搭，更何况，他根本不可能肆无忌惮的把自己手下的武装毒贩派遣出去。
<br/>　　一页一页的翻看着资料，最终，布克尔特滑动鼠标的动作。
<br/>　　佟千雪！
<br/>　　杭山市儿童医院心胸内科主任医生，多次参加过对外援助活动，最近一次是在欧洲某国核弹危机中，救下了一名差点被汽车碾压的小孩。
<br/>　　资料上，并没有写明佟千雪跟龙隐部队之间的关系，但是却附上了一张照片和一段视频，就是那段曾经在网上传得沸沸扬扬，但是却在一夜之间又消失无踪的视频。
<br/>　　而且在佟千雪那也资料最下方，还有中央情报局用红字做的注释，中央情报局方面认为，佟千雪跟“东方龙”特种部队（龙隐部队）队员慕容飞（无常）之间有不寻常关系！
<br/>　　当然，在建立这份资料的时候，对于佟千雪和慕容飞之间的关系，只是中央情报局的推测，并没有明确的证据进行证明。
<br/>　　布克尔特盯着佟千雪的资料，陷入了沉思，良久，他对着紧闭的房门说道：“Tim，你进来一下！”
<br/>　　房间外，黑人武装毒贩Tim听到布克尔特的声音，立刻推门走进了房间，对坐在桌子后面的布克尔特恭敬的说道：“Boss！”
<br/>　　“跟李霄力联系一下，我需要他帮我调查一个人。”布克尔特说道。
<br/>　　“李霄力？”Tim仔细地回忆了一下，说道：“您是说中国那个李霄力？”
<br/>　　“没错。”布克尔特点头道。
<br/>　　“这个混蛋贪婪，胆小，没有任何用处。”Tim说道。
<br/>　　“你错了Tim。”布克尔特对着Tim摇了摇手指，说道：“贪婪，说明这个人可以被利用；胆小，说明这个人足够聪明。胆小的人往往都很聪明，因为他们知道要怎么样才能让自己活下去。你去联系李霄力，告诉他，帮我调查一个人，查清楚了我给十公斤的高纯度cocaine，免费！”
<br/>　　布克尔特的话，让Tim感到意外。虽然以他们现在的生产力十公斤的cocaine并不算多，但是十公斤的高纯度cocaine的价值却并不小，就这么免费提供给李霄力，值得吗？
<br/>　　“Boss，我知道自己这么问或许很不应该，但我还是想要问清楚，您这么做是因为西伦科特将军吗？”Tim看了眼布克尔特，随后小心翼翼地问道。
<br/>　　布克尔特沉默了一会儿，随后点了点头，说道：“你说得不错。”
<br/>　　“西伦科特将军已经死了，您还这么做值得吗？”Tim又问道。
<br/>　　Tim了解自己的Boss，他知道布克尔特如果不想说，是绝对不可能开口的，如果擅自打听，甚至还会找来杀身之祸，现在布克尔特愿意回答，Tim自然不会放过这个机会。
<br/>　　“值得。”布克尔特回答道。
<br/>　　“为什么？”
<br/>　　布克尔特猛地抬起头，两道目光如同两把利剑一般穿透了房间中的黑暗还有有些浑浊的空气，刺在Tim身上，让Tim感觉到无比的惊恐，后悔自己不该多嘴问这个问题。
